# 知识库导航（面试 + AI）

本知识库在未来半年以 `面试` 和 `AI` 为核心目标，目录使用目标驱动方式：

- `01-面试主线`：面试核心准备材料与复盘
- `02-AI主线`：AI 学习、落地、项目实践
- `03-交叉专题`：AI + 前端、项目包装与 STAR 表达
- `90-参考资料`：历史资料与辅助内容

## 使用规则

1. 新增笔记优先放到 `01-面试主线` 或 `02-AI主线`
2. 每周更新一次 `本周计划.md`
3. 周末至少沉淀一篇复盘（面试题或 AI 实践）
4. 无法归类的内容先放 `90-参考资料/历史归档`，后续再整理
