+ 计算机技术与软件专业技术资格(软考)

国企，事业单位比较认可，私企一般不关注。

积分落户、抵扣个税、技能补贴、提高退休金待遇。

报考无学历和资格限制。

考取后终身有效。



+ 计算机程序设计能力考试(PAT)

主要考察算法能力。

PAT的考试分数可以直接作为PAT联盟企业的介绍信，可以免除筛选简历，笔试环节，从平台直接拿到介绍信，直接进入面试环节。



+ 项目管理专业人士资格认证(PMP认证)
+ 敏捷管理专业人士资格认证(ACP认证)

这两个的报名条件比较严格，有指定专业机构的学时认证。

证书有效期只有三年，续证的话需要在三年内攒够35个培训学时。
