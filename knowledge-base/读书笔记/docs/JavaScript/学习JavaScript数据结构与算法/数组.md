

##### 一、介绍


几乎所有的编程语言都原生支持数组,因为数组是最简单的内存数据结构.



相较于其他的数据结构,数组能够加简洁地呈现同样的信息.



##### 二、创建和初始化数组


+  new Array  

```javascript
let daysOfWeek = new Array(); // 初始化数组

dayOfWeek = new Array(7); // 创建指定长度的数组

daysOfWeek = new Array('Sunday', 'Monday', 'Tuesday', 'Wednewday', 'Thursday', 'Friday', 'Saturday'); // 直接将数组元素作为参数传递给它的构造器
```

+  []   
两种方法相比,'[]'直接创建数组更简便; 

```javascript
let daysOfWeek = [];

daysOfWeek = ['Sunday', 'Monday', 'Tuesday', 'Wednewday', 'Thursday', 'Friday', 'Saturday'];
```

##### 四、添加/删除元素


	在javascript中,数组是一个可以修改的对象.如果添加元素,它就会动态增长.而在C和Java等其他语言中,我们要决定数组的大小,想要添加元素就要创建一个全新的数组,不能简单地往其中添加所需元素.



1.  在数组末尾添加/删除元素  

```javascript
let nums = [1,2,3];

// 添加元素
nums[nums.length] = 4;
// or
nums.push(4);

// 删除元素
nums.pop();
```

**添加元素**  
在数组开头插入元素与在末尾插入不同,需要把数组的所有元素向右移动一位,再将插入的元素放在数组开头;  
数组的unshift()方法就是这个逻辑实现的,代码可参考下面这个;   
**删除元素**

也可使用数组的shift()方法.

3.  在任意位置添加或删除元素  
使用数组的splice()添加/删除相应位置上指定数量的元素.  

```javascript
nums.splice(5, 3) // 删除数组索引5开始的三个元素
nums.splice(5, 0, 3, 4) // 在数组索引5后增加元素3,4
```

javascript中提供的排序方法有reverse和sort方法,如果要反序数组,可以使用reverse

看下字符串数组排序情况;   
javascript在进行字符比较时,是根据字符的ASCII值来比较的,;A、a的ASCII值是65、97,所以A排在a前;  
如果希望小写字母排在前面,可以使用localeCompare方法;   
字符串的localeCompare方法: 返回一个数字来指示一个参考字符串是否在排序顺序前面或之后或与给定字符串相同。  
也可以对任何对象类型的数组排序:  

```javascript
const nums = [3,2,1];
console.log(nums.reverse()); // [1,2,3]
```

```javascript
const nums = [1,3,2,4,11,12];

console.log(nums.sort()); // [1,11,12,2,3,4];
```

```javascript
const nums = [1,3,2,4,11,12];

console.log(nums.sort((a,b) => a - b)); // [1,2,3,4,11,12];
```

```javascript
let names = ['Ana', 'ana', 'John', 'john'];

console.log(names.sort());

// 输出 ['Ana', 'John', 'ana', 'john'];
```

```javascript
let names = ['Ana', 'ana', 'John', 'john'];

console.log(names.sort((a,b) => a.localeCompare(b)));

// 输出 ['ana', 'john', 'Ana', 'John'];
```

5.  搜索 
    - indexOf: 返回与参数匹配的第一个元素的索引;
    - lastIndexOf: 返回与参数匹配的最后一个元素的索引;
    - find: 接受一个回调函数,返回第一个满足条件的元素;
    - findIndex: 接受一个回调函数,返回第一个满足条件的元素索引;
    - Includes: 判断数组是否存在某一元素;
6.  输出数组为字符串  
如果想把数组内的所有元素输出为一个字符串,可以使用toString()方法;   
若想用一个不同的分隔符(比如-)把元素隔开,可以使用join方法;  

```javascript
const nums = [1, 2, 3];
console.log(nums.toString()); // '1,2,3'
```

```javascript
const nums = [1, 2, 3];
console.log(nums.join('-')); // '1-2-3'
```
