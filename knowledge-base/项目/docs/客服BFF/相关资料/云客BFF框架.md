#### 一、云客BFF是什么
##### 1、BFF是什么
<font style="color:rgb(52, 73, 94);">BFF，即 Backend For Frontend（服务于前端的后端），BFF 只是一种逻辑分层，而非一种技术。</font>



##### 2、云客BFF是什么
<font style="color:rgb(52, 73, 94);">云客BFF 以 </font>`<font style="color:rgb(52, 73, 94);">Koa2</font>`<font style="color:rgb(52, 73, 94);"> 作为其基础框架，在它的模型基础上，进一步对它进行了一些增强。</font>

<font style="color:rgb(52, 73, 94);"></font>

##### 3、相关技术栈
+ node.js
+ [koa2](https://koa.bootcss.com/)
+ [typescript](https://www.tslang.cn/docs/home.html)
+ [routing-controllers](https://github.com/typestack/routing-controllers#readme)
+ [typedi](https://github.com/typestack/typedi)
+ [grpc](https://grpc.github.io/grpc/node/index.html)
+ [log4js](https://github.com/log4js-node/log4js-node)
