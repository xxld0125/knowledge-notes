递归是一种解决问题的方法, 它从解决问题的各个小部分开始, 知道解决最初的大问题, 递归通常涉及函数调用自身.

每个递归函数都必须有基线条件, 即一个不再递归的条件, 以防止无限递归.



**计算一个数的阶乘**

在数学上我们计算一个数n的阶乘 n! , 表示从1 到 n 的整数的乘积.

我们可以用过循环来写一个计算一个数阶乘的函数.

```javascript
function factorialIterative(number) {
  if (number < 0) return undefined;
  let total = 1;
  for (let n = number; n > 1; n--) {
    total *= n;
  }
  return total;
}
```

 也可以通过递归来重写factorialIterative函数

```javascript
function factorial(n) {
  if (n === 1 || n === 0) {
    return 1;
  }
  return n * factorial(n - 1);
}
```



**斐波那契数列**

****

使用迭代的方法实现fibonacci函数

```javascript
function fibonacciIterative(n) {
  if (n < 1) return 0;
  if (n <= 2) return 1;
  
  let fibNMinus2 = 0;
  let fibNMinus1 = 1;
  let fibN = n;
  for (let i = 2; i <= n; i++) {
    fibN = fibNMinus1 + fibNMinus2;
    fibNMinus2 = fibNMinus1;
    fibNMinus1 = fibN;
  }
  return fibN;
}
```

递归求斐波那契数

```javascript
function fibonacci (n) {
  if (n < 1) return 0;
  if (n <= 2) return 1;
  return fibonacci(n - 1) + fibonacci(n - 2);
}
```



**记忆化斐波那契数**

**还有第三种写fibonacci函数的方法, 叫做记忆化. 记忆化是一种保存前一个结果的值的优化技术 , 类似于缓存.**

```javascript
function fibonacciMemoization (n) {
  const memo = [0, 1];
  const fibonacci = (n) => {
    if (memo[n] != null) return memo[n];
    return memo[n] = fibonacci(n - 1) + fibonacci(n - 2);
    return fibonacci;
  }
}
```

在上面的方法中, 我们声明了一个memo数组来缓存所有的计算结果, 如果结果已经被计算, 我们就直接返回它, 否则就将结果加入缓存中.
