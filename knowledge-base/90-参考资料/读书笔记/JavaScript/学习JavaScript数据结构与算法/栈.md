##### 一、介绍
	上面我们介绍了数组, 它最常用的数据结构, 可以在数组的任意位置上删除或添加元素.但有时我们还需要一种在添加或删除元素时进行更多控制的数据结构.有两种类似数组的数据结构在添加和删除元素时更为可控,它们就是栈和队列.



		栈是一种遵从**后进先出**(LIFO)原则的有序集合. 新添加或待删除的元素都保存在栈的同一端, 称作栈顶, 另一端就叫栈底. 在栈里里, 新元素都靠近栈顶, 旧元素都接近栈底.



##### 二、基于数组实现栈


	选择数组来保存栈内的元素;



```javascript
class Stack {
	constructor() {
		this.items = [];
	}
}
```







	因为数组允许在任何位置添加或删除元素. 为遵守LIFO原则, 需要限制元素的插入和删除功能.



+  向栈内添加元素  
该方法只能添加元素到栈顶,也就是栈的末尾.  

```javascript
push(element) {
	this.items.push(element);
}
```

+  从栈移除元素  
该方法只能移除最后添加进去的元素.  

```javascript
pop() {
	this.items.pop();
}
```

+  查看栈顶元素  
查看最后添加到栈内的元素.  

```javascript
peek() {
	return this.items.[this.items.length - 1];
}
```

+  检查栈是否为空  

```javascript
isEmpty() {
	return this.items.length === 0;
}
```

+  清空栈元素  
移除栈内所有元素   
 

```javascript
clear() {
	this.items = [];
}
```

+  查看栈元素  

```javascript
size() {
	return this.items.length;
}
```



使用Stack类



```javascript
const stack = new Stack();

console.log(stack.isEmpty()); // true

stack.push(5);
stack.push(8);

console.log(stack.peek()); // 8

stack.push(11);

console.log(stack.size()); // 3

console.log(stack.isEmpty()); // false

stack.push(15);
```

##### 三、基于对象的Stack类


		上面我们介绍了用数组来创建Stack类,这种处理比较常见,但是该方法的时间复杂度为O(n).那么我们有其他方法可以直接获取元素,占用较少的内存空间,且仍能保证所有元素按照我们的需要排列吗?



		下面就介绍用javascript对象来存储所有的栈元素,并保证它们的顺序并且遵守LIFO原则.



首先声明一个Stack类.



```javascript
class Stack {
	constructor() {
		this.count = 0; // 使用count 记录栈的大小
		this.items = {};
	}
}
```



+  向栈内插入元素  
在数组的版本中,允许同时向Stack类中添加多个元素. 但是在对象中, push方法一次只允许插入一个元素.  

```javascript
push(element) {
	this.items[this.count] = element;
	this.count++;
}

const stack = new Stack();
stack.push(5);
stack.push(8);

console.log(stack);

// stack:
{
	items: {
    0: 5,
    1: 8
  },
  count: 2
}
```

+  验证一个栈是否为空和它的大小  

```javascript
size() {
	return this.count;
}

isEmpty() {
	return this.count === 0;
}
```

+  在栈中弹出元素  
对象没有数组自带的push方法,需要收到实现移除元素的逻辑,pop方法也一样.  

```javascript
pop() {
  // 校验栈是否为空
	if (this.isEmpty()) {
		return undefined;
	}
	this.count--;
	const result = this.items[this.count];
	delete this.items[this.count];
	return result;
}
```

+  查看栈顶的值并将栈清空  

```javascript
peek() {
	if (this.isEmpty()) {
		return undefined;
	}
	return this.items[this.count - 1];
}

clear() {
	this.items = {};
	this.count = 0;
}
```

+  创建toString方法  
数组版本中,我们使用数组自带的toString方法.在对象版本中,需要收到实现;   
以上我们就实现了对象版本的Stack类,使用该方法创建的stack,除了toString方法,其他方法复杂度均为O(1),可以直接找到目标元素并进行操作. 

```plain
toString() {
	if (this.isEmpty()) {
		return '';
	}
	
	let objString = `${this.items[0]}`;
	for (let i = 1; i < this.count; i++) {
		objString = `${objString},${this.items[i]}`;
	}
	return objString;
}
```



##### 四、保护数据结构内部元素


```plain
const stack = new Stack();
console.log(Object.getOwnPropertyNames(stack));
console.log(Object.keys(stack));
console.log(stack.items);
```



我们可以直接访问count和items属性,且可以对两个属性赋新的值.但是我们希望内部的元素是受保护的,只有通过我们暴露出的方法才能修改内部结构.上面的Stack内部的items和count属性均未收到保护.



下面将优化Stack类,使其只能访问我们在泪中暴露的方法.



```javascript
class Stack {
	constructor() {
		this._count = 0;
		this._items = {};
	}
}
```



一般使用下划线命名约定来标记一个属性为私有属性.



+  方法一:用ES6的限定作用域Symbol实现类  
ES6新增加的Symbol的基本类型, 它是不可变的, 可以用作对象的属性.   
这个方法创建的私有属性算是一种假的私有属性,主要是利用Object.getOwnPropertyNames()方法不会获取Symbol类的属性,但Object.getOwnPropertySymbols()可以获取,同样可以操作私有属性.所以该方法不可行! 

```plain
const _items = Symbol('stackItems');

class Stack() {
	constructor() {
		this[_items] = [];
	}
}
```

+  用ES6的WeakMap实现类  
还有一种数据类型可以确保属性是私有的,这就是WeakMap.   
以上方法实现items在stack类里是真正的私有属性. 但采用这种方法也有不好的地方,首先代码的可读性不强,其次在扩展该类时无法继承私有属性~ 

```javascript
const items = new WeakMap(); // 声明一个WeakMap类型的变量

class Stack {
	constructor() {
		items.set(this, []); //  以this为键,把代表栈的数组存入items
	}
	push(element) {
		const s = items.get(this); // 从WeakMap中取出值,以this为键,从items中取值
		s.push(element);
	}
	pop() {
		const s = items.get(this);
		const r = s.pop();
		return r;
	}
	// 其他方法;
}
```



##### 五、用栈解决问题


		栈的实际应用非常广泛, 下面介绍如何解决十进制转二进制问题,以及任意禁止转换的算法.



		要把十进制转化为二进制, 可以将该十进制数除以2并对商取整,直到结果是0为止.



过程大概如下:

**十进制转二进制算法**



```javascript
function decimalToBinary(decNumber) {
	const remStack = new Stack();
	let number = decNumber;
  let res;
	let binaryString = '';
	
	while (number > 0) {
    // 将余数放到栈中
		rem = Math.floor(number % 2); 
		remStack.push(rem);
		number = Math.floor(number / 2);
	}
	
	while (!remStack.isEmpty) {
		binartyString += remStack.pop().toString();
	}
	
	return binaryString;
}
```



进制转换算法



优化上面的算法,使其能把十进制转化成基数为2~36之间的任意进制



```javascript
function decimalToBinary(decNumber, base) {
	const remStack = new Stack();
  const digits = '0123456789ABCDEFGHIJKLMNOPQRSYUVWXYZ';
	let number = decNumber;
  let rem = '';
	let baseString = '';
  
  if (!(base >= 2 && base <= 36)) {
    return '';
  }
	
	while (number > 0) {
    // 将余数放到栈中
		rem = Math.floor(number % base); 
		remStack.push(rem);
		number = Math.floor(number / base);
	}
	
	while (!remStack.isEmpty) {
		baseString += digits[remStack.pop()];
	}
	
	return baseString;
}
```
