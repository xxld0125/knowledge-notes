### 权衡的艺术
#### 命令式和声明式
早年间流行的 jQuery 就是典型的命令式框架。命令式框架的一大特点就是**<font style="color:#DF2A3F;">关注过程</font>**。

```javascript
$('#app') // 获取 div
  .text('hello world') // 设置文本内容
  .on('click', () => { alert('ok') }) // 绑定点击事件
```



与命令式框架更加关注过程不同，声明式框架更加**<font style="color:#DF2A3F;">关注结果</font>**。结合 Vue.js，我们来看看如何实现上面自然语言描述的功能。

```vue
<div @click="() => alert('ok')">hello world</div>
```
