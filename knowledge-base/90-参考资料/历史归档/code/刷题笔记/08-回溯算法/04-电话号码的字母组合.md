### 一、[17. 电话号码的字母组合](https://leetcode.cn/problems/letter-combinations-of-a-phone-number/)
<font style="color:rgb(44, 62, 80);">给定一个仅包含数字 2-9 的字符串，返回所有它能表示的字母组合。</font>

<font style="color:rgb(44, 62, 80);">给出数字到字母的映射如下（与电话按键相同）。注意 1 不对应任何字母。</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1660031379222-9d0b7914-d869-49e4-9576-bd2bc4690b94.png)

<font style="color:rgb(44, 62, 80);">示例: 输入："23" 输出：["ad", "ae", "af", "bd", "be", "bf", "cd", "ce", "cf"].</font>

<font style="color:rgb(44, 62, 80);">说明：尽管上面的答案是按字典序排列的，但是你可以任意选择答案输出的顺序。</font>

<font style="color:rgb(44, 62, 80);"></font>

#### 思路
<font style="color:rgb(44, 62, 80);">从示例上来说，输入"23"，最直接的想法就是两层for循环遍历了吧，正好把组合的情况都输出了。</font>

<font style="color:rgb(44, 62, 80);">如果输入"233"呢，那么就三层for循环，如果"2333"呢，就四层for循环.......</font>

<font style="color:rgb(44, 62, 80);">大家应该感觉出和</font>[02-组合问题](https://www.yuque.com/u25370234/kb/lggt90)<font style="color:rgb(44, 62, 80);">遇到的一样的问题，就是这for循环的层数如何写出来，此时又是回溯法登场的时候了。</font>

<font style="color:rgb(44, 62, 80);">理解本题后，要解决如下三个问题：</font>

1. <font style="color:rgb(44, 62, 80);">数字和字母如何映射</font>
2. <font style="color:rgb(44, 62, 80);">两个字母就两个for循环，三个字符我就三个for循环，以此类推，然后发现代码根本写不出来</font>
3. <font style="color:rgb(44, 62, 80);">输入1 * #按键等等异常情况</font>

<font style="color:rgb(44, 62, 80);"></font>

#### <font style="color:rgb(44, 62, 80);">数字和字母如何映射</font>
<font style="color:rgb(44, 62, 80);">可以使用map或者定义一个二维数组，例如：string letterMap[10]，来做映射，我这里定义一个二维数组，代码如下：</font>

```javascript
const map = ["","","abc","def","ghi","jkl","mno","pqrs","tuv","wxyz"];
```



#### 回溯法来解决n个for循环的问题
<font style="color:rgb(44, 62, 80);">例如：输入："23"，抽象为树形结构，如图所示：</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1660034866348-514a1228-e8b9-41f2-a817-16648f3b1928.png)

<font style="color:rgb(44, 62, 80);">图中可以看出遍历的深度，就是输入"23"的长度，而叶子节点就是我们要收集的结果，输出["ad", "ae", "af", "bd", "be", "bf", "cd", "ce", "cf"]。</font>

<font style="color:rgb(44, 62, 80);"></font>

#### 回溯三部曲
##### 确定回溯函数参数
<font style="color:rgb(44, 62, 80);">首先需要一个字符串</font>`<font style="color:rgb(44, 62, 80);">s</font>`<font style="color:rgb(44, 62, 80);">来收集叶子节点的结果，然后用一个字符串数组</font>`<font style="color:rgb(44, 62, 80);">result</font>`<font style="color:rgb(44, 62, 80);">保存起来，这两个变量依然定义为全局。</font>

<font style="color:rgb(44, 62, 80);">再来看参数，参数指定是有题目中给的</font>`<font style="color:rgb(44, 62, 80);">digits</font>`<font style="color:rgb(44, 62, 80);">，k是输入的字符长度,然后还要有一个参数就是</font>`<font style="color:rgb(44, 62, 80);">index</font>`<font style="color:rgb(44, 62, 80);">。</font>

<font style="color:rgb(44, 62, 80);">这个index是记录遍历第几个数字，就是用来遍历digits的（题目中给出数字字符串），同时index也表示树的深度。</font>

<font style="color:rgb(44, 62, 80);">代码如下：</font>

```javascript
const result = [], path = [];

const backtracking = function(digits, k, index) {}
```



##### 确定终止条件
<font style="color:rgb(44, 62, 80);">例如输入用例"23"，两个数字，那么根节点往下递归两层就可以了，叶子节点就是要收集的结果集。</font>

<font style="color:rgb(44, 62, 80);">那么终止条件就是如果index 等于 输入的数字个数（digits.length）了（本来index就是用来遍历digits的）。</font>

<font style="color:rgb(44, 62, 80);">然后收集结果，结束本层递归。</font>

<font style="color:rgb(44, 62, 80);">代码如下：</font>

```javascript
if(path.length === k) {
    result.push(path.join(""));
    return;
}
```



##### 确定单层递归逻辑
<font style="color:rgb(44, 62, 80);">首先要取index指向的数字，并找到对应的字符集（手机键盘的字符集）。</font>

<font style="color:rgb(44, 62, 80);">然后for循环来处理这个字符集，代码如下：</font>

```javascript
for(const v of map[n[a]]) {
    path.push(v);
    backtracking(n, k, a + 1);
    path.pop();
}
```





完整代码

```javascript
var letterCombinations = function(digits) {
    const k = digits.length;
    const map = ["","","abc","def","ghi","jkl","mno","pqrs","tuv","wxyz"];
    if(!k) return [];
    if(k === 1) return map[digits].split("");

    const res = [], path = [];
    backtracking(digits, k, 0);
    return res;

    function backtracking(n, k, a) {
        if(path.length === k) {
            res.push(path.join(""));
            return;
        }
        for(const v of map[n[a]]) {
            path.push(v);
            backtracking(n, k, a + 1);
            path.pop();
        }

    }
};
```
