### 例题-[454. 四数相加 II](https://leetcode.cn/problems/4sum-ii/)
<font style="color:rgb(44, 62, 80);">给定四个包含整数的数组列表 A , B , C , D ,计算有多少个元组 (i, j, k, l) ，使得 A[i] + B[j] + C[k] + D[l] = 0。</font>

<font style="color:rgb(44, 62, 80);">为了使问题简单化，所有的 A, B, C, D 具有相同的长度 N，且 0 ≤ N ≤ 500 。所有整数的范围在 -2^28 到 2^28 - 1 之间，最终结果不会超过 2^31 - 1 。</font>

**<font style="color:rgb(44, 62, 80);">例如:</font>**

<font style="color:rgb(44, 62, 80);">输入: A = [ 1, 2] B = [-2,-1] C = [-1, 2] D = [ 0, 2] 输出: 2</font><font style="color:rgb(44, 62, 80);"> </font>**<font style="color:rgb(44, 62, 80);">解释:</font>**<font style="color:rgb(44, 62, 80);"> </font><font style="color:rgb(44, 62, 80);">两个元组如下:</font>

1. <font style="color:rgb(44, 62, 80);">(0, 0, 0, 1) -> A[0] + B[0] + C[0] + D[1] = 1 + (-2) + (-1) + 2 = 0</font>
2. <font style="color:rgb(44, 62, 80);">(1, 1, 0, 0) -> A[1] + B[1] + C[0] + D[0] = 2 + (-1) + (-1) + 0 = 0</font>



#### <font style="color:rgb(44, 62, 80);">思路：</font>
1. <font style="color:rgb(44, 62, 80);">首先定义 一个</font>`<font style="color:rgb(44, 62, 80);">map</font>`<font style="color:rgb(44, 62, 80);">，</font>`<font style="color:rgb(44, 62, 80);">key</font>`<font style="color:rgb(44, 62, 80);">放</font>`<font style="color:rgb(44, 62, 80);">a</font>`<font style="color:rgb(44, 62, 80);">和</font>`<font style="color:rgb(44, 62, 80);">b</font>`<font style="color:rgb(44, 62, 80);">两数之和，</font>`<font style="color:rgb(44, 62, 80);">value</font>`<font style="color:rgb(44, 62, 80);"> 放</font>`<font style="color:rgb(44, 62, 80);">a</font>`<font style="color:rgb(44, 62, 80);">和</font>`<font style="color:rgb(44, 62, 80);">b</font>`<font style="color:rgb(44, 62, 80);">两数之和出现的次数。</font>
2. <font style="color:rgb(44, 62, 80);">遍历大</font>`<font style="color:rgb(44, 62, 80);">A</font>`<font style="color:rgb(44, 62, 80);">和大</font>`<font style="color:rgb(44, 62, 80);">B</font>`<font style="color:rgb(44, 62, 80);">数组，统计两个数组元素之和，和出现的次数，放到</font>`<font style="color:rgb(44, 62, 80);">map</font>`<font style="color:rgb(44, 62, 80);">中。</font>
3. <font style="color:rgb(44, 62, 80);">定义变量</font>`<font style="color:rgb(44, 62, 80);">count</font>`<font style="color:rgb(44, 62, 80);">，用来统计 a+b+c+d = 0 出现的次数。</font>
4. <font style="color:rgb(44, 62, 80);">在遍历大</font>`<font style="color:rgb(44, 62, 80);">C</font>`<font style="color:rgb(44, 62, 80);">和大</font>`<font style="color:rgb(44, 62, 80);">D</font>`<font style="color:rgb(44, 62, 80);">数组，找到如果 </font>`<font style="color:rgb(44, 62, 80);">0-(c+d) </font>`<font style="color:rgb(44, 62, 80);">在map中出现过的话，就用</font>`<font style="color:rgb(44, 62, 80);">count</font>`<font style="color:rgb(44, 62, 80);">把</font>`<font style="color:rgb(44, 62, 80);">map</font>`<font style="color:rgb(44, 62, 80);">中</font>`<font style="color:rgb(44, 62, 80);">key</font>`<font style="color:rgb(44, 62, 80);">对应的</font>`<font style="color:rgb(44, 62, 80);">value</font>`<font style="color:rgb(44, 62, 80);">也就是出现次数统计出来。</font>
5. <font style="color:rgb(44, 62, 80);">最后返回统计值 </font>`<font style="color:rgb(44, 62, 80);">count</font>`<font style="color:rgb(44, 62, 80);"> 就可以了。</font>

<font style="color:rgb(44, 62, 80);"></font>

```javascript
/**
 * @param {number[]} nums1
 * @param {number[]} nums2
 * @param {number[]} nums3
 * @param {number[]} nums4
 * @return {number}
 */
var fourSumCount = function(nums1, nums2, nums3, nums4) {
    const twoSumMap = new Map();
    let count = 0;

    for(const n1 of nums1) {
        for(const n2 of nums2) {
            const sum = n1 + n2;
            twoSumMap.set(sum, (twoSumMap.get(sum) || 0) + 1)
        }
    }

    for(const n3 of nums3) {
        for(const n4 of nums4) {
            const sum = n3 + n4;
            count += (twoSumMap.get(0 - sum) || 0)
        }
    }

    return count;
};
```
