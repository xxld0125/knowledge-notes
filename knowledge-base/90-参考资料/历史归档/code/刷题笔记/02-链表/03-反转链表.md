### 题目一: [206. 反转链表](https://leetcode-cn.com/problems/reverse-linked-list/) (简单)


#### 解法一: 迭代法
**思路**: 

<font style="color:rgb(44, 62, 80);">如果再定义一个新的链表，实现链表元素的反转，其实这是对内存空间的浪费。</font>

<font style="color:rgb(44, 62, 80);">其实只需要改变链表的</font>`<font style="color:rgb(44, 62, 80);">next</font>`<font style="color:rgb(44, 62, 80);">指针的指向，直接将链表反转 ，而不用重新定义一个新的链表，如图所示:</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1652175230807-ced39d3c-ed93-4755-b61a-d0e42bf8ad5f.png)

<font style="color:rgb(44, 62, 80);">之前链表的头节点是元素1， 反转之后头结点就是元素5 ，这里并没有添加或者删除节点，仅仅是改变</font>`<font style="color:rgb(44, 62, 80);">next</font>`<font style="color:rgb(44, 62, 80);">指针的方向。</font>

<font style="color:rgb(44, 62, 80);">那么接下来看一看是如何反转的呢？</font>

<font style="color:rgb(44, 62, 80);">我们拿有示例中的链表来举例，如动画所示</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1652175257739-e3a5dbfb-0bfe-4c7d-8f9a-bf036f8824ea.png)

<font style="color:rgb(44, 62, 80);">首先定义一个</font>`<font style="color:rgb(44, 62, 80);">cur</font>`<font style="color:rgb(44, 62, 80);">指针，指向头结点，再定义一个</font>`<font style="color:rgb(44, 62, 80);">pre</font>`<font style="color:rgb(44, 62, 80);">指针，初始化为</font>`<font style="color:rgb(44, 62, 80);">null</font>`<font style="color:rgb(44, 62, 80);">。</font>

<font style="color:rgb(44, 62, 80);">然后就要开始反转了，首先要把 </font>`<font style="color:rgb(44, 62, 80);">cur->next</font>`<font style="color:rgb(44, 62, 80);">节点用</font>`<font style="color:rgb(44, 62, 80);">tmp</font>`<font style="color:rgb(44, 62, 80);">指针保存一下，也就是保存一下这个节点。</font>

<font style="color:rgb(44, 62, 80);">为什么要保存一下这个节点呢，因为接下来要改变 </font>`<font style="color:rgb(44, 62, 80);">cur->next</font>`<font style="color:rgb(44, 62, 80);"> 的指向了，将</font>`<font style="color:rgb(44, 62, 80);">cur->next</font>`<font style="color:rgb(44, 62, 80);"> 指向</font>`<font style="color:rgb(44, 62, 80);">pre</font>`<font style="color:rgb(44, 62, 80);"> ，此时已经反转了第一个节点了。</font>

<font style="color:rgb(44, 62, 80);">接下来，就是循环走如下代码逻辑了，继续移动</font>`<font style="color:rgb(44, 62, 80);">pre</font>`<font style="color:rgb(44, 62, 80);">和</font>`<font style="color:rgb(44, 62, 80);">cur</font>`<font style="color:rgb(44, 62, 80);">指针。</font>

<font style="color:rgb(44, 62, 80);">最后，</font>`<font style="color:rgb(44, 62, 80);">cur</font>`<font style="color:rgb(44, 62, 80);"> 指针已经指向了</font>`<font style="color:rgb(44, 62, 80);">null</font>`<font style="color:rgb(44, 62, 80);">，循环结束，链表也反转完毕了。 此时我们</font>`<font style="color:rgb(44, 62, 80);">return pre</font>`<font style="color:rgb(44, 62, 80);">指针就可以了，</font>`<font style="color:rgb(44, 62, 80);">pre</font>`<font style="color:rgb(44, 62, 80);">指针就指向了新的头结点。</font>



**代码**

```javascript
/**
 * Definition for singly-linked list.
 * function ListNode(val, next) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.next = (next===undefined ? null : next)
 * }
 */
/**
 * @param {ListNode} head
 * @return {ListNode}
 */
var reverseList = function(head) {
    let cur = head,
        prev = null;
    while (cur) {
        let tmp = cur.next;
        cur.next = prev;
        prev = cur;
        cur = tmp;
    }
    return prev;
};
```

**<font style="color:rgb(38, 38, 38);"></font>**

**<font style="color:rgb(38, 38, 38);">复杂度分析</font>**

+ <font style="color:rgb(38, 38, 38);">时间复杂度：O(n)，其中 n</font>_<font style="color:rgb(38, 38, 38);"></font>_<font style="color:rgb(38, 38, 38);"> 是链表的长度。需要遍历链表一次。</font>
+ <font style="color:rgb(38, 38, 38);">空间复杂度：O(1)</font>_<font style="color:rgb(38, 38, 38);"></font>_<font style="color:rgb(38, 38, 38);">。</font>

#### 解法二: 递归法
**思路**: 

主要的处理逻辑就是将`cur`指向`pre`的过程,当`cur`为空时结束循环, <font style="color:rgb(44, 62, 80);">递归写法和双指针法实质上都是从前往后翻转指针指向;</font>



而**<font style="color:#E8323C;">方法二</font>**的思路是<font style="color:rgb(44, 62, 80);">从后往前翻转指针指向;</font>

**代码**: 

```javascript
/**
 * Definition for singly-linked list.
 * function ListNode(val, next) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.next = (next===undefined ? null : next)
 * }
 */
/**
 * @param {ListNode} head
 * @return {ListNode}
 */

// 递归一
var reverse = function(pre, head) {
  if (!head) return pre;
  
  const tmp = head.next;
  
  head.next = pre;
  
  pre = head;
  
  return reverse(pre, tmp);
  
}

var reverseList = function(head) {
  return reverse(null, head);
}

// 递归二
var reverseList = function(head) {
    if (head === null || head.next === null) {
        return head;
    }
    const p = reverseList(head.next);

    head.next.next = head;
    head.next = null;

    return p;
};
```



**复杂度分析**

****

+ 时间复杂度：O(n)，其中 n 是链表的长度。需要对链表的每个节点进行反转操作。
+ 空间复杂度：O(n)，其中 n 是链表的长度。空间复杂度主要取决于递归调用的栈空间，最多为 n 层。

#### 解法三: 双指针
**思路:**

**双指针与上面的递归二的方法逻辑类似.**

```javascript
/**
 * Definition for singly-linked list.
 * function ListNode(val, next) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.next = (next===undefined ? null : next)
 * }
 */
/**
 * @param {ListNode} head
 * @return {ListNode}
 */
var reverseList = function(head) {
  if (!head || !head.next) return head;
  
  let tmp = null, pre = null, cur = head;
  
  while (cur) {
    tmp = cur.next;
    
    cur.next = pre;
    
    pre = cur;
    
    cur = tmp;
  }
  
  return pre;
}
```







### 题目二: [24. 两两交换链表中的节点](https://leetcode-cn.com/problems/swap-nodes-in-pairs/) (中等)
#### 解法一: 迭代
**思路: **

使用虚拟头节点， 推进指针`cur`从当前虚拟头节点位置出发，故`cur.next=head`，确认接下来两个节点`cur.next`, `cur.next.next`是否为空

若不为空则开始交换: `a`为`cur.next`, `b`为`cur.next.next`, 同时发生`cur.next=b`, `a.next=b.next`, `b.next=a`; `cur`推进两格; 循环继续;

若有一个或一个以上为空则直接返回哨兵`head`节点的`next`;



<font style="color:rgb(44, 62, 80);">初始时，cur指向虚拟头结点，然后进行如下三步：</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1652178835272-a51d7bfc-5c48-44da-9187-dfb105354401.png)

<font style="color:rgb(44, 62, 80);">操作之后，链表如下：</font>



**代码**

```javascript
/**
 * Definition for singly-linked list.
 * function ListNode(val, next) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.next = (next===undefined ? null : next)
 * }
 */
/**
 * @param {ListNode} head
 * @return {ListNode}
 */
var swapPairs = function(head) {
  	if (head === null || head.next === null) {
        return head;
    }
    
    let list = new ListNode(null, head);
    let tmp = list;

    while (list.next && list.next.next) {
        let prev = list.next, next = list.next.next.next;

        list.next = prev.next;
        prev.next.next = prev;
        prev.next = next;

        list = list.next.next;
    }

    return tmp.next;
};
```



#### 解法二: 递归
**思路: **

要把 head 和 head.next 交换，也就是相邻的节点交换;交换之后的 head.next.next 要指向下一对交换节点的 head 节点; 然后递归直到 head 或者 head.next 为 null，也就是不够两个节点进行交换; 要注意递归时传入下一对交换节点的 head 节点要传哪个，



**代码:**

```javascript
/**
 * Definition for singly-linked list.
 * function ListNode(val, next) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.next = (next===undefined ? null : next)
 * }
 */
/**
 * @param {ListNode} head
 * @return {ListNode}
 */
var swapPairs = function(head) {
   if (head === null || head.next === null) {
       return head;
   }

   const tmp = head.next;

   head.next = swapPairs(head.next.next);

   tmp.next = head;

   return tmp;
};
```



### 题目三:  [25. K 个一组翻转链表](https://leetcode-cn.com/problems/reverse-nodes-in-k-group/) (困难)


#### 解法一:
#### 
```javascript
/**
 * Definition for singly-linked list.
 * function ListNode(val, next) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.next = (next===undefined ? null : next)
 * }
 */
/**
 * @param {ListNode} head
 * @param {number} k
 * @return {ListNode}
 */
var myReverse = function (head, tail) {
    let pre = tail.next;
    let p = head;
    while (pre !== tail) {
        const nex = p.next;
        p.next = pre;
        pre = p;
        p = nex;
    }
    return [tail, head];
}

var reverseKGroup = function(head, k) {
    const hair = new ListNode(null, head);

    let pre = hair;

    while (head) {
        let tail = pre;
      	// 查看剩余部分长度是否大于等于 k
        for (let i = 0;i < k;i++) {
            tail = tail.next;
            if (!tail) {
              	// 剩余部分不满足长度, 则不处理, 直接返回;
                return hair.next;
            }
        }

        const nex = tail.next;

        [head, tail] = myReverse(head, tail);
        
        pre.next = head;

        tail.next = nex;

        pre = tail;

        head = tail.next;
    }
    return hair.next;
};
```



### 题目四:92

```javascript
var reverseBetween = function(head, left, right) {
    // 因为头节点有可能发生变化，使用虚拟头节点可以避免复杂的分类讨论
    const dummyNode = new ListNode(-1);
    dummyNode.next = head;

    let pre = dummyNode;
    // 第 1 步：从虚拟头节点走 left - 1 步，来到 left 节点的前一个节点
    // 建议写在 for 循环里，语义清晰
    for (let i = 0; i < left - 1; i++) {
        pre = pre.next;
    }

    // 第 2 步：从 pre 再走 right - left + 1 步，来到 right 节点
    let rightNode = pre;
    for (let i = 0; i < right - left + 1; i++) {
        rightNode = rightNode.next;
    }

    // 第 3 步：切断出一个子链表（截取链表）
    let leftNode = pre.next;
    let curr = rightNode.next;

    // 注意：切断链接
    pre.next = null;
    rightNode.next = null;

    // 第 4 步：同第 206 题，反转链表的子区间
    reverseLinkedList(leftNode);

    // 第 5 步：接回到原来的链表中
    pre.next = rightNode;
    leftNode.next = curr;
    return dummyNode.next;
};

const reverseLinkedList = (head) => {
    let pre = null;
    let cur = head;

    while (cur) {
        const next = cur.next;
        cur.next = pre;
        pre = cur;
        cur = next;
    }
}
```
