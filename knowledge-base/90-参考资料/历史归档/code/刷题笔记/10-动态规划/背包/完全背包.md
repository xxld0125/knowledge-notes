### 经典问题
有`N`件物品和一个最多能背重量为`W`的背包。第`i`件物品的重量是`weight[i]`，得到的价值是`value[i]` 。每件物品都有无限个（也就是可以放入背包多次），求解将哪些物品装入背包里物品价值总和最大。

### 解题思路(二维dp数组)
1. dp数组的定义

有`i`种物品，放入容量为`j`的背包中，最大价值为`dp[i][j]`

2. dp数组的递推公式

`dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - weight[i]] + values[i])`

3. dp数组的初始化

dp数组初始化为0

4. dp数组的遍历顺序

可以先遍历物品后遍历背包，也可以先遍历背包后遍历物品

### 其他完全背包问题
#### [518.零钱兑换II](https://leetcode.cn/problems/coin-change-ii/description/)
思路：结合了完全背包和求组合数，需要了解完全背包和求组合数场景的dp数组递推公式和遍历顺序。

**一维dp数组**

1. dp数组定义

凑成总金额`j`的货币组合数为`dp[j]`。

2. dp数组递推公式

`dp[j] += dp[j - weight[i]]`

3. dp数组初始化

当金额为0时，有一种组合方法

`dp[0] = 1`

4. dp数组遍历顺序

先遍历物品，后遍历背包

5. 完整代码

```javascript
function change(amount: number, coins: number[]): number {
    const dp: number[] = new Array(amount + 1).fill(0);
    dp[0] = 1;

    const len = coins.length;

    // 先遍历物品(正序遍历)
    for (let i = 0; i < len; i++) {
        // 后遍历背包(正序遍历)
        for (let j = coins[i]; j <= amount; j++) {
            dp[j] += dp[j - coins[i]];
        }
    }

    return dp[amount];
};
```

****

**<font style="color:#DF2A3F;">虽然在介绍经典完全背包问题时，提到可以先遍历背包，后遍历物品，但是在求完全背包组合数问题时，只可先遍历物品，后遍历背包。因为纯完全背包求得装满背包的最大价值是多少，和凑成总和的元素有没有顺序没关系，即：有顺序也行，没有顺序也行！</font>**

本题是求凑出来的方案个数，且每个方案个数是为组合数(无顺序要求)。

假设：coins[0] = 1，coins[1] = 5。

先遍历物品的话，就是先把1加入计算，然后再把5加入计算，得到的方法数量只有{1, 5}这种情况。而不会出现{5, 1}的情况。

所以这种遍历顺序中`dp[j]`里计算的是组合数！



当我们先遍历背包时，背包容量的每一个值，都是经过 1 和 5 的计算，包含了{1, 5} 和 {5, 1}两种情况。

此时dp[j]里算出来的就是排列数！

****

**二维dp数组**

1. dp数组定义

i种物品，凑成总金额`j`的货币组合数为`dp[i][j]`。

2. dp数组递推公式

`dp[i][j] = dp[i - 1][j] + dp[i][j - weight[i]]`

3. dp数组初始化

当金额为0时，且无物品时，有一种组合方法

`dp[0][0] = 0`

4. dp数组遍历顺序

先遍历物品，后遍历背包,或先遍历背包后遍历物品都行。

5. 完整代码

```javascript
function change(amount: number, coins: number[]): number {
    const len = coins.length;
    const dp = new Array(len + 1).fill(0).map(() => new Array(amount + 1).fill(0));

    dp[0][0] = 1;

    // 先遍历物品, 后遍历背包
    // for (let i = 1; i <= len; i++) {
    //     for (let j = 0; j <= amount; j++) {
    //         if (j >= coins[i - 1]) {
    //             dp[i][j] = dp[i][j - coins[i - 1]] + dp[i - 1][j];
    //         } else {
    //             dp[i][j] = dp[i - 1][j];
    //         }
    //     }
    // }

    // 先遍历背包, 后遍历物品

            if (j >= coins[i - 1]) {
                dp[i][j] = dp[i][j - coins[i - 1]] + dp[i - 1][j];

            }
        }
    }

    return dp[len][amount];

```



二维dp数组中，需要重点关注dp数组的递推公式，与之前的01背包有不同。

dp数组的递推逻辑是放种类i和不放种类i:

不放种类i的方法数：`dp[i - 1][j]`

放种类i的方法数: `dp[i][j - coins[i - 1]]`



主要差别在放种类`i`的方法数上，因为之前01背包每个物品只能用一次，所以01背包的放物品i的方法数是`dp[i - 1][j - coins[i - 1]`。但是完全背包物品i的定义是物品的种类，所以放种类i时，`i`不需要`-1`。

#### [322.零钱兑换](https://leetcode.cn/problems/coin-change/)
思路：完全背包 + 求最小组合长度



**一维dp数组**

**<font style="color:#DF2A3F;">重点是dp数组递推公式和dp数组初始化</font>**

1. dp数组的定义

容量为j的背包，装满时最小物品数dp[j]。



2. dp数组的递推公式：

凑足总额为`j - coins[i]`的最少个数为`dp[j - coins[i]]`，那么只需要加上一个钱币`coins[i]`即`dp[j - coins[i]] + 1`就是`dp[j]`（考虑`coins[i]`）。

所以`dp[j]`要取所有 `dp[j - coins[i]] + 1` 中最小的。

递推公式：`dp[j] = min(dp[j - coins[i]] + 1, dp[j])`。



3. dp数组初始化

因为是求最小值，所以dp数组需要初始化为最大值，背包容量为0时，`dp[j]`为0。



4. dp数组的遍历顺序

本题求钱币最小个数，那么钱币有顺序和没有顺序都可以，都不影响钱币的最小个数。



5. 完整代码

```javascript
function coinChange(coins: number[], amount: number): number {
    const len = coins.length;

    // dp数组初始化
    const dp: number[] = new Array(amount + 1).fill(Infinity);
    dp[0] = 0;


    // 先遍历物品, 后遍历背包
/*     for (let i = 0; i < len; i++) {
        for (let j = coins[i]; j <= amount; j++) {
            dp[j] = Math.min(dp[j], dp[j - coins[i]] + 1);
        }
    } */

    // 先遍历背包, 后遍历物品
    for (let j = 0; j <= amount; j++) {
        for (let i = 0; i < len; i++) {
            if (j >= coins[i]) {
                dp[j] = Math.min(dp[j], dp[j - coins[i]] + 1);
            }
        }
    }

    return dp[amount] === Infinity ? - 1 : dp[amount];
};
```





**二维dp数组**

1. dp数组的定义

表示使用前 `i` 种硬币组成金额 `j` 所需的最少硬币数`dp[i][j]`。



2. dp数组的递推公式：

对于每个 `i` 和 `j`，我们有两种选择：

+ 不使用第 `i` 种硬币：这种情况下，最少硬币数等于使用前 `i-1` 种硬币组成金额 `j` 的硬币数，即 `dp[i-1][j]`。
+ 使用第 `i` 种硬币：如果 `j` 大于等于当前硬币面额 `coins[i-1]`，则 `dp[i][j]` 可以通过查看组成金额 `j - coins[i-1]` 所需的最少硬币数再加一来得到，即 `dp[i][j - coins[i-1]] + 1`。



可推出：`dp[i][j]=Math.min(dp[i − 1][j], dp[i][j − coins[i − 1]] + 1)`



3. dp数组初始化
+ `dp[i][0]` 被初始化为 0 对于所有 `i`，因为组成金额 0 不需要任何硬币。
+ `dp[0][j]`（除了 `j = 0`）被初始化为一个大数，表示使用 0 种硬币无法组成任何正金额。



4. dp数组的遍历顺序

本题求钱币最小个数，那么钱币有顺序和没有顺序都可以，都不影响钱币的最小个数。



5. 完整代码

```javascript
function coinChange(coins, amount) {
    const n = coins.length;
    
    // 初始化dp数组
    let dp = new Array(n + 1).fill(0).map(() => new Array(amount + 1).fill(Infinity));
    for (let i = 0; i <= n; i++) {
        dp[i][0] = 0;
    }

    // 动态规划填充dp数组
    for (let i = 1; i <= n; i++) {
        for (let j = 1; j <= amount; j++) {
            // 不使用第i种硬币
            dp[i][j] = dp[i - 1][j];
            // 使用第i种硬币，前提是当前金额大于等于硬币面值
            if (j >= coins[i - 1]) {
                dp[i][j] = Math.min(dp[i][j], dp[i][j - coins[i - 1]] + 1);
            }
        }
    }

    // 检查是否有解
    return dp[n][amount] === Infinity ? -1 : dp[n][amount];
}
```



#### [279.完全平方式](https://leetcode.cn/problems/perfect-squares/)
此题与[322.零钱兑换](https://leetcode.cn/problems/coin-change/)一样都是完全背包 + 求最小`dp[j]`数

```javascript
function numSquares(n: number): number {
    const dp: number[] = new Array(n + 1).fill(Infinity);

    dp[0] = 0;

    for (let i = 1; i ** 2 <= n; i++) {
        for (let j = i ** 2; j <= n; j++) {
            dp[j] = Math.min(dp[j], dp[j - i ** 2] + 1);
        }
    }

    return dp[n];
};
```



#### [139.单词拆分](https://leetcode.cn/problems/word-break/)
1. dp数组的定义

`dp[i]` : 字符串长度为i的话，`dp[i]`为true，表示可以拆分为一个或多个在字典中出现的单词。



2. dp数组的递推公式：

若`dp[j]` 是true，且 `[j, i]` 这个区间的子串出现在字典里，那么`dp[i]`一定是true。（j < i ）。

所以递推公式是 if([j, i] 这个区间的子串出现在字典里 && dp[j]是true) 那么 dp[i] = true。



`dp[i] = dp[j] && wordDict.includes(s.slice(j, i))`



3. dp数组初始化

从递推公式中可以看出，`dp[i]` 的状态依靠 `dp[j]`是否为true，那么`dp[0]`就是递推的根基，`dp[0]`一定要为true，否则递推下去后面都都是false了。



4. dp数组的遍历顺序

此题的单词即为背包，单词的组合是要求组合的顺序，所以是先遍历背包后遍历物品。



5. 完整代码



```typescript
const dp: boolean[] = new Array(s.length + 1).fill(false);
dp[0] = true;
for (let i = 1; i <= s.length; i++) {
  for (let j = 0; j < i; j++) {
    const tempStr: string = s.slice(j, i);
    if (wordDict.includes(tempStr) && dp[j] === true) {
      dp[i] = true;
      break;
    }
  }
}
return dp[s.length];
```
