### 打家劫舍问题
你是一个专业的小偷，计划偷窃沿街的房屋。每间房内都藏有一定的现金，影响你偷窃的唯一制约因素就是相邻的房屋装有相互连通的防盗系统，如果两间相邻的房屋在同一晚上被小偷闯入，系统会自动报警。

给定一个代表每个房屋存放金额的非负整数数组，计算你 不触动警报装置的情况下 ，一夜之内能够偷窃到的最高金额。

+ 输入：[1,2,3,1]
+ 输出：4

解释：偷窃 1 号房屋 (金额 = 1) ，然后偷窃 3 号房屋 (金额 = 3)。   偷窃到的最高金额 = 1 + 3 = 4 。



### 解题思路
#### 1.确定dp数组及下标的定义
`dp[i]`：考虑下标`i`以内的房屋，最多可以偷窃的金额为`dp[i]`。



#### 2.确定dp数组的递推公式
`dp[i] = max(dp[i - 2] + nums[i], dp[i - 1])`

<font style="color:rgb(44, 62, 80);"></font>

#### 3.dp数组初始化
从递推公式`dp[i] = max(dp[i - 2] + nums[i], dp[i - 1])`可以看出，递推公式的基础就是`dp[0]`和`dp[1]`。

从`dp[i]`的定义上来讲，`dp[0]`一定是`nums[0]`，`dp[1]`就是`nums[0]`和`nums[1]`的最大值即：`dp[1] = max(nums[0], nums[1])`。



#### 4.确定遍历顺序
`dp[i]`是根据`dp[i - 2]`和`dp[i - 1]`推导出来的，那么一定是从前到后遍历。



#### 5.完整代码
```javascript
var rob = function(nums) {
    const len = nums.length;
    if (len === 1) return nums[0];
    if (len === 2) return Math.max(nums[0], nums[1]);

    const dp = new Array(len).fill(0);
    
    dp[0] = nums[0];
    dp[1] = Math.max(nums[0], nums[1]);

    for (let i = 2; i < len; i++) {
        dp[i] = Math.max(dp[i - 1], dp[i - 2] + nums[i]);
    }

    return dp[len - 1];
};
```



### 扩展：打家劫舍II
#### 题目
在原题的基础上，房屋改为环形，意味着第一个房屋和最后一个房屋是紧挨着的。



#### 解题思路
这题与标准的打家劫舍问题的唯一区别是需要考虑数组是环形的。数组成环，可以拆分为两个非环形的情况：

1. 包含首元素，不包含尾元素
2. 包含尾元素，不包含首元素

取两种情况下的非环形最大金额的最大值，即为环形的最大金额。



#### 完整代码
```javascript
var rob = function(nums) {
    const len = nums.length;

    // 数量少于3时, 直接返回其中的最大值
    if (len <= 3) {
        return Math.max(...nums);
    }

    const dp1 = new Array(len - 1).fill(0);
    const dp2 = new Array(len - 1).fill(0);

    dp1[0] = nums[0];
    dp1[1] = Math.max(nums[0], nums[1]);

    for (let i = 2; i < len - 1; i++) {
        dp1[i] = Math.max(dp1[i - 1], dp1[i - 2] + nums[i]);
    }

    dp2[0] = nums[1];
    dp2[1] = Math.max(nums[1], nums[2]);

    for (let i = 3; i < len; i++) {
        dp2[i - 1] = Math.max(dp2[i - 2], dp2[i - 3] + nums[i]);
    }

    
    return Math.max(dp1[len - 2], dp2[len - 2]);
};
```



### 扩展：打家劫舍III
#### 题目
本题在经典打家劫舍问题的基础上，添加了二叉树的。首先需要先确认二叉树的遍历顺序，因为本题节点的取数依赖它的左右子节点，所以肯定是后续遍历。

对于每个节点都有两种状态：偷与不偷。这里可以使用一个长度为2的数组，记录当前节点偷与不偷所得到的的最大金钱。



#### 1.确定dp数组及下标的定义
下标为0记录偷该节点所得到的的最大金钱，下标为1记录不偷该节点所得到的的最大金钱。



#### 2.确定终止条件
在遍历的过程中，如果遇到空节点的话，很明显，无论偷还是不偷都是0，所以就返回。

#### 3.确定遍历顺序
首先明确的是使用后序遍历。 因为要通过递归函数的返回值来做下一步计算。

通过递归左节点，得到左节点偷与不偷的金钱。

通过递归右节点，得到右节点偷与不偷的金钱。



#### 4.确定单层递归的逻辑
如果是偷当前节点，那么左右孩子就不能偷，`val1 = cur.val + left[0] + right[0]`;（如果对下标含义不理解就再回顾一下dp数组的含义）

如果不偷当前节点，那么左右孩子就可以偷，至于到底偷不偷一定是选一个最大的，所以：`val2 = Math.max(left[0], left[1]) + Math.max(right[0], right[1])`

最后当前节点的状态就是`[val1, val2]`; 即：{不偷当前节点得到的最大金钱，偷当前节点得到的最大金钱}



#### 5.完整代码
```javascript
var rob = function(root) {
    // 动态规划
    // 思路: 后序遍历, 每个节点维护一个dp数组
    // dp[0]表示偷当前节点的最大收益, dp[1]表示不偷当前节点的最大收益

    const dp = dfs(root);

    return Math.max(dp[0], dp[1]);
};

function dfs(node) {
    if (node === null) return [0, 0];

    const left = dfs(node.left);
    const right = dfs(node.right);

    // 偷当前节点的最大收益

    return [val1, val2];
}
```
