### 一、例题-[541. 反转字符串 II](https://leetcode.cn/problems/reverse-string-ii/)
<font style="color:rgb(44, 62, 80);">给定一个字符串 s 和一个整数 k，你需要对从字符串开头算起的每隔 2k 个字符的前 k 个字符进行反转。</font>

<font style="color:rgb(44, 62, 80);">如果剩余字符少于 k 个，则将剩余字符全部反转。</font>

<font style="color:rgb(44, 62, 80);">如果剩余字符小于 2k 但大于或等于 k 个，则反转前 k 个字符，其余字符保持原样。</font>

<font style="color:rgb(44, 62, 80);">示例:</font>

<font style="color:rgb(44, 62, 80);">输入: s = "abcdefg", k = 2  
</font><font style="color:rgb(44, 62, 80);">输出: "bacdfeg"</font>



#### 思路
#### <font style="color:rgb(44, 62, 80);">这道题目其实也是模拟，实现题目中规定的反转规则就可以了。</font>
<font style="color:rgb(44, 62, 80);">一些同学可能为了处理逻辑：每隔</font>`<font style="color:rgb(44, 62, 80);">2k</font>`<font style="color:rgb(44, 62, 80);">个字符的前k的字符，写了一堆逻辑代码或者再搞一个计数器，来统计</font>`<font style="color:rgb(44, 62, 80);">2k</font>`<font style="color:rgb(44, 62, 80);">，再统计前</font>`<font style="color:rgb(44, 62, 80);">k</font>`<font style="color:rgb(44, 62, 80);">个字符。</font>

<font style="color:rgb(44, 62, 80);">其实在遍历字符串的过程中，只要让</font>`<font style="color:rgb(44, 62, 80);"> i += (2 * k)</font>`<font style="color:rgb(44, 62, 80);">，i 每次移动 2 * k 就可以了，然后判断是否需要有反转的区间。</font>

<font style="color:rgb(44, 62, 80);">因为要找的也就是每2 * k 区间的起点，这样写，程序会高效很多。</font>

**<font style="color:rgb(44, 62, 80);">所以当需要固定规律一段一段去处理字符串的时候，要想想在在for循环的表达式上做做文章。</font>**

  


```javascript
/**
 * @param {string} s
 * @param {number} k
 * @return {string}
 */
var reverseStr = function(s, k) {
    const len = s.length;
    const res = Array.from(s);
    for (let i = 0;i < len;i += 2 * k) {
        reverse(res, i, Math.min(i + k, len) - 1);
    }
    function reverse(arr, l, r) {
        while (l < r) {
            [arr[l], arr[r]] = [arr[r], arr[l]];
            l++;
            r--;
        }
    }

    return res.join('');
};
```
