### 一、例题-[151. 颠倒字符串中的单词](https://leetcode.cn/problems/reverse-words-in-a-string/)
<font style="color:rgb(44, 62, 80);">给定一个字符串，逐个翻转字符串中的每个单词。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">示例 1：  
</font><font style="color:rgb(44, 62, 80);">输入: "the sky is blue"  
</font><font style="color:rgb(44, 62, 80);">输出: "blue is sky the"</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">示例 2：</font>

<font style="color:rgb(44, 62, 80);">  
</font><font style="color:rgb(44, 62, 80);">输入: "  hello world!  "  
</font><font style="color:rgb(44, 62, 80);">输出: "world! hello"  
</font><font style="color:rgb(44, 62, 80);">解释: 输入字符串可以在前面或者后面包含多余的空格，但是反转后的字符不能包括。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">示例 3</font>

<font style="color:rgb(44, 62, 80);">  
</font><font style="color:rgb(44, 62, 80);">输入: "a good   example"  
</font><font style="color:rgb(44, 62, 80);">输出: "example good a"  
</font><font style="color:rgb(44, 62, 80);">解释: 如果两个单词间有多余的空格，将反转后单词间的空格减少到只含一个。</font>



#### 思路
**<font style="color:rgb(44, 62, 80);">这道题目可以说是综合考察了字符串的多种操作。</font>**

**<font style="color:rgb(44, 62, 80);"></font>**

<font style="color:rgb(44, 62, 80);">一些同学会使用split库函数，分隔单词，然后定义一个新的string字符串，最后再把单词倒序相加，那么这道题题目就是一道水题了，失去了它的意义。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">所以这里我还是提高一下本题的难度：</font>**<font style="color:rgb(44, 62, 80);">不要使用辅助空间，空间复杂度要求为O(1)。</font>**

**<font style="color:rgb(44, 62, 80);"></font>**

<font style="color:rgb(44, 62, 80);">想一下，我们将整个字符串都反转过来，那么单词的顺序指定是倒序了，只不过单词本身也倒序了，那么再把单词反转一下，单词不就正过来了。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">所以解题思路如下：</font>

+ <font style="color:rgb(44, 62, 80);">移除多余空格</font>
+ <font style="color:rgb(44, 62, 80);">将整个字符串反转</font>
+ <font style="color:rgb(44, 62, 80);">将每个单词反转</font>



<font style="color:rgb(44, 62, 80);">举个例子，源字符串为："the sky is blue "</font>

+ <font style="color:rgb(44, 62, 80);">移除多余空格 : "the sky is blue"</font>
+ <font style="color:rgb(44, 62, 80);">字符串反转："eulb si yks eht"</font>
+ <font style="color:rgb(44, 62, 80);">单词反转："blue is sky the"</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">这样我们就完成了翻转字符串里的单词。</font>

```javascript
/**
 * @param {string} s
 * @return {string}
 */
var reverseWords = function(s) {
    // 字符串转数组
    const arr = Array.from(s);

    // 删除多余空格
    removeExtraSpaces(arr);

    // 翻转
    reverse(arr, 0, arr.length - 1);

    let start = 0;

    for (let i = 0;i <= arr.length;i++) {
        if (arr[i] === ' ' || i === arr.length) {
            // 翻转单词
            reverse(arr, start, i - 1);
            start = i + 1;
        }
    }

    return arr.join('');
};

// 删除多余空格
function removeExtraSpaces(arr) {
    let fast = 0,
        slow = 0;
    
    while (fast < arr.length) {
        if (arr[fast] === ' ' && (fast === 0 || arr[fast - 1] === ' ')) {
            fast++;
        } else {
            arr[slow++] = arr[fast++];
        }
    }
    // 移除尾部空格
    arr.length = arr[slow - 1] === ' ' ? slow - 1 : slow;
}

// 翻转从l到r的字符
function reverse(arr, l, r) {
    while (l < r) {
      // 交换
        [arr[l], arr[r]] = [arr[r], arr[l]];
        l++,
        r--;
    }
}
```
