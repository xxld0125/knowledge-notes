### 一、例题-[104. 二叉树的最大深度](https://leetcode.cn/problems/maximum-depth-of-binary-tree/)
<font style="color:rgb(44, 62, 80);">给定一个二叉树，找出其最大深度。</font>

<font style="color:rgb(44, 62, 80);">二叉树的深度为根节点到最远叶子节点的最长路径上的节点数。</font>

<font style="color:rgb(44, 62, 80);">说明: 叶子节点是指没有子节点的节点。</font>

<font style="color:rgb(44, 62, 80);">示例： 给定二叉树 [3,9,20,null,null,15,7]，</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1652966041131-30d38946-e9cc-4834-8236-1becd639692d.png)

<font style="color:rgb(44, 62, 80);">返回它的最大深度 3 。</font>

<font style="color:rgb(44, 62, 80);"></font>

#### <font style="color:rgb(44, 62, 80);">思路</font>
#### 递归法
<font style="color:rgb(44, 62, 80);">可以使用前序（中左右），也可以使用后序遍历（左右中），使用前序求的就是深度，使用后序求的是高度。</font>

**<font style="color:rgb(44, 62, 80);">而根节点的高度就是二叉树的最大深度</font>**<font style="color:rgb(44, 62, 80);">，所以本题中我们通过后序求的根节点高度来求的二叉树最大深度。</font>



1.确定递归函数的参数和返回值: 参数就是传入树的根节点, 返回就是这棵树的深度<font style="color:rgb(44, 62, 80);">。</font>

2.确定终止条件: 如果为空节点,就返回0, 表示高度0<font style="color:rgb(44, 62, 80);">。</font>

3.确定单层递归的逻辑: 先求它的左子树的深度, 再求它的右子树的深度, 最后取左右深度最大的数值在 + 1(加上当前节点), 就是当前节点为根节点的树的深度<font style="color:rgb(44, 62, 80);">。</font>

<font style="color:rgb(44, 62, 80);"></font>

#### 迭代法-层序遍历
<font style="color:rgb(44, 62, 80);">使用迭代法的话，使用层序遍历是最为合适的，因为最大的深度就是二叉树的层数，和层序遍历的方式极其吻合。</font>

```javascript
/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {TreeNode} root
 * @return {number}
 */
var maxDepth = function(root) {
    let res = 0;
    if (root === null) return res;
    const queue = [];
    queue.push(root);

    while (queue.length) {
        res++;
        let len = queue.length;
        while (len--) {
            const node = queue.shift();
            node.left && queue.push(node.left);
            node.right && queue.push(node.right);
        }
    }
    return res;
};
```





### 二、[559. N 叉树的最大深度](https://leetcode.cn/problems/maximum-depth-of-n-ary-tree/)
#### 1、递归-后序遍历
```javascript
/**
 * // Definition for a Node.
 * function Node(val,children) {
 *    this.val = val;
 *    this.children = children;
 * };
 */

/**
 * @param {Node|null} root
 * @return {number}
 */
var maxDepth = function(root) {
    // 一、后序
    if (root === null) return 0;

    let depth = 0;
    for (let i = 0;i < root.children.length;i++) {
        depth = Math.max(depth, maxDepth(root.children[i]));
    }

    return depth + 1;
};
```



#### 2、递归-前序遍历

        depth++;
        let l = queue.length;

            for (let i = 0;i < len;i++) {
                queue.push(node.children[i]);
            }
        }
    }
    return depth;
};

#### 3、迭代-层序遍历
```javascript
/**
 * // Definition for a Node.
 * function Node(val,children) {
 *    this.val = val;
 *    this.children = children;
 * };
 */

/**
 * @param {Node|null} root
 * @return {number}
 */
var maxDepth = function(root) {
    const getDepth = (node, depth) => {
        res = Math.max(res, depth);
        for (let i = 0;i < node.children.length;i++) {
            if (node.children[i]) {
                getDepth(node.children[i], depth + 1);
            }
        }
        return;
    }

    let res = 0;
    if (root === null) return res;

    getDepth(root, 1);
    return res;
};
```
