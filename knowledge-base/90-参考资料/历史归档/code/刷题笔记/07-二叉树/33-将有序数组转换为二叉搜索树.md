### 一、[108. 将有序数组转换为二叉搜索树](https://leetcode.cn/problems/convert-sorted-array-to-binary-search-tree/)
<font style="color:rgb(44, 62, 80);">将一个按照升序排列的有序数组，转换为一棵高度平衡二叉搜索树。</font>

<font style="color:rgb(44, 62, 80);">本题中，一个高度平衡二叉树是指一个二叉树每个节点 的左右两个子树的高度差的绝对值不超过 1。</font>

<font style="color:rgb(44, 62, 80);">示例:</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1656569314305-f485e211-a7e6-44da-b4cd-37c0ad21b508.png)



#### 思路
**<font style="color:rgb(44, 62, 80);">本质就是寻找分割点，分割点作为当前节点，然后递归左区间和右区间</font>**<font style="color:rgb(44, 62, 80);">。</font>

<font style="color:rgb(44, 62, 80);">本题其实要比</font>[19-从中序与后序遍历序列构造二叉树](https://www.yuque.com/u25370234/kb/ytxki7)<font style="color:rgb(44, 62, 80);">和</font><font style="color:rgb(44, 62, 80);"> </font>[20-最大二叉树](https://www.yuque.com/u25370234/kb/ytxki7)<font style="color:rgb(44, 62, 80);">简单一些，因为有序数组构造二叉搜索树，寻找分割点就比较容易了。</font>

<font style="color:rgb(44, 62, 80);">分割点就是数组中间位置的节点。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">那么为问题来了，如果数组长度为偶数，中间节点有两个，取哪一个？</font>

<font style="color:rgb(44, 62, 80);">取哪一个都可以，只不过构成了不同的平衡二叉搜索树。</font>

<font style="color:rgb(44, 62, 80);">例如：输入：[-10,-3,0,5,9]</font>

<font style="color:rgb(44, 62, 80);">如下两棵树，都是这个数组的平衡二叉搜索树：</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1656570051621-891c90f9-bfc9-4ea1-817d-56c17afe431c.png)

<font style="color:rgb(44, 62, 80);">如果要分割的数组长度为偶数的时候，中间元素为两个，是取左边元素 就是树1，取右边元素就是树2。</font>

**<font style="color:rgb(44, 62, 80);">这也是题目中强调答案不是唯一的原因。 理解这一点，这道题目算是理解到位了</font>**<font style="color:rgb(44, 62, 80);">。</font>

<font style="color:rgb(44, 62, 80);"></font>

#### 递归
+ 确定递归函数返回值及其参数

<font style="color:rgb(44, 62, 80);">删除二叉树节点，增加二叉树节点，都是用递归函数的返回值来完成，这样是比较方便的。</font>

<font style="color:rgb(44, 62, 80);">本题要构造二叉树，依然用递归函数的返回值来构造中节点的左右孩子。</font>

<font style="color:rgb(44, 62, 80);">再来看参数，首先是传入数组，然后就是左下标left和右下标right，我们在</font>[19-从中序与后序遍历序列构造二叉树](https://www.yuque.com/u25370234/kb/ytxki7)<font style="color:rgb(44, 62, 80);">中提过，在构造二叉树的时候尽量不要重新定义左右区间数组，而是用下标来操作原数组。</font>

```javascript
traversal(nums, left, right) {};
```

+ 确定终止条件

<font style="color:rgb(44, 62, 80);">这里定义的是左闭右闭的区间，所以当区间 </font>`<font style="color:rgb(44, 62, 80);">left > right</font>`<font style="color:rgb(44, 62, 80);">的时候，就是空节点了。</font>  

```javascript
if (left > right) return null;
```

+ 确定单层递归逻辑

<font style="color:rgb(44, 62, 80);">首先取数组中间元素的位置，不难写出</font>`<font style="color:rgb(71, 101, 130);">const mid = (left + right) / 2</font>`<font style="color:rgb(44, 62, 80);">，</font>**<font style="color:rgb(44, 62, 80);">这么写其实有一个问题，就是数值越界，例如</font>**`**<font style="color:rgb(44, 62, 80);">left</font>**`**<font style="color:rgb(44, 62, 80);">和</font>**`**<font style="color:rgb(44, 62, 80);">right</font>**`**<font style="color:rgb(44, 62, 80);">都是最大int，这么操作就越界了，在二分法中尤其需要注意！</font>**

<font style="color:rgb(44, 62, 80);">取了中间位置，就开始以中间位置的元素构造节点，代码</font>`<font style="color:rgb(44, 62, 80);">const root = new TreeNode(nums[mid])</font>`<font style="color:rgb(44, 62, 80);">，接着划分区间，</font>`<font style="color:rgb(44, 62, 80);">root</font>`<font style="color:rgb(44, 62, 80);">的左孩子接住下一层左区间的构造节点，右孩子接住下一层右区间构造的节点。</font>

<font style="color:rgb(44, 62, 80);">最后返回root节点，单层递归整体代码如下：</font>

```javascript
const mid = left + ((right - left) / 2);
const root = new TreeNode(nums[mid]);
root.left = traversal(nums, left, mid - 1);
root.right = traversal(nums, mid + 1, right);

return root;
```

<font style="color:rgb(44, 62, 80);">这里</font>`const mid = left + ((right - left) / 2)`，<font style="color:rgb(44, 62, 80);">的写法相当于是如果数组长度为偶数，中间位置有两个元素，取靠左边的。</font>

<font style="color:rgb(44, 62, 80);"></font>

**整体代码**

```javascript
function traversal(nums, left, right) {
  if (left > right) return null;
  const mid = left + ((right - left) / 2);
  const root = new TreeNode(nums[mid]);
  root.left = traversal(nums, left, mid - 1);
  root.right = traversal(nums, mid + 1, right);
  return root;
}

function sortedArrayToBST(nums) {
  const root = traversal(nums, 0, nums.length - 1);
  return root;
}
```

#### <font style="color:rgb(44, 62, 80);">迭代法</font>
<font style="color:rgb(44, 62, 80);">迭代法可以通过三个队列来模拟，一个队列放遍历的节点，一个队列放左区间下标，一个队列放右区间下标。</font>

```javascript
var sortedArrayToBST = function(nums) {
    if(nums.length===0){
        return null;
    }
    let root=new TreeNode(0); //初始根节点
    let nodeQue=[root]; //放遍历的节点,并初始化
    let leftQue=[0]; //放左区间的下标,初始化
    let rightQue=[nums.length-1]; // 放右区间的下标
    
    while(nodeQue.length){
        let curNode=nodeQue.pop();
        let left=leftQue.pop();
        let right=rightQue.pop();
        let mid=left+Math.floor((right-left)/2);
        
        curNode.val=nums[mid]; //将下标为mid的元素给中间节点
        
        // 处理左区间
        if(left<=mid-1){
            curNode.left=new TreeNode(0);
            nodeQue.push(curNode.left);
            leftQue.push(left);
            rightQue.push(mid-1);
        }
        
        // 处理右区间
        if(right>=mid+1){
            curNode.right=new TreeNode(0);
            nodeQue.push(curNode.right);
            leftQue.push(mid+1);
            rightQue.push(right);
        }
    }
    return root;
};
```



### 总结
**<font style="color:rgb(44, 62, 80);">在</font>**[19-从中序与后序遍历序列构造二叉树](https://www.yuque.com/u25370234/kb/ytxki7)<font style="color:rgb(44, 62, 80);">和 </font>[20-最大二叉树](https://www.yuque.com/u25370234/kb/ytxki7)**<font style="color:rgb(44, 62, 80);">之后，我们顺理成章的应该构造一下二叉搜索树了，一不小心还是一棵平衡二叉搜索树</font>**<font style="color:rgb(44, 62, 80);">。</font>

<font style="color:rgb(44, 62, 80);">思路也是一样的，不断中间分割，然后递归处理左区间，右区间，也可以说是分治。</font>

<font style="color:rgb(44, 62, 80);">此时相信大家应该对通过递归函数的返回值来增删二叉树很熟悉了，这也是常规操作。</font>

<font style="color:rgb(44, 62, 80);">在定义区间的过程中我们又一次强调了循环不变量的重要性。</font>

<font style="color:rgb(44, 62, 80);">最后依然给出迭代的方法，其实就是模拟取中间元素，然后不断分割去构造二叉树的过程。</font>
