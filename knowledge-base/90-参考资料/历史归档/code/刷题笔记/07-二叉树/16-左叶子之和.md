<font style="color:rgb(44, 62, 80);">计算给定二叉树的所有左叶子之和。</font>

#### 思路
**<font style="color:rgb(44, 62, 80);">首先要注意是判断左叶子，不是二叉树左侧节点，所以不要上来想着层序遍历。</font>**

**<font style="color:rgb(44, 62, 80);"></font>**

<font style="color:rgb(44, 62, 80);">因为题目中其实没有说清楚左叶子究竟是什么节点，那么我来给出左叶子的明确定义：</font>**<font style="color:rgb(44, 62, 80);">如果左节点不为空，且左节点没有左右孩子，那么这个节点的左节点就是左叶子</font>****<font style="color:rgb(44, 62, 80);">。</font>**

  


<font style="color:rgb(44, 62, 80);">大家思考一下如下图中二叉树，左叶子之和究竟是多少？</font>

**<font style="color:rgb(44, 62, 80);"></font>**

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1654336321117-799fd3a1-6b84-4ae8-9922-48e79dada2f5.png)



**<font style="color:rgb(44, 62, 80);">其实是0，因为这棵树根本没有左叶子！</font>**

**<font style="color:rgb(44, 62, 80);"></font>**

<font style="color:rgb(44, 62, 80);">那么</font>**<font style="color:rgb(44, 62, 80);">判断当前节点是不是左叶子是无法判断的，必须要通过节点的父节点来判断其左孩子是不是左叶子。</font>**

<font style="color:rgb(44, 62, 80);">如果该节点的左节点不为空，该节点的左节点的左节点为空，该节点的左节点的右节点为空，则找到了一个左叶子，判断代码如下：</font>

```javascript
if (node.left !== null && node->left->left == NULL && node->left->right == NULL) {
    左叶子节点处理逻辑
}
```



#### 一、递归法
<font style="color:rgb(44, 62, 80);">因为要通过递归函数的返回值来累加求取左叶子数值之和, 所以递归的遍历顺序是后序遍历(左右中)。</font>



##### 1、确定递归函数的参数和返回值
<font style="color:rgb(44, 62, 80);">判断一个树的左叶子节点之和，那么一定要传入树的根节点，递归函数的返回值为数值之和，所以为使用题目中给出的函数就可以了。</font>

<font style="color:rgb(44, 62, 80);"></font>

##### 2、确定终止条件
```javascript
if (root === null) return 0;
```



##### 3、确定单层递归逻辑
<font style="color:rgb(44, 62, 80);">当遇到左叶子节点的时候，记录数值，然后通过递归求取左子树左叶子之和，和 右子树左叶子之和，相加便是整个树的左叶子之和。</font>

```javascript
let leftValue = sumOfLeftLeaves(root.left);
let rightValue = sumOfLeftLeaves(root.right);

let midValue = 0;

if (root.left !== null && root.left.left === null && root.left.right === null) {
  midValue = root.left.val;
}

return midValue + leftValue + rightValue;
```





##### 4、整体代码
```javascript
/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {TreeNode} root
 * @return {number}
 */
var sumOfLeftLeaves = function(root) {
    if (root === null) return 0;

    let leftValue = sumOfLeftLeaves(root.left);
    let rightValue = sumOfLeftLeaves(root.right);
    
    let midValue = 0;

    if (root.left !== null && root.left.left === null && root.left.right === null) {
        midValue = root.left.val;
    }

    return midValue + leftValue + rightValue;
};
```



#### 二、迭代法
```javascript
/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {TreeNode} root
 * @return {number}
 */
var sumOfLeftLeaves = function(root) {
    let res = 0;
    if (root === null) return res;
    const stack = [root];

    while (stack.length) {
        let node = stack.pop();
        if (node === null) {
            node = stack.pop();
            if (node.left !== null && node.left.left === null && node.left.right === null) {
                res += node.left.val;
            }
            continue;
        }

        stack.push(node);
        stack.push(null);
        node.right && stack.push(node.right);
        node.left && stack.push(node.left);
    }
    return res;
};
```
