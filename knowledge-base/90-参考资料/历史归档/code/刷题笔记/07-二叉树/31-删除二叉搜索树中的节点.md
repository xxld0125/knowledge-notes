### 一、[450. 删除二叉搜索树中的节点](https://leetcode.cn/problems/delete-node-in-a-bst/)
<font style="color:rgb(44, 62, 80);">给定一个二叉搜索树的根节点 root 和一个值 key，删除二叉搜索树中的 key 对应的节点，并保证二叉搜索树的性质不变。返回二叉搜索树（有可能被更新）的根节点的引用。</font>

<font style="color:rgb(44, 62, 80);">一般来说，删除节点可分为两个步骤：</font>

<font style="color:rgb(44, 62, 80);">首先找到需要删除的节点； 如果找到了，删除它。 说明： 要求算法时间复杂度为 $O(h)$，h 为树的高度。</font>

#### 思路
<font style="color:rgb(44, 62, 80);">搜索树的节点删除要比节点增加复杂的多，有很多情况需要考虑。</font>

<font style="color:rgb(44, 62, 80);"></font>

#### 递归
+ 确定递归函数参数以及返回值

```javascript
function deleteNode(root, key) {}
```

+ 确定终止条件

遇到空返回，其实这也说明没有找打删除的节点，遍历到空姐点直接返回。

```javascript
if (root === null) return root;
```

+ 确定单层递归逻辑

有五种情况

    - 没找到删除的节点，遍历到空节点直接返回
    - 找到删除的节点
        * 左右孩子都为空， 直接删除节点，返回`null`为根节点。
        * 删除节点的左孩子为空，右孩子不为空，删除节点，右孩子补位，返回右孩子为跟节点。
        * 删除节点的右孩子为空，左孩子不为空，删除节点，左孩子补位，返回左孩子为跟节点。
        * 左右孩子均不为空，则将删除节点的左子树节点放到删除节点的右子树的最左面节点的左孩子上，返回删除节点右孩子为新的节点。

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">最后一种比较难理解</font>

<font style="color:rgb(44, 62, 80);"></font>

整体代码

```javascript
/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {TreeNode} root
 * @param {number} key
 * @return {TreeNode}
 */
var deleteNode = function(root, key) {
    // 第一种情况：没找到删除的节点，遍历到空节点直接返回了
    if (root === null) return root;
    if (root.val === key) {
        // 第二种情况：左右孩子都为空（叶子节点），直接删除节点， 返回NULL为根节点
        if (root.left === null && root.right === null) return null;
        // 第三种情况：其左孩子为空，右孩子不为空，删除节点，右孩子补位 ，返回右孩子为根节点
        else if (root.left === null) return root.right;
        // 第四种情况：其右孩子为空，左孩子不为空，删除节点，左孩子补位，返回左孩子为根节点
        else if (root.right === null) return root.left;
        // 第五种情况：左右孩子节点都不为空，则将删除节点的左子树放到删除节点的右子树的最左面节点的左孩子的位置
        // 并返回删除节点右孩子为新的根节点。
        else {
            // 找右子树最左面的节点
            let cur = root.right;
            while (cur.left) {
                cur = cur.left;
            }
            // 把要删除的节点（root）左子树放在cur的左孩子的位置
            cur.left = root.left;
            // 返回旧root的右孩子作为新root
            root = root.right;
            return root;
        }
    }
    if (root.val > key) root.left = deleteNode(root.left, key);
    if (root.val < key) root.right = deleteNode(root.right, key);

    return root;
};
```



#### 普通二叉树的删除方式
<font style="color:rgb(44, 62, 80);">这里我在介绍一种通用的删除，普通二叉树的删除方式（没有使用搜索树的特性，遍历整棵树），用交换值的操作来删除目标节点。</font>

<font style="color:rgb(44, 62, 80);">代码中目标节点（要删除的节点）被操作了两次：</font>

+ <font style="color:rgb(44, 62, 80);">第一次是和目标节点的右子树最左面节点交换。</font>
+ <font style="color:rgb(44, 62, 80);">第二次直接被</font>`<font style="color:rgb(44, 62, 80);">null</font>`<font style="color:rgb(44, 62, 80);">覆盖了。</font>

```javascript
function deleteNode(root, key) {
  if (root === null) return root;
  if (root.val === key) {
    if (root.right === null) return root.left;// 这里第二次操作目标值：最终删除的作用
```

#### <font style="color:rgb(44, 62, 80);">迭代法</font>


```javascript
var deleteNode = function (root, key) {
    const deleteOneNode = target => {
        if (!target) return target
        if (!target.right) return target.left
        let cur = target.right
        while (cur.left) {
            cur = cur.left
        }
        cur.left = target.left
        return target.right
    }

    if (!root) return root
    let cur = root
    let pre = null
    while (cur) {
        if (cur.val === key) break
        pre = cur
        cur.val > key ? cur = cur.left : cur = cur.right
    }
    if (!pre) {
        return deleteOneNode(cur)
    }
    if (pre.left && pre.left.val === key) {
        pre.left = deleteOneNode(cur)
    }
    if (pre.right && pre.right.val === key) {
        pre.right = deleteOneNode(cur)
    }
    return root
}
```



### 总结
<font style="color:rgb(44, 62, 80);">大家会发现二叉搜索树删除节点比增加节点复杂的多。</font>

**<font style="color:rgb(44, 62, 80);">因为二叉搜索树添加节点只需要在叶子上添加就可以的，不涉及到结构的调整，而删除节点操作涉及到结构的调整</font>**<font style="color:rgb(44, 62, 80);">。</font>

<font style="color:rgb(44, 62, 80);">这里我们依然使用递归函数的返回值来完成把节点从二叉树中移除的操作。</font>

**<font style="color:rgb(44, 62, 80);">这里最关键的逻辑就是第五种情况（删除一个左右孩子都不为空的节点），这种情况一定要想清楚</font>**<font style="color:rgb(44, 62, 80);">。</font>

<font style="color:rgb(44, 62, 80);">而且就算想清楚了，对应的代码也未必可以写出来，所以</font>**<font style="color:rgb(44, 62, 80);">这道题目即考察思维逻辑，也考察代码能力</font>**<font style="color:rgb(44, 62, 80);">。</font>

<font style="color:rgb(44, 62, 80);">递归中我给出了两种写法，推荐大家学会第一种（利用搜索树的特性）就可以了，第二种递归写法其实是比较绕的。</font>

<font style="color:rgb(44, 62, 80);">最后我也给出了相应的迭代法，就是模拟递归法中的逻辑来删除节点，但需要一个pre记录cur的父节点，方便做删除操作。</font>
