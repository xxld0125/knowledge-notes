### 一、[617. 合并二叉树](https://leetcode.cn/problems/merge-two-binary-trees/)
<font style="color:rgb(44, 62, 80);">给定两个二叉树，想象当你将它们中的一个覆盖到另一个上时，两个二叉树的一些节点便会重叠。</font>

<font style="color:rgb(44, 62, 80);">你需要将他们合并为一个新的二叉树。合并的规则是如果两个节点重叠，那么将他们的值相加作为节点合并后的新值，否则不为 NULL 的节点将直接作为新二叉树的节点。</font>

<font style="color:rgb(44, 62, 80);">示例 1:</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1654576412342-5cacaee8-c07a-4b58-ba15-c53d8736dac4.png)

<font style="color:rgb(44, 62, 80);">注意: 合并必须从两个树的根节点开始。</font>



#### <font style="color:rgb(38, 38, 38);">思路</font>
<font style="color:rgb(44, 62, 80);">相信这道题目很多同学疑惑的点是如何同时遍历两个二叉树呢？</font>

<font style="color:rgb(44, 62, 80);">其实和遍历一个树逻辑是一样的，只不过传入两个树的节点，同时操作。</font>



#### 递归
<font style="color:rgb(44, 62, 80);">二叉树使用递归，就要想使用前中后哪种遍历方式？</font>

**<font style="color:rgb(44, 62, 80);">本题使用哪种遍历都是可以的！</font>**

<font style="color:rgb(44, 62, 80);">我们下面以前序遍历为例。</font>

<font style="color:rgb(44, 62, 80);">动画如下：</font>

#### ![](https://cdn.nlark.com/yuque/0/2022/gif/25743026/1654577934811-aeb48ea3-f1f3-4e50-919d-de78aca9b4d1.gif)
<font style="color:rgb(44, 62, 80);">那么我们来按照递归三部曲来解决：</font>

**<font style="color:rgb(44, 62, 80);">1.确定递归函数的参数和返回值：</font>**

<font style="color:rgb(44, 62, 80);">首先那么要合入两个二叉树，那么参数至少是要传入两个二叉树的根节点，返回值就是合并之后二叉树的根节点。</font>

<font style="color:rgb(44, 62, 80);">代码如下：</font>

```javascript
function mergeTrees(root1, root2) {
  return root1;
}
```

**<font style="color:rgb(44, 62, 80);"></font>**

**<font style="color:rgb(44, 62, 80);">2.确定终止条件：</font>**

<font style="color:rgb(44, 62, 80);">因为是传入了两个树，那么就有两个树遍历的节点</font>`<font style="color:rgb(44, 62, 80);">t1</font>`<font style="color:rgb(44, 62, 80);"> 和 </font>`<font style="color:rgb(44, 62, 80);">t2</font>`<font style="color:rgb(44, 62, 80);">，如果</font>`<font style="color:rgb(44, 62, 80);">t1 === null </font>`<font style="color:rgb(44, 62, 80);">了，两个树合并就应该是 </font>`<font style="color:rgb(44, 62, 80);">t2</font>`<font style="color:rgb(44, 62, 80);"> 了啊（如果</font>`<font style="color:rgb(44, 62, 80);">t2</font>`<font style="color:rgb(44, 62, 80);">也为</font>`<font style="color:rgb(44, 62, 80);">null</font>`<font style="color:rgb(44, 62, 80);">也无所谓，合并之后就是</font>`<font style="color:rgb(44, 62, 80);">null</font>`<font style="color:rgb(44, 62, 80);">）。</font>

<font style="color:rgb(44, 62, 80);">反过来如果</font>`<font style="color:rgb(44, 62, 80);">t2 === null</font>`<font style="color:rgb(44, 62, 80);">，那么两个数合并就是</font>`<font style="color:rgb(44, 62, 80);">t1</font>`<font style="color:rgb(44, 62, 80);">（如果</font>`<font style="color:rgb(44, 62, 80);">t1</font>`<font style="color:rgb(44, 62, 80);">也为</font>`<font style="color:rgb(44, 62, 80);">null</font>`<font style="color:rgb(44, 62, 80);">也无所谓，合并之后就是</font>`<font style="color:rgb(44, 62, 80);">null</font>`<font style="color:rgb(44, 62, 80);">）。</font>

<font style="color:rgb(44, 62, 80);">代码如下：</font>

```javascript
if (t1 === null) return t2; // 如果t1为空，合并之后就应该是t2
if (t2 === null) return t1; // 如果t2为空，合并之后就应该是t1
```

**<font style="color:rgb(44, 62, 80);"></font>**

**<font style="color:rgb(44, 62, 80);">3.确定单层递归的逻辑：</font>**

<font style="color:rgb(44, 62, 80);">单层递归的逻辑就比较好些了，这里我们用重复利用一下t1这个树，t1就是合并之后树的根节点（就是修改了原来树的结构）。</font>

<font style="color:rgb(44, 62, 80);">那么单层递归中，就要把两棵树的元素加到一起。</font>

```javascript
t1.val += t2.val;
```

<font style="color:rgb(44, 62, 80);">接下来t1 的左子树是：合并 t1左子树 t2左子树之后的左子树。</font>

<font style="color:rgb(44, 62, 80);">t1 的右子树：是 合并 t1右子树 t2右子树之后的右子树。</font>

<font style="color:rgb(44, 62, 80);">最终t1就是合并之后的根节点。</font>

<font style="color:rgb(38, 38, 38);">代码如下：</font>

```javascript
t1.left = mergeTrees(t1.left, t2.left);
t1.right = mergeTrees(t1.right, t2.right);
return t1;
```

<font style="color:rgb(44, 62, 80);">此时前序遍历，完整代码就写出来了，如下：</font>

```javascript
/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {TreeNode} root1
 * @param {TreeNode} root2
 * @return {TreeNode}
 */
var mergeTrees = function(root1, root2) {
    if (root1 === null) return root2;
    if (root2 === null) return root1;

    root1.val += root2.val;

    root1.left = mergeTrees(root1.left, root2.left);
    root1.right = mergeTrees(root1.right, root2.right);

    return root1;
};
```



中序和后续遍历类似， 在此就不再介绍



#### 迭代
可以使用层序遍历合并两个数

```javascript
/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {TreeNode} root1
 * @param {TreeNode} root2
 * @return {TreeNode}
 */
var mergeTrees = function(root1, root2) {
    if (root1 === null) return root2;
    if (root2 === null) return root1;

    const queue = [root1, root2];

    while (queue.length) {
        let node1 = queue.shift();
        let node2 = queue.shift();

        node1.val += node2.val;

        if (node1.left !== null && node2.left !== null) {
            queue.push(node1.left);
            queue.push(node2.left);
        }
        if (node1.right !== null && node2.right !== null) {
            queue.push(node1.right);
            queue.push(node2.right);
        }

        if (node1.left === null && node2.left !== null) {
            node1.left = node2.left;
        }
        if (node1.right === null && node2.right !== null) {
            node1.right = node2.right;
        }
    }

    return root1;
};
```



#### 总结
<font style="color:rgb(44, 62, 80);">合并二叉树，也是二叉树操作的经典题目，如果没有接触过的话，其实并不简单，因为我们习惯了操作一个二叉树，一起操作两个二叉树，还会有点懵懵的。</font>

<font style="color:rgb(44, 62, 80);">这不是我们第一次操作两棵二叉树了，在</font>[08-对称二叉树](https://www.yuque.com/u25370234/kb/yqhatu)<font style="color:rgb(44, 62, 80);">中也一起操作了两棵二叉树。</font>

<font style="color:rgb(44, 62, 80);">迭代法中，一般一起操作两个树都是使用队列模拟类似层序遍历，同时处理两个树的节点，这种方式最好理解，如果用模拟递归的思路的话，要复杂一些。</font>
