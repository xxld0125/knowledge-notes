### 一、例题-[20. 有效的括号](https://leetcode.cn/problems/valid-parentheses/)
<font style="color:rgb(44, 62, 80);">给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串，判断字符串是否有效。</font>

<font style="color:rgb(44, 62, 80);">有效字符串需满足：</font>

+ <font style="color:rgb(44, 62, 80);">左括号必须用相同类型的右括号闭合。</font>
+ <font style="color:rgb(44, 62, 80);">左括号必须以正确的顺序闭合。</font>
+ <font style="color:rgb(44, 62, 80);">注意空字符串可被认为是有效字符串。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">示例 1:</font>

+ <font style="color:rgb(44, 62, 80);">输入: "()"</font>
+ <font style="color:rgb(44, 62, 80);">输出: true</font>

<font style="color:rgb(44, 62, 80);">示例 2:</font>

+ <font style="color:rgb(44, 62, 80);">输入: "()[]{}"</font>
+ <font style="color:rgb(44, 62, 80);">输出: true</font>

<font style="color:rgb(44, 62, 80);">示例 3:</font>

+ <font style="color:rgb(44, 62, 80);">输入: "(]"</font>
+ <font style="color:rgb(44, 62, 80);">输出: false</font>

<font style="color:rgb(44, 62, 80);">示例 4:</font>

+ <font style="color:rgb(44, 62, 80);">输入: "([)]"</font>
+ <font style="color:rgb(44, 62, 80);">输出: false</font>

<font style="color:rgb(44, 62, 80);">示例 5:</font>

+ <font style="color:rgb(44, 62, 80);">输入: "{[]}"</font>
+ <font style="color:rgb(44, 62, 80);">输出: true</font>



#### 思路
<font style="color:rgb(44, 62, 80);">由于栈结构的特殊性，</font>**<font style="color:rgb(44, 62, 80);">非常适合做对称匹配类的题目</font>**<font style="color:rgb(44, 62, 80);">。</font>

<font style="color:rgb(44, 62, 80);">首先要弄清楚，字符串里的括号不匹配有几种情况。</font>

**<font style="color:rgb(44, 62, 80);">一些同学，在面试中看到这种题目上来就开始写代码，然后就越写越乱。</font>**

<font style="color:rgb(44, 62, 80);">建议要写代码之前要分析好有哪几种不匹配的情况，如果不动手之前分析好，写出的代码也会有很多问题。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">先来分析一下 这里有三种不匹配的情况，</font>

1. <font style="color:rgb(44, 62, 80);">第一种情况，字符串里左方向的括号多余了 ，所以不匹配。</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1652705295112-5f306ca6-94a5-44d6-b8ec-7675502472f2.png)

2. <font style="color:rgb(44, 62, 80);">第二种情况，括号没有多余，但是 括号的类型没有匹配上。 </font>



3. <font style="color:rgb(44, 62, 80);">第三种情况，字符串里右方向的括号多余了，所以不匹配。</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1652705321490-94d75f3e-bd4f-46e3-a57d-5d04a85a61d0.png)

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">所以我们的代码只要覆盖了这三种不匹配的情况，就不会出问题，可以看出 动手之前分析好题目的重要性。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">第一种情况：已经遍历完了字符串，但是栈不为空，说明有相应的左括号没有右括号来匹配，所以return false</font>

<font style="color:rgb(44, 62, 80);">第二种情况：遍历字符串匹配的过程中，发现栈里没有要匹配的字符。所以return false</font>

<font style="color:rgb(44, 62, 80);">第三种情况：遍历字符串匹配的过程中，栈已经为空了，没有匹配的字符了，说明右括号没有找到对应的左括号return false</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">那么什么时候说明左括号和右括号全都匹配了呢，就是字符串遍历完之后，栈是空的，就说明全都匹配了。</font>



```javascript
/**
 * @param {string} s
 * @return {boolean}
 */
var isValid = function(s) {
    const map = {
        '(': ')',
        '{': '}',
        '[': ']'
    }
    const stack = [];
    for (let k of s) {
        if (map.hasOwnProperty(k)) {
            stack.push(map[k]);
            continue;
        }
        if (stack.pop() !== k) {
            return false;
        }
    }
    return !stack.length;
};
```
