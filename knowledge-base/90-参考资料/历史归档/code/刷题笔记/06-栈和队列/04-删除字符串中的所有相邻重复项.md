### 一、例题-[1047. 删除字符串中的所有相邻重复项](https://leetcode.cn/problems/remove-all-adjacent-duplicates-in-string/)
<font style="color:rgb(44, 62, 80);">给出由小写字母组成的字符串 S，重复项删除操作会选择两个相邻且相同的字母，并删除它们。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">在 S 上反复执行重复项删除操作，直到无法继续删除。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">在完成所有重复项删除操作后返回最终的字符串。答案保证唯一。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">示例：</font>

+ <font style="color:rgb(44, 62, 80);">输入："abbaca"</font>
+ <font style="color:rgb(44, 62, 80);">输出："ca"</font>
+ <font style="color:rgb(44, 62, 80);">解释：例如，在 "abbaca" 中，我们可以删除 "bb" 由于两字母相邻且相同，这是此时唯一可以执行删除操作的重复项。之后我们得到字符串 "aaca"，其中又只有 "aa" 可以执行重复项删除操作，所以最后的字符串为 "ca"。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">提示：</font>

+ <font style="color:rgb(44, 62, 80);">1 <= S.length <= 20000</font>
+ <font style="color:rgb(44, 62, 80);">S 仅由小写英文字母组成。</font>

<font style="color:rgb(44, 62, 80);"></font>

#### <font style="color:rgb(44, 62, 80);">思路</font>
<font style="color:rgb(44, 62, 80);">本题要删除相邻相同元素，其实也是匹配问题，相同左元素相当于左括号，相同右元素就是相当于右括号，匹配上了就删除。</font>

<font style="color:rgb(44, 62, 80);">那么再来看一下本题：可以把字符串顺序放到一个栈中，然后如果相同的话 栈就弹出，这样最后栈里剩下的元素都是相邻不相同的元素了。</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1652706175128-eb060974-1f0e-40c5-8e07-431aed447273.png)

####   
解法一: 使用栈
```javascript
/**
 * @param {string} s
 * @return {string}
 */
var removeDuplicates = function(s) {
    const stack = [];

    for (let k of s) {
        if (!stack.length) {
            stack.push(k);
            continue;
        }
        const pre = stack.pop();
        if (pre !== k) {
            stack.push(pre);
            stack.push(k);
        }
    }
    return stack.join('')
};
```

#### 解法二: 双指针(模拟栈)
```javascript
// 原地解法（双指针模拟栈）
var removeDuplicates = function(s) {
    s = [...s];
    let top = -1; // 指向栈顶元素的下标
    for(let i = 0; i < s.length; i++) {
        if(top === -1 || s[top] !== s[i]) { // top === -1 即空栈
            s[++top] = s[i]; // 入栈
        } else {
            top--; // 推出栈
        }
    }
    s.length = top + 1; // 栈顶元素下标 + 1 为栈的长度
    return s.join('');
};
```
