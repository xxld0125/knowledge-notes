### 什么是单调栈
<font style="color:rgb(51, 51, 51);">单调递增或单调减的栈，跟单调队列差不多，但是只用到它的一端，利用它可以用来解决一些ACM/ICPC和OI的题目，如RQNOJ 的诺诺的队列等。</font>

### 题目一:[503. 下一个更大元素 II](https://leetcode-cn.com/problems/next-greater-element-ii/)
#### 解法一: 单调栈
**思路**: 我们可以使用单调栈解决本题。单调栈中保存的是下标，从栈底到栈顶的下标在数组 nums 中对应的值是单调不升的。



每次我们移动到数组中的一个新的位置 i，我们就将当前单调栈中所有对应值小于nums[i] 的下标弹出单调栈，这些值的下一个更大元素即为 nums[i]（证明很简单：如果有更靠前的更大元素，那么这些位置将被提前弹出栈）。随后我们将位置 i 入栈。



但是注意到只遍历一次序列是不够的，例如序列 [2,3,1]，最后单调栈中将剩余 [3,1]，其中元素 [1]的下一个更大元素还是不知道的。



一个朴素的思想是，我们可以把这个循环数组「拉直」，即复制该序列的前 n−1 个元素拼接在原序列的后面。这样我们就可以将这个新序列当作普通序列，用上文的方法来处理。



<font style="color:rgb(38, 38, 38);">而在本题中，我们不需要显性地将该循环数组「拉直」，而只需要在处理时对下标取模即可。</font>

**代码**

```javascript
/**
 * @param {number[]} nums
 * @return {number[]}
 */
var nextGreaterElements = function(nums) {
    const n = nums.length;
    const res = new Array(n).fill(-1);
    const stack = [];
    for (let i = 0;i < 2 * n - 1;i++) {
        while (stack.length && nums[stack[stack.length - 1]] < nums[i % n]) {
            res[stack[stack.length - 1]] = nums[i % n];
            stack.pop();
        }
        stack.push(i % n);
    }
    return res;
};
```

#### 解法二:  双循环
**思路: 通过双循环进行遍历判断大小**

**代码**

```javascript
/**
* @param {number[]} nums
* @return {number[]}
*/
var nextGreaterElements = function(nums) {
  const res = [];
  const len = nums.length;
  for (let i = 0;i < len;i++) {
    for (let j = i + 1;j < i + len;j++) {
      const index  = j % len; 
      if (nums[index] > nums[i]) {
        res[i] = nums[index];
        break;
      };
    };
    if(res[i] === undefined) res[i] = -1;
  }
  return res;
};
```



### 题目二:[962. 最大宽度坡](https://leetcode-cn.com/problems/maximum-width-ramp/)
**思路: 要求 i < j 且 A[i] <= A[j]。这样的坡的宽度为 j - i, 找到最大坡度, 所以我们找到数组内的单调递减数组的下标, 并从尾部遍历, 与栈内存储的下标对应值进行比较,可以找出最大坡度.**

**注: 如果要求是i < j 且 A[i] >= A[j] , 则相同, 首先找到数组内的单调递增数组的下标,接下来的操作相同.**

**代码:**

```javascript
/**
 * @param {number[]} nums
 * @return {number}
 */
var maxWidthRamp = function(nums) {
    const stack = [];
    // 将递减元素的下标推入栈中
    for (let i = 0;i < nums.length;i++) {
        if (!stack.length || nums[i] < nums[stack[stack.length - 1]]) {
            stack.push(i);
        }
    }
    // 初始化最大坡度
    let max = 0;

    // 倒序遍历,找到最大坡度
    for (let i = nums.length - 1;i >= 0;i--) {
        while (stack.length && nums[stack[stack.length - 1]] <= nums[i]) {
            max = Math.max(max, i - stack.pop())
        }
    }
    return max;
};
```
