二分查找的运用是建立在数组有序的基础上的，如果数组无序，我们要先对数组进行排序，如果数组有多个维度，我们针对需要二分查找的维度进行排序。

### 一、红蓝二分法
这里介绍一种二分查找的方法:红蓝二分法.



#### 1、什么是红蓝二分法
`l` 指针掌管左边蓝色区域，`r` 指针掌管右边红色区域，两者互不冲突，通过不断向目标元素靠近扩大掌管区域，直到两者掌管区域接壤，即 `l +1 === r `时终止。

开始时，`l `指针和 `r` 指针取在搜索区间界外，`l = 首个元素下标−1`，`r = 末尾元素下标 + 1`，此时所有元素均未着色；

循环条件始终为 `l + 1 !== r,` 当` l+1 === r` 时跳出循环，此时蓝红区域划分完成，所有元素均已着色；

`mid` 指针取值始终为 `Math.floor((l + r) / 2) `,

`l` 指针和 `r` 指针变化的时候直接变为 `mid` 指针，无需 `+1` 或者 `-1 `；



<font style="color:#F5222D;">唯一需要会变化的地方就是判断目标元素最终落在</font><font style="color:#F5222D;">左边蓝色区域还是右边红色区域.</font>

<font style="color:#F5222D;"></font>



#### 2、使用中的注意点
1. **循环条件**

模板中循环条件判断也可写为 `l + 1 < r`，在不能保证 `l + 1 < r`在初始状态成立的情况下，就不能写` l + 1 !== r`，通常情况下写成`l + 1 !== r`是因为判等判不等操作肯定比判 `l + 1 < r` 即判大于小于快的（自行回忆计算机组成原理相关知识点或者自行模拟两种操作的位运算）。但保险起见，在 `l`指针和 `r `指针跨数组或者跨区间的情况下，建议统一采用 `l + 1 < r` 。

[1855. 下标对中的最大距离](https://leetcode-cn.com/problems/maximum-distance-between-a-pair-of-values/)



2. **搜索完成后的特殊状态**

若二分查找 `while` 循环跳出后 `l === 首个元素下标−1` ，即整个搜索区间都被染成红色，即左边蓝色区域长度为 0 ，如果搜索目标位于左边蓝色区域，则搜索结果不存在，一般需要进行后处理。

若二分查找 `while` 循环跳出后 `r === 末尾元素下标+1` ，即整个搜索区间都被染成蓝色，即右边红色区域长度为 0 ，如果搜索目标位于右边红色区域，则搜索结果不存在，一般需要进行后处理。



3. **排序**

<font style="color:rgb(38, 38, 38);">二分查找的运用是建立在数组有序的基础上的，如果数组无序，我们要先对数组进行排序，如果数组有多个维度，我们针对需要二分查找的维度进行排序。</font>

<font style="color:rgb(38, 38, 38);"></font>

4. **红蓝边界**

开始时，`l` 指针和 `r` 指针取在搜索区间界外，另一个重要作用就是构造了下标边界。

如果搜索的是整个 `nums` 数组，设数组长度为 `n` ，搜索下标范围`[0,n−1]`，蓝红的边界则为 `l=−1`, `r = n` 。

拓展一下，如果搜索的是 `nums` 数组中的 `[a,b]`区间，其中`[a,b]⊆[0,n−1]` ，蓝红的边界则为 `l=a−1` 则 `r = b+1`。

下标边界比迭代器写法更简洁，判断更方便，运行速度也更快，非常强大。



**例: **[**852. 山脉数组的峰顶索引**](https://leetcode-cn.com/problems/peak-index-in-a-mountain-array/)

此题按照上面方法介绍的`l = -1,` `r = n`,会报错;因为在此题中,最大值肯定不会在数组第一位或最后一位, 取值范围为`[1, n-2]`.所以`l` 和 `r`的初始值为`0`,  `n-1`.

也可以使用`l = -1,` `r = n`的边界条件, 当需要处理边界条件.即中间值`m`,需要和前一个值`pre`比较时,`m`为0的情况,此时需要手动给`pre`赋值`arr[0] - 1`。



5. **构造二分查找区间**

当问题能够转化为区间内二分判定问题的时候，构造搜索区间，在蓝红二分法中，设 `l` 指针初始为 `l<sub>0</sub>`，r 指针初始为 `r<sub>0</sub>`，二分查找结束时，`l` 指针终止为`l<sub>t</sub>`，`r` 指针终止为 `r<sub>t</sub>`，`l<sub>0</sub> ≤ l<sub>t</sub> < r<sub>t </sub>≤ r<sub>0</sub>`恒成立，同时 `target ∈ [l<sub>t</sub>, r<sub>t</sub>] ⊆ [l<sub>0</sub>, r<sub>0</sub>]`，双指针 `l` 和 `r` 始终对目标元素 `target` 进行夹逼，这是一条非常重要的性质。

6. **<font style="color:rgb(38, 38, 38);">逐步缩小二分查找区间</font>**

在二维有序的查找中，一般采用 线性查找（外层）+ 二分查找（内层） 或 二分查找（外层）+ 二分查找（内层） 方案，在有序的情况下，对每一次内层的二分查找，前一次蓝色区间 <font style="color:rgb(38, 38, 38);">⊂ </font>当前蓝色区间前一次蓝色区间 ⊂ 当前蓝色区间，当前 l 指针初始位置可以继承前一次二分查找完毕 l 指针的最终位置，同理，前一次红色区间 <font style="color:rgb(38, 38, 38);">⊂</font> 当前红色区间前一次红色区间 ⊂ 当前红色区间，当前 rr 指针初始位置可以继承前一次二分查找完毕 r 指针的最终位置。当然无法继承的指针需要进行重置。



**例子： **[**1385. 两个数组间的距离值**](https://leetcode-cn.com/problems/find-the-distance-value-between-two-arrays/)

此题要使用该方法主要的是确认移动红蓝边界的判断条件。因为题目要求的是对 arr1

  中任意一个元素 x 我们只需要在排序好的 arr2， 数组中寻找是否存在位于区间 [x-d, x+d] 的元素，如果存在，x 不符合距离要求。

判断条件，我们就设置`arr2[m] < arr1[i] - d`，找到最后一个小于`arr1[i] - d`的元素下标。接下来我们需要判断当前下标是否为`arr2`的最后一个元素，是则满足，不是就判断下一个元素是否满足大于`arr1[i] + d`，是则满足。

```javascript
/**
 * @param {number[]} arr1
 * @param {number[]} arr2
 * @param {number} d
 * @return {number}
 */
var findTheDistanceValue = function(arr1, arr2, d) {
    let res = 0,
        n = arr2.length;
    arr2.sort((a, b) => a - b);

    for (let i = 0;i < arr1.length;i++) {

        let l = -1,
            r = n;
        
        while (l < r - 1) {
            const m = (l + r) >> 1;
            if (arr2[m] < arr1[i] - d) {
                l = m;
            } else {
                r = m;
            }
        }

        if (l >= n - 1 || (l < n - 1 && arr2[l + 1] > arr1[i] + d)) {
            res++;
        }
    }
    return res;
};
```

### 二、[633. 平方数之和](https://leetcode-cn.com/problems/sum-of-square-numbers/)


这里还是用双指针的方法好点时间复杂度`O(n)`,用二分查找的话时间复杂度为`O(nlogn)`.



#### 解法一: 二分查找 
**思路:**

**1、通过二分查找法,找到c的平方跟并向上取整e,作为遍历的终止边界;**

**2、遍历**`**0 - e,**`**在各个循环中,判断**`**i - e**`**中是否有满足**`**a**<sup>**2**</sup>** + b**<sup>**2**</sup>** = c.**`

```javascript
/**
 * @param {number} c
 * @return {boolean}
 */
var judgeSquareSum = function(c) {
    if ([0, 1, 2].includes(c)) return true;

    let s = -1,
        e = c;
    while (s < e - 1) {
        const m = Math.floor((s + e) / 2);
        if (m * m < c) {
            s = m;
        } else {
            e = m;
        }
    }

    if (e * e === c) {
        return true;
    }

    let flag = false;
    for (let i = 0;i < e;i++) {
        let l = i - 1;
            r = e;
        
        while (l < r - 1) {
            const m = Math.floor((l + r) / 2);
            
            if (m * m < c - i * i) {
                l = m;
            } else {
                r = m;
            }
        }

        if (r < e && r * r + i * i === c) {
            flag = true;
        }
    }
    return flag;
};
```

#### 解法三: 数学
**思路:**

<font style="color:rgb(38, 38, 38);">费马平方和定理:</font>

<font style="color:rgb(38, 38, 38);"> </font><font style="color:rgb(140, 140, 140);">一个非负整数 </font>_<font style="color:rgb(140, 140, 140);">c</font>_<font style="color:rgb(140, 140, 140);"> 如果能够表示为两个整数的平方和，当且仅当 c</font>_<font style="color:rgb(140, 140, 140);">c</font>_<font style="color:rgb(140, 140, 140);"> 的所有形如 4k + 3的</font>**<font style="color:rgb(140, 140, 140);">质因子</font>**<font style="color:rgb(140, 140, 140);">的幂均为偶数。</font>

<font style="color:rgb(38, 38, 38);">因此我们需要对 c 进行</font>**<font style="color:rgb(38, 38, 38);">质因数分解</font>**<font style="color:rgb(38, 38, 38);">，再判断</font>**<font style="color:rgb(38, 38, 38);">所有</font>**<font style="color:rgb(38, 38, 38);">形如 4k + 3 的质因子的幂是否均为偶数即可。</font>

```javascript
var judgeSquareSum = function(c) {
    for (let base = 2; base * base <= c; base++) {
        // 如果不是因子，枚举下一个
        if (c % base !== 0) {
            continue;
        }

        // 计算 base 的幂
        let exp = 0;
        while (c % base == 0) {
            c /= base;
            exp++;
        }

        // 根据 Sum of two squares theorem 验证
        if (base % 4 === 3 && exp % 2 !== 0) {
            return false;
        }
    }

    // 例如 11 这样的用例，由于上面的 for 循环里 base * base <= c ，base == 11 的时候不会进入循环体
    // 因此在退出循环以后需要再做一次判断
    return c % 4 !== 3;
};
```

### 五、[33. 搜索旋转排序数组](https://leetcode-cn.com/problems/search-in-rotated-sorted-array/)
#### 解法一: 两次二分查找
**思路:**

1. 通过一次二分查找,获取最大值的位置;
2. 通过判断最大值的位置,得知旋转前后的数组与元素顺序是否变化;
3. 如果为旋转后未变化,则依旧为单调递增数组,再使用一次二分查找即可;
4. 如果变化,通过最大值的位置,可分为左右两个单调递增的数组;
5. 通过与数组第一个元素对比判断,target是在左侧或右侧;
6. 再进行二分查找即可;

```javascript
/**
 * @param {number[]} nums
 * @param {number} target
 * @return {number}
 */
var search = function(nums, target) {
    // 只有一个元素时,直接判断该元素是否与目标值相等;
    if (nums.length === 1) return nums[0] === target ? 0 : -1;

    // 找到数组中最大值的位置;
    const n = nums.length;
    let s = -1;
        e = n;
    
    while (s < e - 1) {
        const m = Math.floor((s + e) / 2);
        if (nums[m] >= nums[0]) {
            s = m
        } else {
            e = m;
        }
    }

    let len = n;

    let l = -1,
        r = len;

    
    // 判断最大值是否为最后一个元素,是的话该数组则旋转前后元素顺序相同;
    if (e !== n) {
        // 判断目标值是否大于第一个元素,是的话,目标值则在前半段的数组中;
        if (target >= nums[0]) {
            r = e;
            len = e;
        } else {
            l = s;
        }
    }

    while (l < r - 1) {
        const m = Math.floor((l + r ) / 2);
        if (nums[m] < target) {
            l = m;
        } else {
            r = m;
        }
    }

    if (r === len || r !== len && nums[r] !== target) {
        return -1;
    }
    return r;
};
```

/**
 * @param {number[]} nums
 * @param {number} target
 * @return {number}
 */
var search = function(nums, target) {
    const n = nums.length;

    let l = -1,
        r = n;

    while (l !== r - 1) {
        const m = Math.floor((l + r) / 2);
        if (nums[m] === target) {
            return m;
        }

        // 判断当前元素的位置
        if (nums[0] <= nums[m]) {
            // 当前元素位于左侧单调数组中;
            if (nums[0] <= target && target <= nums[m]) {
                r = m;
            } else {
                l = m;
            }
        } else {
            // 当前元素位于右侧单调数组中
            if (nums[m] < target && target <= nums[n - 1]) {
                l = m;
            } else {
                r = m
            }
        }
    }
    return -1;
};

### 六、[153. 寻找旋转排序数组中的最小值](https://leetcode-cn.com/problems/find-minimum-in-rotated-sorted-array/)


#### 解法一:二分查找
**思路:**

1. <font style="color:rgb(38, 38, 38);">通过二分查找找到数组中的最大值的下标</font>`<font style="color:rgb(38, 38, 38);">l</font>`<font style="color:rgb(38, 38, 38);">.</font>
2. <font style="color:rgb(38, 38, 38);">判断    </font>`<font style="color:rgb(38, 38, 38);">l === n - 1</font>`<font style="color:rgb(38, 38, 38);">,是的话,最小值则为</font>`<font style="color:rgb(38, 38, 38);">num[0]</font>`<font style="color:rgb(38, 38, 38);">,否则最小值为</font>`<font style="color:rgb(38, 38, 38);">nums[l + 1]</font>`<font style="color:rgb(38, 38, 38);">.</font>

```javascript
/**
 * @param {number[]} nums
 * @return {number}
 */
var findMin = function(nums) {
    const n = nums.length;

    let l = -1,
        r = n;
    
    while (l !== r - 1) {
        const m = Math.floor((l + r) / 2);

        if (nums[m] >= nums[0]) {
            l = m;
        } else {
            r = m;
        }
    }

    if (r === n) {
        return nums[0]
    } else {
        return nums[r];
    }
};
```



### 七、[167. 两数之和 II - 输入有序数组](https://leetcode-cn.com/problems/two-sum-ii-input-array-is-sorted/)

/**

 * @param {number} target
 * @return {number[]}

var twoSum = function(numbers, target) {
    const n = numbers.length;

    for (let i = 0;i < n;i++) {
        const cur = numbers[i];

        let l = i,
            r = n;

                return [i + 1, m + 1];
            } else if (numbers[m] < target - cur) {
                l = m
            } else {
                r = m;
            }
        }
    }

};
