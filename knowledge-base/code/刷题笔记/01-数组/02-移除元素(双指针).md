### 一、例题-[27. 移除元素](https://leetcode-cn.com/problems/remove-element/)
<font style="color:rgb(44, 62, 80);">题目给你一个数组 </font>`<font style="color:rgb(44, 62, 80);">nums</font>`<font style="color:rgb(44, 62, 80);"> 和一个值 </font>`<font style="color:rgb(44, 62, 80);">val</font>`<font style="color:rgb(44, 62, 80);">，你需要 原地 移除所有数值等于 </font>`<font style="color:rgb(44, 62, 80);">val</font>`<font style="color:rgb(44, 62, 80);"> 的元素，并返回移除后数组的新长度。</font>

<font style="color:#E8323C;">要求不要使用额外的数组空间，你必须仅使用 O(1) 额外空间并</font>**<font style="color:#E8323C;">原地</font>**<font style="color:#E8323C;">修改输入数组。</font>

<font style="color:#E8323C;"></font>

思路: <font style="color:rgb(44, 62, 80);">因为数组的元素在内存地址中是连续的，不能单独删除数组中的某个元素，只能覆盖。</font>

<font style="color:rgb(44, 62, 80);"></font>

```javascript
/**
 * @param {number[]} nums
 * @param {number} val
 * @return {number}
 */
var removeElement = function(nums, val) {
    let k = 0;
    for (let i = 0;i < nums.length;i++) {
        if (nums[i] !== val) {
            nums[k++] = nums[i];
        }
    }
    return k;
};
```

### [27. 移除元素](https://leetcode-cn.com/problems/remove-element/)
#### 解法一: 双指针
**思路:**

<font style="color:rgb(89, 89, 89);">当slow位置的元素为0,且fast位置元素不为0时,交换位置,这样所有为0的元素都移动到尾部.</font>

```javascript
/**
 * @param {number[]} nums
 * @return {void} Do not return anything, modify nums in-place instead.
 */
var moveZeroes = function(nums) {
    const n = nums.length;
    if (n === 1) return n;

    let slow = 0,
        fast = 1;
    
    while (fast < n) {
        if (nums[slow] === 0 && nums[fast] !== 0) {
            [nums[slow], nums[fast]] = [nums[fast], nums[slow]]
            slow++;
        } else if (nums[slow] !== 0) {
            slow++;
        }
        fast++;
    }
};
```



#### 解法二: sort排序
**思路:**

1.巧用`sort`这个API，来判断第二个参数`b`是否为0

2.若`b`不为0，则返回0，a和b的相对位置不变；若b为0，则返回-1，b在a的后面，因此最终所有的0都会在数组最后面

**sort方法知识点回顾**

如果指明了 `compareFunction` ，那么数组会按照调用该函数的返回值排序。即 a 和 b 是两个将要被比较的元素：

如果 `compareFunction(a, b) 小于 0` ，那么 `a` 会被排列到 `b` 之前；

如果 `compareFunction(a, b) 等于 0 `， `a` 和 `b `的相对位置不变；

如果 `compareFunction(a, b) 大于 0` ， `b `会被排列到 `a` 之前。

```javascript
/**
 * @param {number[]} nums
 * @return {void} Do not return anything, modify nums in-place instead.
 */
 var moveZeroes = function(nums) {
     nums.sort((a,b) => b? 0: -1)
};
```

### [844. 比较含退格的字符串](https://leetcode-cn.com/problems/backspace-string-compare/)
#### 解法一:栈
空间复杂度为O(n),不满足题目O(1)的要求.

```javascript
/**
 * @param {string} s
 * @param {string} t
 * @return {boolean}
 */
var backspaceCompare = function(s, t) {
    const stack1 = coculate(s);
    const stack2 = coculate(t);
    function coculate (str) {
        const res = [];
        for (let i = 0; i < str.length; i++) {
            if (str[i] !== '#') {
                res.push(str[i]);
            } else {
                res.pop();
            }
        }
        return res;
    }
    return stack1.join() === stack2.join(); 
};
```



#### 解法二:双指针
**思路:**

<font style="color:rgb(38, 38, 38);">由于 </font>`<font style="color:rgb(38, 38, 38);">#</font>`<font style="color:rgb(38, 38, 38);"> 号只会消除左边的一个字符，所以对右边的字符无影响，所以我们选择从后往前遍历 S，T 字符串.</font>

从后往前遍历 str，所遇情况有三，如下所示：

1. 若当前字符是 #，则 b 自增 1；
2. 若当前字符不是 #，且 b 不为 0，则 b 自减 1；
3. 若当前字符不是 #，且 b 为 0，则代表当前字符不会被消除，添加的final中。

最后对比两个处理后的字符串是否相同即可.

```javascript
/**
 * @param {string} s
 * @param {string} t
 * @return {boolean}
 */
var handler = function (str) {
    let final = '';
    let b = 0;

    for (let i = str.length - 1;i >= 0;i--) {
        if (str[i] === '#') {
            b++;
        } else if (b > 0) {
            b--;
        } else {
            final += str[i];
        }
    }
    return final;
}
var backspaceCompare = function(s, t) {
    return handler(s) === handler(t);
};
```



### [977. 有序数组的平方](https://leetcode-cn.com/problems/squares-of-a-sorted-array/)
#### 解法一: sort
**思路:**

直接遍历获取平方在通过sort排序.

```javascript
/**
 * @param {number[]} nums
 * @return {number[]}
 */
var sortedSquares = function(nums) {
    return nums.map(v => v * v).sort((a, b) => a - b);
};
```



#### 解法二:双指针
**思路:**

数组其实是有序的， 只不过负数平方之后可能成为最大数了。



那么数组平方的最大值就在数组的两端，不是最左边就是最右边，不可能是中间。



此时可以考虑双指针法了，`i`指向起始位置，`j`指向终止位置。



定义一个新数组`result`，和A数组一样的大小，让`k`指向`result`数组终止位置。



如果`A[i] * A[i] < A[j] * A[j]`, 那么`result[k--] = A[j] * A[j]`。



如果`A[i] * A[i] >= A[j] * A[j]`, 那么`result[k--] = A[i] * A[i]`。

```javascript
/**
 * @param {number[]} nums
 * @return {number[]}
 */
var sortedSquares = function(nums) {
    const res = [];
    let s = 0,
        e = nums.length -1;
    while (s <= e) {
        if (Math.abs(nums[s]) > Math.abs(nums[e])) {
            res.push(nums[s] ** 2);
            s++;
        } else {
            res.push(nums[e] ** 2);
            e--;
        }
    }

    return res.reverse();
};
```
