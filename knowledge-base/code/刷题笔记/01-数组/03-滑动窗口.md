### 一、滑动窗口
#### 例题:
<font style="color:rgb(44, 62, 80);">给定一个含有 n 个正整数的数组和一个正整数 s ，找出该数组中满足其和 ≥ s 的长度最小的 连续 子数组，并返回其长度。如果不存在符合条件的子数组，返回 0。(</font>[209. 长度最小的子数组](https://leetcode-cn.com/problems/minimum-size-subarray-sum/)<font style="color:rgb(44, 62, 80);">)</font>

<font style="color:rgb(44, 62, 80);">示例：</font>

<font style="color:rgb(44, 62, 80);">输入：s = 7, nums = [2,3,1,2,4,3] 输出：2 解释：子数组 [4,3] 是该条件下的长度最小的子数组。</font>

<font style="color:rgb(44, 62, 80);">所谓滑动窗口，</font>**<font style="color:rgb(44, 62, 80);">就是不断的调节子序列的起始位置和终止位置，从而得出我们要想的结果</font>**<font style="color:rgb(44, 62, 80);">。</font>

<font style="color:rgb(44, 62, 80);">这里还是以上例题来举例，s=7， 数组是 2，3，1，2，4，3，来看一下查找的过程：</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1652007711263-3de0b9f5-a7a3-46a1-baca-a2a299351c09.png)

<font style="color:rgb(44, 62, 80);">最后找到 4，3 是最短距离。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">其实从动画中可以发现滑动窗口也可以理解为双指针法的一种！只不过这种解法更像是一个窗口的移动，所以叫做滑动窗口更适合一些。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">在本题中实现滑动窗口，主要确定如下三点：</font>

+ <font style="color:rgb(44, 62, 80);">窗口内是什么？</font>
+ <font style="color:rgb(44, 62, 80);">如何移动窗口的起始位置？</font>
+ <font style="color:rgb(44, 62, 80);">如何移动窗口的结束位置？</font>

<font style="color:rgb(44, 62, 80);">窗口就是 满足其和 ≥ s 的长度最小的 连续 子数组。</font>

<font style="color:rgb(44, 62, 80);">窗口的起始位置如何移动：如果当前窗口的值大于s了，窗口就要向前移动了（也就是该缩小了）。</font>

<font style="color:rgb(44, 62, 80);">窗口的结束位置如何移动：窗口的结束位置就是遍历数组的指针，窗口的起始位置设置为数组的起始位置就可以了。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">解题的关键在于 窗口的起始位置如何移动，如图所示：</font>

<font style="color:rgb(44, 62, 80);"></font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1652007886786-cc846c67-8556-4b49-80bd-c022d40a925a.png)

<font style="color:rgb(44, 62, 80);">可以发现</font>**<font style="color:rgb(44, 62, 80);">滑动窗口的精妙之处在于根据当前子序列和大小的情况，不断调节子序列的起始位置。从而将O(n^2)暴力解法降为O(n)。</font>**



**<font style="color:rgb(44, 62, 80);">为什么时间复杂度是O(n)</font>**<font style="color:rgb(44, 62, 80);">。</font>

<font style="color:rgb(44, 62, 80);">不要以为for里放一个while就以为是</font>`<font style="color:rgb(44, 62, 80);">O(n^2)</font>`<font style="color:rgb(44, 62, 80);">啊， 主要是看每一个元素被操作的次数，每个元素在滑动窗后进来操作一次，出去操作一次，每个元素都是被被操作两次，所以时间复杂度是 </font>`<font style="color:rgb(44, 62, 80);">2 × n</font>`<font style="color:rgb(44, 62, 80);">也就是</font>`<font style="color:rgb(44, 62, 80);">O(n)</font>`<font style="color:rgb(44, 62, 80);">。</font>





```javascript
/**
 * @param {number} target
 * @param {number[]} nums
 * @return {number}
 */
var minSubArrayLen = function(target, nums) {
    const n = nums.length;
    let l = r = sum = 0,
        res = n + 1;
    
    while (r < n) {
        sum += nums[r++];

        while (sum >= target) {
            res = Math.min(res, r - l);
            sum -= nums[l++];
        }
    }

    return res > n ? 0 : res;
};
```



### [904. 水果成篮](https://leetcode-cn.com/problems/fruit-into-baskets/)
#### 解法一:滑动窗口
**思路:**

<font style="color:rgb(38, 38, 38);">如果当前窗口的水果种类>=2, 且正在采摘的是新类型,则需要将之前的水果清空一种类型.</font>

```javascript
/**
 * @param {number[]} fruits
 * @return {number}
 */
var totalFruit = function(fruits) {
    const n = fruits.length;
    let l = r = 0,
        res = 0;
    let sum = new Map;

    while (r < n) {
        if (sum.size >= 2 && !sum.has(fruits[r])) {
            while (sum.get(fruits[l]) > 1) {
                sum.set(fruits[l], sum.get(fruits[l]) - 1);
                l++;
            }
            sum.delete(fruits[l]);
            l++;
        }
        sum.set(fruits[r], (sum.get(fruits[r]) || 0) + 1);
        r++;
        res = Math.max(r - l, res);
    }
    return res;
};
```



### [76. 最小覆盖子串](https://leetcode-cn.com/problems/minimum-window-substring/)(困难)
#### 解法一: 滑动窗口
**思路:**

<font style="color:rgb(38, 38, 38);">通过变量</font>`<font style="color:rgb(38, 38, 38);">valid</font>`<font style="color:rgb(38, 38, 38);">判断滑动窗口中的字符串包含</font>`<font style="color:rgb(38, 38, 38);">t</font>`<font style="color:rgb(38, 38, 38);">中所有的字符.</font>

```javascript
/**
 * @param {string} s
 * @param {string} t
 * @return {string}
 */
var minWindow = function(s, t) {
    const n = s.length;

    let l = r = valid = 0,
        res = s + 'a';
    
    const mapS = new Map;
    const mapT = new Map;
    
    for (let i = 0;i < t.length;i++) {
        mapT.set(t[i], (mapT.get(t[i]) || 0) + 1);
    }
    
    while (r < n) {
        if (mapT.has(s[r])) {
            mapS.set(s[r], (mapS.get(s[r]) || 0) + 1);
            if (mapS.get(s[r]) === mapT.get(s[r])) {
                valid++;
            }
        }
        r++;
        
        while (valid === mapT.size) {
            if (res.length > r - l) {
                res = s.slice(l, r);
            }
            if (mapT.has(s[l])) {
                if (mapS.get(s[l]) === mapT.get(s[l])) {
                    valid--;
                }
                mapS.set(s[l], mapS.get(s[l]) - 1);
            }
            l++;
        }
    }
    return res.length > s.length ? '' : res;
};
```
