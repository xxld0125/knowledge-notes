### **一、什么是辗转相除法：**
<font style="color:rgb(89, 89, 89);"></font>

GCD算法(Greatest Common Divisor) 即欧几里得算法,也叫辗转相除法。是以除数和余数反复做除法运算，当余数为 0 时，取当前算式除数为最大公约数的计算公式。如下：

1997 / 615 = 3 (余 152)

615 / 152 = 4(余7)

152 / 7 = 21(余5)

7 / 5 = 1 (余2)

5 / 2 = 2 (余1)

2 / 1 = 2 (余0)

至此，最大公约数为1。



**一般用于处理最大公因子类型问题**

### 一、[914. 卡牌分组](https://leetcode-cn.com/problems/x-of-a-kind-in-a-deck-of-cards/)
#### 思路：首先通过哈希表获取各字母出现的次数，再取各次数的最大公约数，若大于2则满足分组条件
```javascript
var hasGroupsSizeX = function(deck) {
    let map = new Map()
    for(let n of deck){//统计频次
        map.set(n,map.has(n)?map.get(n)+1:1)
    }
    let arr = [...map.values()]
    let res = arr[0]
    return arr.every(i => (res = gcd(res, i)) > 1)//求最大公约数是否大于1

};
```

### 二、1071. 字符串的最大公因子
思路：

如果str1，str2有公因子 abc，那么 str1 就是 m 个 abc 的重复，str2 是 n 个 abc 的重复，<font style="color:rgb(38, 38, 38);">如果能循环以它的约数为长度的字符串，自然也能够循环以它为长度的字符串，所以这个理论长度就是我们要找的最优解。</font>

```javascript
/**
 * @param {string} str1
 * @param {string} str2
 * @return {string}
 */
var gcdOfStrings = function(str1, str2) {
    if (str1 + str2 !== str2 + str1) {
        return '';
    }
    const gcd = (a, b) => (b === 0 ? a : gcd(b, a % b));
    return str1.substring(0, gcd(str1.length, str2.length))
};
```
