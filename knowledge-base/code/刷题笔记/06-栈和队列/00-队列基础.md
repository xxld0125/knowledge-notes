##### 一、介绍
	队列与栈非常类似,区别在于其使用了与后进先出不用的原则, 即先进先出原则(FIFO), 队列在尾部添加新元素, 并从顶部移除元素.



	最常见的队列的例子就是排队.



##### 二、队列数据结构
声明Queue类:



```javascript
class Queue {
	constructor() {
		this.count = 0;
		this.lowestCount = 0; // 顶部元素下标
		this.items = {};
	}
}
```







		Queue类和Stack类非常相似,只是在添加和移除元素的原则不同.



+  向队列添加元素  
添加元素到队列末尾  

```javascript
enqueue(element) {
	this.items[this.count] = element;
  this.count++;
}
```

+  从队列移除元素  
从队列顶部移除元素  

```javascript
dequeue() {
  if (this.isEmpty()) {
		return undefined;
  }
  const result = this.items[this.lowestCount];
  delete this.items[this.lowestCount];
  this.lowestCount++;
  return result;
}
```

+  查看队列头元素  

```javascript
peek() {
	if (this.isEmpty()) {
		return undefined;
	}
	return this.items[this.lowestCount];
}
```

+  检查队列是否为空并获取它的长度  

```javascript
isEmpty() {
  return this.count - this.lowestCount === 0;
}

size() {
  return this.count - this.lowestCount;
}
```

+  清空队列  
可以调用dequeue方法直到返回undefined, 也可以简单地将队列中的属性值重设为和构造函数中的一样.  

```javascript
clear() {
	this.items = {};
	this.count = 0;
	this.lowestCount = 0;
}
```

+  toString方法  

```javascript
toString() {
	if (this.isEmpty()) {
		return '';
	}
	let objString = `${this.items[this.lowestCount]}`;
	for (let i = this.lowestCount + 1;i < this.count;i++) {
		objString = `${objString}, ${this.items[i]}`;
	}
	return objString;
}
```



以上我们完成了Queue类的实现, Queue类和Stack类非常类似, 主要区别还是在于dequeue方法和peek方法, 这是由于先进先出和后进先出原则不同导致的.



接下来我们开始使用Queue类.



```javascript
const queue = new Queue();

console.log(queue.isEmpty())); // true;

queue.enqueue('John');

queue.enqueue('Jack');

console.log(queue.toString()); // 'John,Jack'

queue.enqueue('Camila');

console.log(queue.toString()); // 'John,Jack,Camila'

console.log(queue.size()); // 3

console.log(queue.isEmpty()); // false

queue.dequeue(); // 移除John

queue.dequeue(); // 移除Jack

console.log(queue.toString()); // Camila
```

##### 三、双端队列数据结构


		涮段队列(deque, 或称double-ended queue)是一种允许我们同时从前端和后端添加和移除元素的特殊队列.



		在计算机科学中, 双端队列的一个常见应用是存储一系列的撤销操作. 每当用户在软件中进行一个操作, 该操作会被存进一个双端队列. 每当用户点击撤销按钮时, 该操作会被从双端队列中弹出, 表示它被从后面移除了. 在进行了预先定义的一定数量的操作后, 最先进行的操作会被从双端队列的前端移除. 由于双端队列同时遵守了先进先出和后进先出原则, 可以说他是把队列和栈相结合的一种数据结构.



接下来我们创建一个Deque类:



先声明体一个Deque类及其构造函数.



```javascript
class Deque {
  constructor() {
    this.count = 0;
    this.lowestCount = 0;
    this.items = {};
  }
}
```



双端队列是一种特殊队列, 我们可以从上面看出其构造函数内的部分代码和队列相同, 包括相同的内部属性和以下方法: isEmpty()、clear、size、toString.



既然双端队列允许从两端添加和移除元素, 还会有以下几个方法:



+ addFront:	在前端添加新元素.
+ adback:	在后端添加新元素.
+ removeFront:	在前端移除元素.
+ removeBack:	在后端移除元素.
+ peekFront:	返回前端第一个元素.
+ peekBack:	返回后端最后一个元素.



下面我们实现addFront方法:



```javascript
addFront(element) {
	if (this.isEmpty) {
		this.addBack(element);
	} else if (this.lowestCount > 0) {
		this.lowestCount--;
	} else {
		for (let i = this.count; i > 0; i--) {
			this.items[i] = this.items[i - 1];
		}
		this.count++;
		this.lowestCount = 0;
		this.items[0] = element;
	}
}
```



使用Deque类:



```javascript
const deque = new Deque();

console.log(deque.isEmpty());

deque.addBack('John');

deque.addBack('Jack');

console.log(deque.toString()); // 'John,Jack'

deque.addBack('Camila');

console.log(deque.toString()); // 'John,Jack,Camila'

console.log(deque.size()); // 3

deque.removeFront(); // 移除John

console.log(deque.toString()); // 'Jack,Camila'

deque.removeBack(); // 移除Camila

console.log(deque.toString()); // 'Jack'

deque.addFront('John');

console.log(deque.toString()); // 'John,Jack'
```

+  循环队列——击鼓传花问题  
击鼓传花游戏是循环队列的一个例子, 在这个游戏中, 孩子们围成一个圆圈, 把花尽快地传递给旁边的人. 某一时刻传花停止, 这个时候花在谁手里, 谁就退出圆圈, 重复这个过程, 直到只剩一个孩子.  
在下面这个示例例中, 我们实现一个模拟的击鼓传花游戏.   
尝试使用hotPotato算法   
以上算法的输出如下: 

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1646401361000-20d9e63a-5cee-40b6-bdea-e170cfa6cb8e.png)  
下图模拟了这个输出过程 ![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1646401380385-b74794d0-a192-4428-8c1f-0c179d9497b7.png)



```javascript
function hotPotato(elementList, num) {
	const queue = new Queue();
  const elimitatedList = [];
  
  // 将所有元素加入队列
  for (let i = 0; i < elementList.length; i++) {
    queue.enqueue(elementList[i]);
  }
  
  // 只剩下一个人时, 这个人就是胜者
  while (queue.size() > 1) {
    for (let i = 0; i < num; i++) {
      // 模拟击鼓传花
      // 即从队列的开头移除一项, 在将其添加到队列末尾
      queue.enqueue(queue.dequeue());
    }
    // 一旦达到给定的传递次数, 拿着花的那个人就被淘汰了
    // 即从队列中移除
    elimitatedList.push(queue.dequeue());
  }
  
  return {
    eliminated: elimitatedList,
    winner: queue.dequeue();
  }
}
```

```javascript
const names = ['John','Jack','Camila','Ingrid','Carl'];
const result = hotPotato(names, 7);

result.elimenated.forEach(name => {
  console.log(`${name}在击鼓传花游戏中被淘汰`);
})

console.log(`胜利者: ${result.winner}`);
```

+  回文检查器  
回文是正反都能读通的单词、词组、数或一系列的序列.  
有很多算法可以检查一个词组或字符串是否为回文. 最简单的方法就是将字符串反向排列并检查它和原字符串是否相同. 也可以用栈来完成, 但利用数据结构来解决这个问题最简单的就是使用双端队列.  
具体方法如下:  

```javascript
// 判断传入的字符串是否为回文字符串, 不符合则返回false
function palindromeChecker(aString) {
	if (aString === undefined || aString === null || (aString !== null && aString.length === 0)) {
		return false;
	}
	const deque = new Deque();
  
  // 将所有字母转化为小写,同时移除所有空格
  const lowerString = aString.toLocaleLowerCase().split(' ').join('');
  let isEqual = true;
  let firstChar, lastChar;
  
  // 将字符串存入双端队列中
  for (let i = 0; i < lowerString.length; i++) {
    deque.addBack(lowerString.charAt(i));
  }
  
 	// 如果只有一个字符, 肯定是回文
  while (deque.size() > 1 && isEqual) {
    // 从前端、后端各移除一个元素, 且需相同,否则该字符串不是回文
    firstChar = deque.removeFront();
    lastChar = deque.removeBack();
    if (firstChar !== lastChar) {
			isEqual = false;
    }
  }
  
  return isEqual;
}
```
