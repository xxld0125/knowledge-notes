### 例题:[面试题 02.07. 链表相交](https://leetcode.cn/problems/intersection-of-two-linked-lists-lcci/)
<font style="color:rgb(44, 62, 80);">简单来说，就是求两个链表交点节点的</font>**<font style="color:rgb(44, 62, 80);">指针</font>**<font style="color:rgb(44, 62, 80);">。 这里同学们要注意，交点不是数值相等，而是指针相等。</font>

<font style="color:rgb(44, 62, 80);">看如下两个链表，目前</font>`<font style="color:rgb(44, 62, 80);">curA</font>`<font style="color:rgb(44, 62, 80);">指向链表A的头结点，</font>`<font style="color:rgb(44, 62, 80);">curB</font>`<font style="color:rgb(44, 62, 80);">指向链表</font>`<font style="color:rgb(44, 62, 80);">B</font>`<font style="color:rgb(44, 62, 80);">的头结点：</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1652181165305-97248ff6-3668-408a-9c19-c96a9358f39e.png)

<font style="color:rgb(44, 62, 80);">我们求出两个链表的长度，并求出两个链表长度的差值，然后让</font>`<font style="color:rgb(44, 62, 80);">curA</font>`<font style="color:rgb(44, 62, 80);">移动到，和</font>`<font style="color:rgb(44, 62, 80);">curB</font>`<font style="color:rgb(44, 62, 80);"> 末尾对齐的位置，如图：</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1652181210424-a1e8dff7-3633-4d91-b874-8c01385718f5.png)

<font style="color:rgb(44, 62, 80);">此时我们就可以比较</font>`<font style="color:rgb(44, 62, 80);">curA</font>`<font style="color:rgb(44, 62, 80);">和cur</font>``<font style="color:rgb(44, 62, 80);">B是否相同，如果不相同，同时向后移动</font>`<font style="color:rgb(44, 62, 80);">curA</font>`<font style="color:rgb(44, 62, 80);">和</font>`<font style="color:rgb(44, 62, 80);">curB</font>`<font style="color:rgb(44, 62, 80);">，如果遇到</font>`<font style="color:rgb(44, 62, 80);">curA</font>`<font style="color:rgb(44, 62, 80);"> == </font>`<font style="color:rgb(44, 62, 80);">curB</font>`<font style="color:rgb(44, 62, 80);">，则找到交点。</font>

<font style="color:rgb(44, 62, 80);">否则循环退出返回空指针。</font>

<font style="color:rgb(44, 62, 80);"></font>

```javascript
/**
 * Definition for singly-linked list.
 * function ListNode(val) {
 *     this.val = val;
 *     this.next = null;
 * }
 */

/**
 * @param {ListNode} headA
 * @param {ListNode} headB
 * @return {ListNode}
 */
var getListLen = function(head) {
    let len = 0, cur = head;
    while(cur) {
       len++;
       cur = cur.next;
    }
    return len;
} 
var getIntersectionNode = function(headA, headB) {
    let tmpA = headA,
        tmpB = headB,
        lenA = getListLen(headA),
        lenB = getListLen(headB);

    let a = headA,
        b = headB,
        l = Math.abs(lenA - lenB);
    
    if (lenA > lenB) {
        while (l--) {
            a = a.next;
        }
    } else {
        while (l--) {
            b = b.next;
        }
    }

    while (a && a !== b) {
        a = a.next;
        b = b.next;
    }
    return a;
};
```
