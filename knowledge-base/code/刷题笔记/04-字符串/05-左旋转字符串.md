### 一、例题-[剑指 Offer 58 - II. 左旋转字符串](https://leetcode.cn/problems/zuo-xuan-zhuan-zi-fu-chuan-lcof/)


<font style="color:rgb(44, 62, 80);">字符串的左旋转操作是把字符串前面的若干个字符转移到字符串的尾部。请定义一个函数实现字符串左旋转操作的功能。比如，输入字符串"abcdefg"和数字2，该函数将返回左旋转两位得到的结果"cdefgab"。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">示例 1：  
</font><font style="color:rgb(44, 62, 80);">输入: s = "abcdefg", k = 2  
</font><font style="color:rgb(44, 62, 80);">输出: "cdefgab"</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">示例 2：  
</font><font style="color:rgb(44, 62, 80);">输入: s = "lrloseumgh", k = 6  
</font><font style="color:rgb(44, 62, 80);">输出: "umghlrlose"</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">限制：  
</font><font style="color:rgb(44, 62, 80);">1 <= k < s.length <= 10000</font>

<font style="color:rgb(44, 62, 80);"></font>

#### 思路
<font style="color:rgb(44, 62, 80);">为了让本题更有意义，提升一下本题难度：</font>**<font style="color:rgb(44, 62, 80);">不能申请额外空间，只能在本串上操作</font>**<font style="color:rgb(44, 62, 80);">。</font>



<font style="color:rgb(44, 62, 80);">不能使用额外空间的话，模拟在本串操作要实现左旋转字符串的功能还是有点困难的。</font>



<font style="color:rgb(44, 62, 80);">那么我们可以想一下上一题目</font>[04-翻转字符串里的单词](https://www.yuque.com/u25370234/kb/vuqs3y)<font style="color:rgb(44, 62, 80);">中讲过，使用整体反转+局部反转就可以实现，反转单词顺序的目的。</font>



<font style="color:rgb(44, 62, 80);">这道题目也非常类似，依然可以通过局部反转+整体反转 达到左旋转的目的。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">具体步骤为：</font>

1. <font style="color:rgb(44, 62, 80);">反转区间为前n的子串</font>
2. <font style="color:rgb(44, 62, 80);">反转区间为n到末尾的子串</font>
3. <font style="color:rgb(44, 62, 80);">反转整个字符串</font>



<font style="color:rgb(44, 62, 80);">最后就可以得到左旋n的目的，而不用定义新的字符串，完全在本串上操作</font><font style="color:rgb(44, 62, 80);">。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">例如 ：示例1中 输入：字符串abcdefg，n=2</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1652604881744-bb121854-604e-400f-b3db-76b16b1242f6.png)

<font style="color:rgb(44, 62, 80);">最终得到左旋2个单元的字符串：cdefgab</font>

<font style="color:rgb(44, 62, 80);"></font>

```javascript
/**
 * @param {string} s
 * @param {number} n
 * @return {string}
 */
var reverseLeftWords = function(s, n) {
    const len = s.length;
    
    const arr = Array.from(s);

    reverse(arr, 0, n - 1);

    reverse(arr, n, len - 1);

    reverse(arr, 0, len - 1);

    return arr.join('');
};

function reverse(arr, l, r) {
    while (l < r) {
        [arr[l], arr[r]] = [arr[r], arr[l]];
        l++;
        r--;
    }
}
```
