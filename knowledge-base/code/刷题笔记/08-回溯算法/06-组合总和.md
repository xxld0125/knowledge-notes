### 一、[39. 组合总和](https://leetcode.cn/problems/combination-sum/)
<font style="color:rgb(44, 62, 80);">给定一个无重复元素的数组 candidates 和一个目标数 target ，找出 candidates 中所有可以使数字和为 target 的组合。</font>

<font style="color:rgb(44, 62, 80);">candidates 中的数字可以无限制重复被选取。</font>

<font style="color:rgb(44, 62, 80);">说明：</font>

+ <font style="color:rgb(44, 62, 80);">所有数字（包括 target）都是正整数。</font>
+ <font style="color:rgb(44, 62, 80);">解集不能包含重复的组合。</font>

<font style="color:rgb(44, 62, 80);">示例 1： 输入：candidates = [2,3,6,7], target = 7, 所求解集为： [ [7], [2,2,3] ]</font>

<font style="color:rgb(44, 62, 80);">示例 2： 输入：candidates = [2,3,5], target = 8, 所求解集为： [   [2,2,2,2],   [2,3,3],   [3,5] ]</font>

#### 回溯三部曲
##### 递归函数参数
<font style="color:rgb(44, 62, 80);">这里依然是定义两个全局变量，二维数组result存放结果集，数组path存放符合条件的结果。</font>

**<font style="color:rgb(44, 62, 80);">startIndex来控制for循环的起始位置，对于组合问题，什么时候需要startIndex呢？</font>**



<font style="color:rgb(44, 62, 80);">我举过例子，如果是一个集合来求组合的话，就需要startIndex，例如：</font>[02-组合问题](https://www.yuque.com/u25370234/kb/lggt90)<font style="color:rgb(44, 62, 80);">，</font>[03-组合总结III](https://www.yuque.com/u25370234/kb/ne3adm)<font style="color:rgb(44, 62, 80);">。</font>

<font style="color:rgb(44, 62, 80);">如果是多个集合取组合，各个集合之间相互不影响，那么就不用startIndex，例如：</font>[04-电话号码的字母组合](https://www.yuque.com/u25370234/kb/dim49s)

##### 单层搜索的逻辑
<font style="color:rgb(44, 62, 80);">单层for循环依然是从startIndex开始，搜索candidates集合。</font>

**<font style="color:rgb(44, 62, 80);">注意本题和</font>**[02-组合问题](https://www.yuque.com/u25370234/kb/lggt90)<font style="color:rgb(44, 62, 80);">，</font>[03-组合总结III](https://www.yuque.com/u25370234/kb/ne3adm)**<font style="color:rgb(44, 62, 80);">的一个区别是：本题元素为可重复选取的</font>**<font style="color:rgb(44, 62, 80);">。</font>

<font style="color:rgb(44, 62, 80);">如何重复选取呢，看代码，注释部分：</font>

```javascript
for (int i = startIndex; i < candidates.length; i++) {
    sum += candidates[i];
    path.push(candidates[i]);
    backtracking(i, sum); // 关键点:不用i+1了，表示可以重复读取当前的数
    path.pop();        // 回溯
    sum -= candidates[i]; // 回溯
}
```



##### 完整代码
```javascript
var combinationSum = function(candidates, target) {
    const res = [], path = [];

    // 排序
    candidates.sort((a,b) => a - b);

    backtracking(0, 0);

    return res;

    function backtracking(startIndex, sum) {
        if (sum > target) {
            return;
        }
        if (sum === target) {
            res.push([...path]);
            return;
        }

        for (let i = startIndex;i < candidates.length;i++) {
            const n = candidates[i];

            path.push(n);
            sum += n;

            backtracking(i, sum);

            path.pop(); // 回溯
            sum -= n; // 回溯
        }
    }
}
```





#### 剪枝优化
<font style="color:rgb(44, 62, 80);">在这个树形结构中：</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1660122304008-f788f040-094b-487a-865d-74bd6125b479.png)

<font style="color:rgb(44, 62, 80);">以及上面的版本一的代码大家可以看到，对于sum已经大于target的情况，其实是依然进入了下一层递归，只是下一层递归结束判断的时候，会判断sum > target的话就返回。</font>

<font style="color:rgb(44, 62, 80);">其实如果已经知道下一层的sum会大于target，就没有必要进入下一层递归了。</font>

<font style="color:rgb(44, 62, 80);">那么可以在for循环的搜索范围上做做文章了。</font>

<font style="color:rgb(44, 62, 80);"></font>

**<font style="color:rgb(44, 62, 80);">对总集合排序之后，如果下一层的sum（就是本层的 sum + candidates[i]）已经大于target，就可以结束本轮for循环的遍历</font>**<font style="color:rgb(44, 62, 80);">。</font>

<font style="color:rgb(44, 62, 80);">如图：</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1660122353932-1943a1b2-191d-4f37-b713-600eeefcb76c.png)



##### 完整代码
```javascript
var combinationSum = function(candidates, target) {
    const res = [], path = [];

    // 排序
    candidates.sort((a,b) => a - b);


    backtracking(0, 0);

    return res;

    function backtracking(startIndex, sum) {
        if (sum === target) {
            res.push([...path]);
        }

        for (let i = startIndex;i < candidates.length;i++) {
            const n = candidates[i];

            // 剪枝优化
            if (n > target - sum) break;

            path.push(n);
            sum += n;

            backtracking(i, sum);

            path.pop(); // 回溯
            sum -= n; // 回溯
        }
    }
};
```
