### 一、例题-[222. 完全二叉树的节点个数](https://leetcode.cn/problems/count-complete-tree-nodes/)
<font style="color:rgb(44, 62, 80);">给出一个完全二叉树，求出该树的节点个数。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">示例 1：</font>

+ <font style="color:rgb(44, 62, 80);">输入：root = [1,2,3,4,5,6]</font>
+ <font style="color:rgb(44, 62, 80);">输出：6</font>

<font style="color:rgb(44, 62, 80);">示例 2：</font>

+ <font style="color:rgb(44, 62, 80);">输入：root = []</font>
+ <font style="color:rgb(44, 62, 80);">输出：0</font>

<font style="color:rgb(44, 62, 80);">示例 3：</font>

+ <font style="color:rgb(44, 62, 80);">输入：root = [1]</font>
+ <font style="color:rgb(44, 62, 80);">输出：1</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">提示：</font>

+ <font style="color:rgb(44, 62, 80);">树中节点的数目范围是[0, 5 * 10^4]</font>
+ <font style="color:rgb(44, 62, 80);">0 <= Node.val <= 5 * 10^4</font>
+ <font style="color:rgb(44, 62, 80);">题目数据保证输入的树是 完全二叉树</font>

<font style="color:rgb(44, 62, 80);"></font>

#### <font style="color:rgb(44, 62, 80);">思路</font>
什么是完全二叉树,之前有介绍: [01-二叉树理论基础](https://www.yuque.com/u25370234/kb/mcitgl)<font style="color:rgb(44, 62, 80);">。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">完全二叉树只有两种情况</font>

<font style="color:rgb(44, 62, 80);">情况一：就是满二叉树</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1653315406280-d42e6b0c-2765-48ca-b11c-4a175d10c16d.png)

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">情况二：最后一层叶子节点没有满。</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1653315415569-3be84f46-7523-4f06-9299-b3f970b82e22.png)

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">对于情况一，可以直接用 2^树深度 - 1 来计算，注意这里根节点深度为1。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">对于情况二，分别递归左孩子，和右孩子，递归到某一深度一定会有左孩子或者右孩子为满二叉树，然后依然可以按照情况1来计算。</font>

<font style="color:rgb(44, 62, 80);"></font>

```javascript
var countNodes = function(root) {
  // 利用完全二叉树的特点
  if (root === null) return 0;
  let left = root.left;
  let right = root.right;
  
  let leftHeight = 0,
       rightHeight = 0;
  
  while (left) {
    left = left.left;
    leftHeight++;  
  }
  
  while (right) {
    right = right.right;
    rightHeight++;
  }
  
  if (leftHeight === rightHeight) {
    return Math.pow(2, leftHeight + 1) - 1;
  }
  
  return countNodes(root.left) + countNodes(root.right) + 1;
}
```
