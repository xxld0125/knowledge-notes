### 一、例题-[102. 二叉树的层序遍历](https://leetcode.cn/problems/binary-tree-level-order-traversal/)
<font style="color:rgb(44, 62, 80);">给你一个二叉树，请你返回其按 层序遍历 得到的节点值。 （即逐层地，从左到右访问所有节点）。</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1652804300981-a6a53402-a092-4c81-9e53-8f8b00bf64d9.png)



#### 思路
<font style="color:rgb(44, 62, 80);">层序遍历一个二叉树。就是从左到右一层一层的去遍历二叉树。这种遍历的方式和我们之前讲过的都不太一样</font>**<font style="color:rgb(44, 62, 80);">。</font>**

**<font style="color:rgb(44, 62, 80);"></font>**

<font style="color:rgb(44, 62, 80);">需要借用一个辅助数据结构即队列来实现，</font>**<font style="color:rgb(44, 62, 80);">队列先进先出，符合一层一层遍历的逻辑，而是用栈先进后出适合模拟深度优先遍历也就是递归的逻辑。</font>**

**<font style="color:rgb(44, 62, 80);"></font>**

**<font style="color:rgb(44, 62, 80);">而这种层序遍历方式就是图论中的广度优先遍历，只不过我们应用在二叉树上。</font>**

![](https://cdn.nlark.com/yuque/0/2022/gif/25743026/1652804433081-399f4380-d64b-4ba2-bbf8-7b56a24d74a2.gif)  
<font style="color:rgb(44, 62, 80);">	这样就实现了层序从左到右遍历二叉树。</font>

<font style="color:rgb(44, 62, 80);">代码如下：</font>**<font style="color:rgb(44, 62, 80);">这份代码也可以作为二叉树层序遍历的模板，打十个就靠它了</font>**<font style="color:rgb(44, 62, 80);">。</font>

<font style="color:rgb(44, 62, 80);"></font>

```javascript
var levelOrder = function(root) {
  // 二叉树的层序遍历
  let res = [], queue = [];
  queue.push(root);
  if (root === null) {
    return res;
  }
  while (queue.length) {
    // 记录当前层级节点数
    let len = queue.length;
    // 存放每一层的节点
    let curLever = [];
    for (let i = 0;i < len;i++) {
      let node = queue.shift();
      curLevel.push(node.val);
      // 存放当前层下一层的节点
      node.left && queue.push(node.left);
      node.right && queue.push(node.right);
    }
    // 把每一层的结果放到结果数组
    res.push(curLevel);
  }
  return res;
}
```
