### 一、[106. 从中序与后序遍历序列构造二叉树](https://leetcode.cn/problems/construct-binary-tree-from-inorder-and-postorder-traversal/)


<font style="color:rgb(44, 62, 80);">根据一棵树的中序遍历与后序遍历构造二叉树。</font>

<font style="color:rgb(44, 62, 80);">注意: 你可以假设树中没有重复的元素。</font>

<font style="color:rgb(44, 62, 80);">例如，给出</font>

<font style="color:rgb(44, 62, 80);">中序遍历 inorder = [9,3,15,20,7] 后序遍历 postorder = [9,15,7,20,3] 返回如下的二叉树：</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1654349738647-70f7728a-2417-4c73-a2f7-e55070f8ee4a.png)



#### 思路
<font style="color:rgb(44, 62, 80);">首先回忆一下如何根据两个顺序构造一个唯一的二叉树，相信理论知识大家应该都清楚，就是以 后序数组的最后一个元素为切割点，先切中序数组，根据中序数组，反过来在切后序数组。一层一层切下去，每次后序数组最后一个元素就是节点元素。</font>

<font style="color:rgb(44, 62, 80);">如果让我们肉眼看两个序列，画一棵二叉树的话，应该分分钟都可以画出来。</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1654350032673-f3370792-2fb4-4bc1-b586-f07d7a109870.png)

<font style="color:rgb(44, 62, 80);">那么代码应该怎么写呢？</font>

<font style="color:rgb(44, 62, 80);">说到一层一层切割，就应该想到了递归。</font>

<font style="color:rgb(44, 62, 80);">来看一下一共分几步：</font>

+ <font style="color:rgb(44, 62, 80);">第一步：如果数组大小为零的话，说明是空节点了。</font>
+ <font style="color:rgb(44, 62, 80);">第二步：如果不为空，那么取后序数组最后一个元素作为节点元素。</font>
+ <font style="color:rgb(44, 62, 80);">第三步：找到后序数组最后一个元素在中序数组的位置，作为切割点</font>
+ <font style="color:rgb(44, 62, 80);">第四步：切割中序数组，切成中序左数组和中序右数组 （顺序别搞反了，一定是先切中序数组）</font>
+ <font style="color:rgb(44, 62, 80);">第五步：切割后序数组，切成后序左数组和后序右数组</font>
+ <font style="color:rgb(44, 62, 80);">第六步：递归处理左区间和右区间</font>

<font style="color:rgb(44, 62, 80);"></font>

不难写出如下代码

```javascript
function traversal(inorder, postorder) {
  // 第一步	
  if (postorder.length === 0) return null;
  
  // 第二步: 后序遍历数组最后一个元素, 就是当前的中间节点
  
  let rootValue = postorder[postorder.length - 1];
  let root = new TreeNode(rootValue);
  
  // 叶子节点
  if (postorder.length === 1) return root;
  
  // 第三步: 找到切割点
  let delimiterIndex;
  for (delimiterIndex = 0; delimiterIndex < inorder.length; delimiterIndex++) {
    if (inorder[delimiterIndex] === rootValue) break;
  }
  
  // 第四步: 切割中序数组, 得到 中序左数组和中序右数组;
  // 第五步: 切割后序数组, 得到 后序左数组和后序有数组;
  
  // 第六步
  root.left = traversal(中序左数组, 后序左数组);
  root.right = traversal(中序右数组, 后序右数组);
  return root;  
}
```

<font style="color:rgb(44, 62, 80);"></font>

**<font style="color:rgb(44, 62, 80);">难点大家应该发现了，就是如何切割，以及边界值找不好很容易乱套。</font>**

<font style="color:rgb(44, 62, 80);">此时应该注意确定切割的标准，是左闭右开，还有左开右闭，还是左闭右闭，这个就是不变量，要在递归中保持这个不变量。</font>



**<font style="color:rgb(44, 62, 80);">在切割的过程中会产生四个区间，把握不好不变量的话，一会左闭右开，一会左闭右闭，必然乱套！</font>**

**<font style="color:rgb(44, 62, 80);"></font>**

<font style="color:rgb(44, 62, 80);">首先要切割中序数组，为什么先切割中序数组呢？</font>

<font style="color:rgb(44, 62, 80);">切割点在后序数组的最后一个元素，就是用这个元素来切割中序数组的，所以必要先切割中序数组。</font>

<font style="color:rgb(44, 62, 80);">中序数组相对比较好切，找到切割点（后序数组的最后一个元素）在中序数组的位置，然后切割，如下代码中我坚持左闭右开的原则：</font>

<font style="color:rgb(44, 62, 80);"></font>

```javascript
// 找到中序遍历的切割点
let delimiterIndex;
for (delimiterIndex = 0; delimiterIndex < inorder.length; delimiterIndex++) {
    if (inorder[delimiterIndex] === rootValue) break;
}

// 左闭右开区间：[0, delimiterIndex)
let leftInorder = inorder.slice(0, delimiterIndex);
// [delimiterIndex + 1, end)
let rightInorder = inorder.slice(delimiterIndex + 1);
```

<font style="color:rgb(44, 62, 80);">接下来就要切割后序数组了。</font>

<font style="color:rgb(44, 62, 80);">首先后序数组的最后一个元素指定不能要了，这是切割点 也是 当前二叉树中间节点的元素，已经用了。</font>

<font style="color:rgb(44, 62, 80);">后序数组的切割点怎么找？</font>

<font style="color:rgb(44, 62, 80);">后序数组没有明确的切割元素来进行左右切割，不像中序数组有明确的切割点，切割点左右分开就可以了。</font>

**<font style="color:rgb(44, 62, 80);">此时有一个很重的点，就是中序数组大小一定是和后序数组的大小相同的（这是必然）。</font>**

<font style="color:rgb(44, 62, 80);">中序数组我们都切成了左中序数组和右中序数组了，那么后序数组就可以按照左中序数组的大小来切割，切成左后序数组和右后序数组。</font>

<font style="color:rgb(44, 62, 80);">代码如下：</font>

```javascript
// postorder 舍弃末尾元素，因为这个元素就是中间节点，已经用过了
postorder.splice(postorder.length - 1, 1);

// 左闭右开，注意这里使用了左中序数组大小作为切割点：[0, leftInorder.size)
let leftPostorder = postorder.slice(0, leftInorder.length);
// [leftInorder.size(), end)
let rightPostorder = postorder.slice(leftInorder.length);
```

<font style="color:rgb(44, 62, 80);">此时，中序数组切成了左中序数组和右中序数组，后序数组切割成左后序数组和右后序数组。</font>

<font style="color:rgb(44, 62, 80);">接下来可以递归了，代码如下：</font>

```javascript
root.left = traversal(leftInorder, leftPostorder);
root.right = traversal(rightInorder, rightPostorder);
```



##### 完整代码如下:
```javascript
traversal(inorder, postorder) {
  if (postorder.length === 0) return null;
  
  let rootValue = postorder[postorder.length - 1];
  
  const root = new TreeNode(rootValue);
  
  if (postorder.length === 1) return root;
  
  let delimiterIndex;
  for (delimiterIndex = 0; delimiterIndex < inorder.length; delimiterIndex++) {
    if (inorder[delimiterIndex] === rootValue) break;
  }
  
  let leftInorder = inorder.slice(0, delimiterIndex);
  let rightInorder = inorder.slice(delimiterIndex + 1);
  
  postorder.splice(postorder.length - 1);
  
  let leftPostorder = postorder.slice(0, leftInorder.length);
  let rightPostorder = postorder.slice(leftInorder.length);
  
  root.left = traversal(leftInorder, leftPostorder);
  root.right = traversal(rightInorder, rightPostorder);
  
  return root;
}
```





##### 代码优化:
```javascript
var buildTree = function(inorder, postorder) {
    if (!inorder.length) return null;
    const rootVal = postorder.pop(); // 从后序遍历的数组中获取中间节点的值， 即数组最后一个值
    let rootIndex = inorder.indexOf(rootVal); // 获取中间节点在中序遍历中的下标
    const root = new TreeNode(rootVal); // 创建中间节点
    root.left = buildTree(inorder.slice(0, rootIndex), postorder.slice(0, rootIndex)); // 创建左节点
    root.right = buildTree(inorder.slice(rootIndex + 1), postorder.slice(rootIndex)); // 创建右节点
    return root;
};
```



### 二、[105. 从前序与中序遍历序列构造二叉树](https://leetcode.cn/problems/construct-binary-tree-from-preorder-and-inorder-traversal/)
<font style="color:rgb(44, 62, 80);">根据一棵树的前序遍历与中序遍历构造二叉树。</font>

<font style="color:rgb(44, 62, 80);">注意: 你可以假设树中没有重复的元素。</font>

<font style="color:rgb(44, 62, 80);">例如，给出</font>

<font style="color:rgb(44, 62, 80);">前序遍历 preorder = [3,9,20,15,7] 中序遍历 inorder = [9,3,15,20,7] 返回如下的二叉树：</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1654355153315-bd6c431e-0038-45d3-9d97-6d79e95e82ec.png)

#### 思路:
与上题一样



```javascript
/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {number[]} preorder
 * @param {number[]} inorder
 * @return {TreeNode}
 */
var buildTree = function(preorder, inorder) {
    if (preorder.length === 0) return null;

    const rootValue = preorder[0];

    const root = new TreeNode(rootValue);

    if (preorder.length === 1) return root;

    let delimiterIndex;
    for (delimiterIndex = 0; delimiterIndex < inorder.length; delimiterIndex++) {
        if (inorder[delimiterIndex] === rootValue) break;
    }

    let leftInorder = inorder.slice(0, delimiterIndex);
    let rightInorder = inorder.slice(delimiterIndex + 1);

    preorder.splice(0, 1);

    let leftPreorder = preorder.slice(0, leftInorder.length);
    let rightPreorder = preorder.splice(leftInorder.length);

    root.left = buildTree(leftPreorder, leftInorder);
    root.right = buildTree(rightPreorder, rightInorder);

    return root;
};
```



##### 代码优化
```javascript
var buildTree = function(preorder, inorder) {
  if (!preorder.length) return null;
  const rootVal = preorder.shift(); // 从前序遍历的数组中获取中间节点的值， 即数组第一个值
  const index = inorder.indexOf(rootVal); // 获取中间节点在中序遍历中的下标
  const root = new TreeNode(rootVal); // 创建中间节点
  root.left = buildTree(preorder.slice(0, index), inorder.slice(0, index)); // 创建左节点
  root.right = buildTree(preorder.slice(index), inorder.slice(index + 1)); // 创建右节点
  return root;
};
```



### 三、总结
前序和中序可以唯一确定一颗二叉树

后序和中序可以唯一确定一颗二叉树



那么前序和后序是否可以唯一确定一颗二叉树



**答案是不可以, ****<font style="color:rgb(44, 62, 80);">因为没有中序遍历无法确定左右部分，也就是无法分割。</font>**
