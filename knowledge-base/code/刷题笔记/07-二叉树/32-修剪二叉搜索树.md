### 一、[669. 修剪二叉搜索树](https://leetcode.cn/problems/trim-a-binary-search-tree/)
<font style="color:rgb(44, 62, 80);">给定一个二叉搜索树，同时给定最小边界L 和最大边界 R。通过修剪二叉搜索树，使得所有节点的值在[L, R]中 (R>=L) 。你可能需要改变树的根节点，所以结果应当返回修剪好的二叉搜索树的新的根节点。</font>

#### 思路
#### 递归法
+ 确定递归函数的参数以及返回值

<font style="color:rgb(44, 62, 80);">这里我们为什么需要返回值呢？</font>

<font style="color:rgb(44, 62, 80);">因为是要遍历整棵树，做修改，其实不需要返回值也可以，我们也可以完成修剪（其实就是从二叉树中移除节点）的操作。</font>

<font style="color:rgb(44, 62, 80);">但是有返回值，更方便，可以通过递归函数的返回值来移除节点。</font>

```javascript
function trimBST(root, low, high) {}
```

+ 确定终止条件

<font style="color:rgb(44, 62, 80);">修剪的操作并不是在终止条件上进行的，所以就是遇到空节点返回就可以了。</font>

```javascript
if (root === null) return root;
```

+ 确定单层逻辑

<font style="color:rgb(44, 62, 80);">如果</font>`<font style="color:rgb(44, 62, 80);">root</font>`<font style="color:rgb(44, 62, 80);">（当前节点）的元素小于</font>`<font style="color:rgb(44, 62, 80);">low</font>`<font style="color:rgb(44, 62, 80);">的数值，那么应该递归右子树，并返回右子树符合条件的头结点。</font>

```javascript
if (root.val < low) {
  const right = trimBST(root.right, low, high);
  return right;
}
```

<font style="color:rgb(44, 62, 80);">如果</font>`<font style="color:rgb(44, 62, 80);">root</font>`<font style="color:rgb(44, 62, 80);">(当前节点)的元素大于</font>`<font style="color:rgb(44, 62, 80);">high</font>`<font style="color:rgb(44, 62, 80);">的，那么应该递归左子树，并返回左子树符合条件的头结点。</font>

```javascript
if (root.val > high) {
  const left = trimBST(root.left, low, high);
  return left;
}
```

<font style="color:rgb(44, 62, 80);">接下来要将下一层处理完左子树的结果赋给</font>`<font style="color:rgb(44, 62, 80);">root.left</font>`<font style="color:rgb(44, 62, 80);">，处理完右子树的结果赋给</font>`<font style="color:rgb(44, 62, 80);">root.right</font>`<font style="color:rgb(44, 62, 80);">。最后返回</font>`<font style="color:rgb(44, 62, 80);">root</font>`<font style="color:rgb(44, 62, 80);">节点。</font>

```javascript
root.left = trimBST(root.left, low, high);
root.right = trimBST(root.right, low, high);
return root;
```



整体代码

```javascript
function trimBST(root, low, high) {
  if (root === null) return root;
  if (root.val < low) return trimBST(root.right, low, high);
  if (root.val > high) return trimBST(root.left, low, high);
  root.left = trimBST(root.left, low, high);
  root.right = trimBST(root.right, low, high);
  return root;
}
```



#### 迭代法
<font style="color:rgb(44, 62, 80);">因为二叉搜索树的有序性，不需要使用栈模拟递归的过程。</font>

<font style="color:rgb(44, 62, 80);">在剪枝的时候，可以分为三步：</font>

+ <font style="color:rgb(44, 62, 80);">将</font>`<font style="color:rgb(44, 62, 80);">root</font>`<font style="color:rgb(44, 62, 80);">移动到</font>`<font style="color:rgb(44, 62, 80);">[low, high]</font>`<font style="color:rgb(44, 62, 80);"> 范围内，注意是左闭右闭区间</font>
+ <font style="color:rgb(44, 62, 80);">剪枝左子树</font>
+ <font style="color:rgb(44, 62, 80);">剪枝右子树</font>

```javascript
function trimBST(root, low, high) {
  if (root === null) return root;
  // 处理头结点，让root移动到[L, R] 范围内，注意是左闭右闭
  while (root !== null && (root.val < low || root.val > high) {
    if (root.val < low) root = root.right; // 小于L往右走
    else root = root.left; // 大于R往左走
  }

  let cur = root;
  // 此时root已经在[L, R] 范围内，处理左孩子元素小于L的情况
  while (cur !== null) {
    while (cur.left && cur.left.val < low) {
      cur.left = cur.left.right;
    }
    cur = cur.left;
  }

  cur = root;
  // 此时root已经在[L, R] 范围内，处理右孩子大于R的情况
  while (cur !== null) {
    while (cur.right && cur.right > high) {
      cur.right = cur.right.left;
    }
    cur = cur.right;
  }

  return root;
}
```
