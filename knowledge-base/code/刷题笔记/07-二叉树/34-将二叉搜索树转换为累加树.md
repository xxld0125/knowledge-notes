#### <font style="color:rgb(44, 62, 80);">思路</font>
<font style="color:rgb(44, 62, 80);">一看到累加树，相信很多小伙伴都会疑惑：如何累加？遇到一个节点，然后在遍历其他节点累加？怎么一想这么麻烦呢。</font>

<font style="color:rgb(44, 62, 80);">然后再发现这是一棵二叉搜索树，二叉搜索树啊，这是有序的啊。</font>

<font style="color:rgb(44, 62, 80);">那么有序的元素如果求累加呢？</font>

**<font style="color:rgb(44, 62, 80);">其实这就是一棵树，大家可能看起来有点别扭，换一个角度来看，这就是一个有序数组[2, 5, 13]，求从后到前的累加数组，也就是[20, 18, 13]，是不是感觉这就简单了。</font>**

<font style="color:rgb(44, 62, 80);">因为数组大家都知道怎么遍历啊，从后向前，挨个累加就完事了，这换成了二叉搜索树，看起来就别扭了一些是不是。</font>

<font style="color:rgb(44, 62, 80);">那么知道如何遍历这个二叉树，也就迎刃而解了，</font>**<font style="color:rgb(44, 62, 80);">从树中可以看出累加的顺序是右中左，所以我们需要反中序遍历这个二叉树，然后顺序累加就可以了</font>**<font style="color:rgb(44, 62, 80);">。</font>

**<font style="color:rgb(44, 62, 80);"></font>**

#### 递归
<font style="color:rgb(44, 62, 80);">遍历顺序如图所示：</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1656581945464-dcb6c271-cf30-4c18-882d-7ee9be84c05a.png)

<font style="color:rgb(44, 62, 80);">本题依然需要一个</font>`<font style="color:rgb(44, 62, 80);">pre</font>`<font style="color:rgb(44, 62, 80);">指针记录当前遍历节点</font>`<font style="color:rgb(44, 62, 80);">cur</font>`<font style="color:rgb(44, 62, 80);">的前一个节点，这样才方便做累加。</font>

<font style="color:rgb(44, 62, 80);">pre指针的使用技巧，我们在</font>[25-二叉搜索树的最小绝对差](https://www.yuque.com/u25370234/kb/rc7vqk)<font style="color:rgb(44, 62, 80);">和</font>[26-二叉搜索树中的众数](https://www.yuque.com/u25370234/kb/gw9eg9)<font style="color:rgb(44, 62, 80);">都提到了，这是常用的操作手段。</font>



+ 递归函数参数以及返回值

<font style="color:rgb(44, 62, 80);">这里很明确了，不需要递归函数的返回值做什么操作了，要遍历整棵树。</font>

<font style="color:rgb(44, 62, 80);">同时需要定义一个全局变量</font>`<font style="color:rgb(44, 62, 80);">pre</font>`<font style="color:rgb(44, 62, 80);">，用来保存</font>`<font style="color:rgb(44, 62, 80);">cur</font>`<font style="color:rgb(44, 62, 80);">节点的前一个节点的数值。</font>

```javascript

let pre;
function traversal(cur) {};
```

+ 确定终止条件

遇空就终止

```javascript
if (cur === null) return;
```



+ 确定单层递归逻辑

<font style="color:rgb(44, 62, 80);">注意</font>**<font style="color:rgb(44, 62, 80);">要右中左来遍历二叉树</font>**<font style="color:rgb(44, 62, 80);">， 中节点的处理逻辑就是让cur的数值加上前一个节点的数值。</font>

```javascript
traversal(cur.right);
cur.val += pre;
pre = cur.val;
traversal(cur.left);
```

****

**整体代码**

```javascript
function convertBST(root) {
  let pre = 0;
  
  const traversal = function (cur) {
    if (cur === null) return;
    traversal(cur.right);
    cur.val += pre;
    pre = cur.val;
    traversal(cur.left);
  }
  
  traversal(root);
  return root;
}

```



#### 迭代法
<font style="color:rgb(44, 62, 80);">迭代法其实就是中序模板题了，在</font>[03-二叉树的迭代遍历](https://www.yuque.com/u25370234/kb/smsmle)<font style="color:rgb(44, 62, 80);">和</font>[04-二叉树的统一迭代法](https://www.yuque.com/u25370234/kb/xp51r2)<font style="color:rgb(44, 62, 80);">可以选一种自己习惯的写法。</font>

<font style="color:rgb(44, 62, 80);">这里给出其中的一种，代码如下：</font>

```javascript
function convertBST = function (root) {
  let pre = 0;
  let cur = root;
  let stack = [];
  
  while (cur !== null || stack.length) {
    while (cur !== null) {
      stack.push(cur);
      cur = cur.right;
    }
    
    cur = stack.pop();
    cur.val += pre;
    pre = cur.val;
    cur = cur.left;
  }
  
  return root;
}
```
