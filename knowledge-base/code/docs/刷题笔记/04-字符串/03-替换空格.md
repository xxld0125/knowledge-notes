### 一、例题-[剑指 Offer 05. 替换空格](https://leetcode.cn/problems/ti-huan-kong-ge-lcof/)
<font style="color:rgb(44, 62, 80);">请实现一个函数，把字符串 s 中的每个空格替换成"%20"。</font>

<font style="color:rgb(44, 62, 80);">示例 1： 输入：s = "We are happy."  
</font><font style="color:rgb(44, 62, 80);">	输出："We%20are%20happy."</font>



#### 思路
<font style="color:rgb(44, 62, 80);">如果想把这道题目做到极致，就不要只用额外的辅助空间了！</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">首先扩充数组到每个空格替换成"%20"之后的大小。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">然后从后向前替换空格，也就是双指针法，过程如下：</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">i指向新长度的末尾，j指向旧长度的末尾。</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1652598799049-cd6fb457-555d-4ff9-9fbd-007930aa7ca2.png)

<font style="color:rgb(44, 62, 80);">为什么要从后向前填充，从前向后填充不行么？</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">从前向后填充就是O(n^2)的算法了，因为每次添加元素都要将添加元素之后的所有元素向后移动。</font>

<font style="color:rgb(44, 62, 80);"></font>

**<font style="color:rgb(44, 62, 80);"> 	其实很多数组填充类的问题，都可以先预先给数组扩容带填充后的大小，然后在从后向前进行操作。</font>**

**<font style="color:rgb(44, 62, 80);"></font>**

<font style="color:rgb(44, 62, 80);">这么做有两个好处：</font>

1. <font style="color:rgb(44, 62, 80);">不用申请新数组。</font>
2. <font style="color:rgb(44, 62, 80);">从后向前填充元素，避免了从前先后填充元素要来的 每次添加元素都要将添加元素之后的所有元素向后移动。</font>

<font style="color:rgb(44, 62, 80);">时间复杂度，空间复杂度均超过100%的用户。</font>





```javascript
/**
 * @param {string} s
 * @return {string}
 */
var replaceSpace = function(s) {
    // return s.replace(/\s/g, '%20');

    const len = s.length;
    const res = Array.from(s);
    let count = 0;
    
    // 获取空格数量
    for (let i = 0;i < len;i++) {
        if (res[i] === ' ') {
            count++;
        }
    }

    let l = len - 1,
        r = len + count * 2 - 1;
    
    while (l >= 0) {
        if (res[l] === ' ') {
            res[r--] = '0';
            res[r--] = '2';
            res[r--] = '%';
            l--;
        } else {
            res[r--] = res[l--];
        }
    }

    return res.join('');
};
```
