### 一、例题-[239. 滑动窗口最大值](https://leetcode.cn/problems/sliding-window-maximum/)
<font style="color:rgb(44, 62, 80);">给定一个数组 nums，有一个大小为 k 的滑动窗口从数组的最左侧移动到数组的最右侧。你只可以看到在滑动窗口内的 k 个数字。滑动窗口每次只向右移动一位。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">返回滑动窗口中的最大值。</font>

  
	**<font style="color:rgb(44, 62, 80);">进阶：</font>**

<font style="color:rgb(44, 62, 80);">你能在线性时间复杂度内解决此题吗？</font>

<font style="color:rgb(44, 62, 80);"></font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1652708021678-e986855b-49eb-4047-8a02-21033e173b0b.png)

<font style="color:rgb(44, 62, 80);">提示：</font>

+ <font style="color:rgb(44, 62, 80);">1 <= nums.length <= 10^5</font>
+ <font style="color:rgb(44, 62, 80);">-10^4 <= nums[i] <= 10^4</font>
+ <font style="color:rgb(44, 62, 80);">1 <= k <= nums.length</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);"></font>

#### <font style="color:rgb(44, 62, 80);">思路</font>
<font style="color:rgb(44, 62, 80);">这是使用单调队列的经典题目。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">难点是如何求一个区间里的最大值呢, 使用暴力方法的话，遍历一遍的过程中每次从窗口中在找到最大的数值，这样很明显是$O(n × k)$的算法。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">有的同学可能会想用一个大顶堆（优先级队列）来存放这个窗口里的k个数字，这样就可以知道最大的最大值是多少了， </font>**<font style="color:rgb(44, 62, 80);">但是问题是这个窗口是移动的，而大顶堆每次只能弹出最大值，我们无法移除其他数值，这样就造成大顶堆维护的不是滑动窗口里面的数值了。所以不能用大顶堆。</font>**

**<font style="color:rgb(44, 62, 80);"></font>**

<font style="color:rgb(44, 62, 80);">此时我们需要一个队列，这个队列呢，放进去窗口里的元素，然后随着窗口的移动，队列也一进一出，每次移动之后，队列告诉我们里面的最大值是什么。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">这个队列应该长这个样子：</font>

```javascript
class MyQueue {
    pop(value) {
      
    }
    void push(value) {
      
    }
    front() {
        return que.front();
    }
};
```

<font style="color:rgb(44, 62, 80);">每次窗口移动的时候，调用que.pop(滑动窗口中移除元素的数值)，que.push(滑动窗口添加元素的数值)，然后que.front()就返回我们要的最大值。</font>

  
	<font style="color:rgb(44, 62, 80);">然后在分析一下，队列里的元素一定是要排序的，而且要最大值放在出队口，要不然怎么知道最大值呢。</font>

<font style="color:rgb(44, 62, 80);">但如果把窗口里的元素都放进队列里，窗口移动的时候，队列需要弹出元素。</font>

<font style="color:rgb(44, 62, 80);">那么问题来了，已经排序之后的队列 怎么能把窗口要移除的元素（这个元素可不一定是最大值）弹出呢。</font>



**<font style="color:rgb(44, 62, 80);">其实队列没有必要维护窗口里的所有元素，只需要维护有可能成为窗口里最大值的元素就可以了，同时保证队里里的元素数值是由大到小的。</font>**

<font style="color:rgb(44, 62, 80);">那么这个维护元素单调递减的队列就叫做</font>**<font style="color:rgb(44, 62, 80);">单调队列，即单调递减或单调递增的队列。C++中没有直接支持单调队列，需要我们自己来一个单调队列</font>**

**<font style="color:rgb(44, 62, 80);">不要以为实现的单调队列就是 对窗口里面的数进行排序，如果排序的话，那和优先级队列又有什么区别了呢。</font>**

<font style="color:rgb(44, 62, 80);">来看一下单调队列如何维护队列里的元素。</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1652708638682-4405e0f5-2f72-48f5-ad67-a6eb5eb527bb.png)



<font style="color:rgb(44, 62, 80);">对于窗口里的元素{2, 3, 5, 1 ,4}，单调队列里只维护{5, 4} 就够了，保持单调队列里单调递减，此时队列出口元素就是窗口里最大元素。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">此时大家应该怀疑单调队列里维护着{5, 4} 怎么配合窗口经行滑动呢？</font>

<font style="color:rgb(44, 62, 80);">设计单调队列的时候，pop，和push操作要保持如下规则：</font>

<font style="color:rgb(44, 62, 80);"></font>

1. <font style="color:rgb(44, 62, 80);">pop(value)：如果窗口移除的元素value等于单调队列的出口元素，那么队列弹出元素，否则不用任何操作</font>
2. <font style="color:rgb(44, 62, 80);">push(value)：如果push的元素value大于入口元素的数值，那么就将队列入口的元素弹出，直到push元素的数值小于等于队列入口元素的数值为止</font>

<font style="color:rgb(44, 62, 80);"></font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1652708741185-4c561814-51bf-4da0-93fd-5ed2540842b5.png)

<font style="color:rgb(44, 62, 80);">  
</font>

```javascript
var maxSlidingWindow = function (nums, k) {
  class MonoQueue {
    queue;
    constructor() {
      this.queue = [];
    }
    enqueue(value) {
      let back = this.queue[this.queue.length - 1];
      while (back !== undefined && back < value) {
        this.queue.pop();
        back = this.queue[this.queue.length - 1];
      }
      this.queue.push(value);
    }
    dequeue(value) {
      let front = this.front();
      if (front === value) {
        this.queue.shift();
      }
    }
    front() {
      return this.queue[0];
    }
  }
  let helperQueue = new MonoQueue();
  let i = 0, j = 0;
  let resArr = [];
  
  while (j < k) {
    helperQueue.enqueue(nums[j++]);
  }
  resArr.push(helperQueue.front());
  while (j < nums.length) {
    helperQueue.enqueue(nums[i]);
    helperQueue.dequeue(nums[i]);
    resArr.push(helperQueue.front());
    j++;
    i++;
  }
  return resArr;
}
```
