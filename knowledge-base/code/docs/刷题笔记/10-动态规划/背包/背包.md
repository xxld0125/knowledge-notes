### 解题思路
1. 判断题目是否为背包类问题
2. 判断是01背包还是完全背包
3. 使用动归五部曲解题
    1. 定义dp数组的含义

优先使用一维dp数组，没有思路的话，考虑使用二维dp数组

    2. 确定dp数组的递推公式

目前遇到的问题有以下几类

        1. 根据物品和背包，求价值类问题
            1. 求最大价值
            2. 求是否刚好装满背包(容积与价值相同)
            3. 求装满有几种方法
        2. 求组合数
            1. 求组合数(不要求顺序)
            2. 求组合数(要求顺序)
    3. 确定dp数组的遍历顺序

一般问题都是先遍历物品，后遍历背包。

特殊场景：

当遇到求组合数时，判断是否要求组合的顺序，是的话则先遍历背包，后遍历物品。

    4. dp数组的初始化

需要根据题目具体分析
