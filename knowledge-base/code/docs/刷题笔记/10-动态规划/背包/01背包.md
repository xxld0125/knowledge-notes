### 经典的01背包问题
有`n`件物品和一个最多能背重量为 `w` 的背包。第`i`件物品的重量是`weight[i]`，得到的价值是`value[i]` 。每件物品只能用一次，求解将哪些物品装入背包里物品价值总和最大。



### 解题思路(二维数组)
使用动归五部曲分析

#### 1.确定dp数组的定义
`dp[i][j] `表示从下标为`[0-i]`的物品里任意取，放进容量为`j`的背包，价值总和最大是多少。



#### 2.确定递推公式
`dp[i][j]`的含义：从下标为`[0-i]`的物品里任意取，放进容量为`j`的背包，价值总和最大是多少。

那么可以有两个方向推出来dp[i][j]，

+ 不放物品`i`：由`dp[i - 1][j]`推出，即背包容量为j，里面不放物品`i`的最大价值，此时`dp[i][j]`就是`dp[i - 1][j]`。(其实就是当物品i的重量大于背包j的重量时，物品i无法放进背包中，所以背包内的价值依然和前面相同。)
+ 放物品`i`：由`dp[i - 1][j - weight[i]]`推出，`dp[i - 1][j - weight[i]] `为背包容量为`j - weight[i]`的时候不放物品`i`的最大价值，那么`dp[i - 1][j - weight[i]] + value[i] `（物品`i`的价值），就是背包放物品`i`得到的最大价值

所以递归公式： `dp[i][j] = max(dp[i - 1][j], dp[i - 1][j - weight[i]] + value[i]);`



#### 3.dp数组如何初始化
如果背包容量`j`为0的话，即`dp[i][0]`，无论是选取哪些物品，背包价值总和一定为0。

```javascript
for (let i = 0; i < n; i++) {
  dp[i][0] = 0;
}
```



![](https://cdn.nlark.com/yuque/0/2023/png/25743026/1701073343679-c3d8d50b-5e47-4ee1-8415-e9344aa7ef6d.png)



状态转移方程 `dp[i][j] = max(dp[i - 1][j], dp[i - 1][j - weight[i]] + value[i])`， 可以看出 `i` 是由 `i-1` 推导出来，那么`i`为`0`的时候就一定要初始化。

`dp[0][j]`，即：`i`为0，存放编号0的物品的时候，各个容量的背包所能存放的最大价值。

那么很明显当 `j < weight[0]`的时候，`dp[0][j] `应该是 0，因为背包容量比编号0的物品重量还小。

当`j >= weight[0]`时，`dp[0][j]` 应该是`value[0]`，因为背包容量放足够放编号0物品。

 	代码初始化如下：

```javascript
for (let j = 0; j < weight[0]; j++) {
  dp[0][j] = 0;
}
for (let j = weight[0]; j <= w; j++) {
  dp[0][j] = value[0];
}
```

`dp[0][j]`和 `dp[i][0]` 都已经初始化了，那么其他下标应该初始化多少呢？

其实从递归公式： `dp[i][j] = max(dp[i - 1][j], dp[i - 1][j - weight[i]] + value[i])`; 可以看出`dp[i][j] `是由左上方数值推导出来了，那么 其他下标初始为什么数值都可以，因为都会被覆盖。

初始-1，初始-2，初始100，都可以！

但只不过一开始就统一把dp数组统一初始为0，更方便一些。

#### 4.确定遍历顺序
从下图可以看出，有两个遍历的维度：物品和背包重量：

![](https://cdn.nlark.com/yuque/0/2023/png/25743026/1701073587459-2079cbbd-5038-4bf1-a804-de6da7f4657c.png)



所以需要确认，先遍历物品还是先遍历背包重量？



先遍历物品，后遍历背包重量

```javascript
for (let i = 0; i < n; i++) {
  for (let j = 0; j <= w; j++) {
    if (j < weight[0]) {
      dp[i][j] = dp[i - 1][j];
    } else {
    	dp[i][j] = Math.max(dp[i - 1][j], dp[i - 1][j - weight[i]] + values[i]);
    }
  }
}
```



先遍历背包重量，后遍历物品

```javascript
for (let j = 0; j <= w; j++) {
  for (let i = 0; i < n; i++) {
    if (j < weight[0] {
      dp[i][j] = dp[i - 1][j];
    } else {
      dp[i][j] = Math.max(dp[i - 1][j], dp[i - 1][j - weight[i]] + values[i]);
    }
  }
}
```

	

上面两种遍历顺序都是可以的，根据递推公式：`dp[i][j] = max(dp[i - 1][j], dp[i - 1][j - weight[i]] + value[i])`，可以看出`dp[i][j]`是靠`dp[i-1][j]`和`dp[i - 1][j - weight[i]]`推导出来的。

`dp[i-1][j]`和`dp[i - 1][j - weight[i]] `都在`dp[i][j]`的左上角方向（包括正上方向），虽然两个for循环遍历的次序不同，但是`dp[i][j]`所需要的数据就是左上角，根本不影响`dp[i][j]`公式的推导！

但先遍历物品再遍历背包这个顺序更好理解。



#### 5.完整代码
```javascript
function testWeightBagProblem(n, w, weight, values) {
  const dp = Array(n).fill().map(v => Array(w + 1).fill(0));

  for (let j = weight[0]; j <= w; j++) {
    dp[0][j] = weight[0];
  }
  
  for (let i = 0; i < n; i++) {
    for (let j = 0; j <= w; j++) {
      if (j <= weight[0]) {
        dp[i][j] = dp[i - 1][j];
      }else {
        dp[i][j] = Math.max(dp[i - 1][j], dp[i - 1][j - weight[i]] + values[i]);
    	}
    }   
  }

  return dp[n - 1][w];
}

```

  


### 解题思路(一维数组)
#### 1.确定dp数组的定义
在一维dp数组中，`dp[j]`表示：容量为`j`的背包，所背的物品价值可以最大为`dp[j]`。



#### 2.确定递推公式
`dp[j]`可以通过`dp[j - weight[i]]`推导出来，`dp[j - weight[i]]`表示容量为`j - weight[i]`的背包所背的最大价值。

`dp[j - weight[i]] + value[i]` 表示 容量为 j - 物品i重量 的背包 加上 物品i的价值。（也就是容量为j的背包，放入物品i了之后的价值即：dp[j]）

此时`dp[j]`有两个选择，一个是取自己`dp[j]`相当于 二维dp数组中的`dp[i-1][j]`，即不放物品`i`，一个是取`dp[j - weight[i]] + value[i]`，即放物品`i`，指定是取最大的，毕竟是求最大价值，

所以递推公式为：

```javascript
dp[j] = Math.max(dp[j], dp[j - weight[i]] + values[i]);
```



#### 3.dp数组如何初始化
`dp[j]`表示：容量为`j`的背包，所背的物品价值可以最大为`dp[j]`，那么`dp[0]`就应该是0，因为背包容量为0所背的物品的最大价值就是0。

那么dp数组除了下标0的位置，初始为0，其他下标应该初始化多少呢？

看一下递归公式：`dp[j] = max(dp[j], dp[j - weight[i]] + value[i])`;

dp数组在推导的时候一定是取价值最大的数，如果题目给的价值都是正整数那么非0下标都初始化为0就可以了。

这样才能让dp数组在递归公式的过程中取的最大的价值，而不是被初始值覆盖了。

那么我假设物品价值都是大于0的，所以dp数组初始化的时候，都初始为0就可以了。



#### 4.确定遍历顺序
```javascript
for (let i = 0; i < n; i++) { // 遍历物品
  for (let j = w; j >= weight[i]; j--) { // 遍历背包容量
  	dp[j] = Math.max(dp[j], dp[j - weight[i]] + values[i]);
  }
}
```

一维dp数组的遍历顺序和二维是不一样的，二维dp遍历的时候，背包容量是从小到大，而一维dp遍历的时候，背包是从大到小。

倒序遍历是为了保证物品i只被放入一次！。但如果一旦正序遍历了，那么物品0就会被重复加入多次！  


举一个例子：物品0的重量`weight[0] = 1`，价值`value[0] = 15`

如果正序遍历

`dp[1] = dp[1 - weight[0]] + value[0] = 15`

`dp[2] = dp[2 - weight[0]] + value[0] = 30`

此时`dp[2]`就已经是30了，意味着物品0，被放入了两次，所以不能正序遍历。

为什么倒序遍历，就可以保证物品只放入一次呢？

倒序就是先算`dp[2]`

`dp[2] = dp[2 - weight[0]] + value[0] = 15` （dp数组已经都初始化为0）

`dp[1] = dp[1 - weight[0]] + value[0] = 15`

所以从后往前循环，每次取得状态不会和之前取得状态重合，这样每种物品就只取一次了。



再来看看两个嵌套for循环的顺序，代码中是先遍历物品嵌套遍历背包容量，那可不可以先遍历背包容量嵌套遍历物品呢？

不可以！因为一维dp的写法，背包容量一定是要倒序遍历（原因上面已经讲了），如果遍历背包容量放在上一层，那么每个dp[j]就只会放入一个物品，即：背包里只放入了一个物品。

倒序遍历的原因是，本质上还是一个对二维数组的遍历，并且右下角的值依赖上一层左上角的值，因此需要保证左边的值仍然是上一层的，从右向左覆盖。



<font style="color:#DF2A3F;">一维dp数组遍历的特殊性：</font>

+ <font style="color:#DF2A3F;">物品是正序遍历，背包容量是倒序遍历</font>
+ <font style="color:#DF2A3F;">先遍历物品，再遍历背包容量</font>

<font style="color:#DF2A3F;"></font>

#### 5.完整代码
```javascript
function testWeightBagProblem(n, w, weight, values) {
  const dp = Array(w + 1).fill(0);
  
  for (let i = 0; i < n; i++) {
    for (let j = w; j >= weight[i]; j--) {
      dp[j] = Math.max(dp[j], dp[j - weight[i]] + values[i]);
    }
  }

  return dp[j];
}
```

#### 3.[494.目标和](https://leetcode.cn/problems/target-sum/description/)
之前的背包问题一般是，根据背包容积求最大价值，判断背包是否装满(背包容积与价值相同)。

本题是求装满背包有多少种方法，解题思路：



**一维dp数组**

1. 确定dp数组以及下标的含义

`dp[j]`表示：填满`j`（包括`j`）这么大容积的包，有`dp[j]`种方法

2. 确定递推公式

`dp[j] += dp[j - nums[i]]`

注意：求装满背包有几种方法类似的题目，递推公式基本都是这样的。

3. dp数组如何初始化

`dp[0] `初始化为1 ，`dp[j]`其他下标对应的数值应该初始化为0。

4. 确定遍历顺序

01背包问题一维dp的遍历，`nums`放在外循环，`target`在内循环，且内循环倒序。

5. 具体代码实现

```typescript
function findTargetSumWays(nums: number[], target: number): number {
  const sum = nums.reduce((pre, cur) => pre + cur, 0);

  // 处理边界场景
  // 处理负数和位非整数的异常场景
  if ((sum - target) % 2) return 0;
  // 处理目标和绝对值大于非负整数数组和的异常场景
  if (Math.abs(target) > sum) return 0;

  // 负数和
  const reg = (sum - target) / 2;

  // 定义dp数组
  const dp = new Array(reg + 1).fill(0);

  // dp数组初始化
  // 当容量为0时, 有一种方法
  dp[0] = 1;

  for (let i = 0; i < nums.length; i++) {
    for (let j = reg; j >= nums[i]; j--) {
      dp[j] += dp[j - nums[i]];
    }
  }

  return dp[reg];
};
```

**二维dp数组**

1. 确定dp数组以及下标的含义

`dp[i][j]`表示：i个物品, 填满`j`（包括`j`）这么大容积的包，有`dp[i][j]`种方法

2. 确定递推公式

`j < num`(数组nums中的第i个元素)时:

`dp[i][j] = dp[i - 1][j]`

`j >= num`(数组nums中的第i个元素)时:

`dp[i][j] = dp[i - 1][j] + dp[i - 1][j - num]`

注意：求装满背包有几种方法类似的题目，递推公式基本都是这样的。

3. dp数组如何初始化

当无物品(i = 0)时, 此时只有背包容量为0(j = 0)下, 有一种方法

`dp[0][0] = 1`

4. 确定遍历顺序

01背包问题二维dp的遍历，先遍历物品, 后遍历背包, 都是正序遍历

5. 具体代码实现

```typescript
function findTargetSumWays(nums: number[], target: number): number {
    // 处理边界场景
    // 1.target + sum 为奇数时, 正数部分则为非整数, 但是根据题目不会存在该场景
    const sum = nums.reduce((pre, cur) => pre + cur, 0);
    if ((sum + target) % 2) return 0;

    // 2.target绝对值 > sum时, 无该场景
    if (Math.abs(target) > sum) return 0;

    const neg = (sum - target) / 2;
    const len = nums.length;

    // 动态规划: 二维数组
    // 1.dp数组定义: 数组nums的前i个数中选取元素, 使得这些元素之和等于j, 一共有dp[i][j]种方法
    // 2.递推公式:
    //  当1 <= i <= n时, 对于数组nums中的第i个元素num, 遍历o <= j <= halfSum, 计算dp[i][j]
    //      如果j < num, 则不能选num, 此时dp[i][j] = dp[i - 1][j]
    //      如果j > num, 则如果不选num, 方案数是dp[i - 1][j], 如果选num, 方案数dp[i - 1][j - num], 此时dp[i][j] = dp[i - 1][j] + dp[i - 1][j - num];
    // 3.dp数组初始化:
    //    当前i为0时(没有任何元素可以选取时), j只能是0, 方案数: 1
    //    当前i为0时(没有任何元素可以选取时), j >= 1, 方案数: 0

    // 之前维护二维dp数组时, 物品维度的数组长度为len, 但是此题目中, 物品0的计算需要依赖无物品的值, 所以物品长度为len + 1,下标为0时表示无物品, 1为物品0
    const dp: number[][] = new Array(len + 1).fill(0).map(() => new Array(neg + 1).fill(0));

    // 问题: 初始化场景
    // 当无物品(i = 0)时, 此时只有背包容量为0(j = 0)下, 有一种方法
    dp[0][0] = 1;

    for (let i = 1; i <= len; i++) {
        const num = nums[i - 1];
        for (let j = 0; j <= neg; j++) {
            dp[i][j] = dp[i - 1][j];
            if (j >= num) {
                dp[i][j] += dp[i - 1][j - num];
            }
        }
    }

    return dp[len][neg];
};
```





#### 4.[474.一和零](https://leetcode.cn/problems/ones-and-zeroes/description/)
本题与标准01背包的区别就是背包有两个维度，一个是m，一个是n，而不同长度的字符串就是不同大小的待装物品。



**二维dp数组**

1. 确定dp数组以及下标的含义

最多有`i`个0, `j`个1组成的最大子集的大小`dp[i][j]`

2. 确定递推公式

`dp[i][j] = Math.max(dp[i][j], dp[i - x][j - y] + 1)`

3. dp数组如何初始化

dp数组初始化为0

4. 确定遍历顺序

物品正序遍历, 背包容量倒序遍历

5. 具体代码实现

```typescript
function findMaxForm(strs: string[], m: number, n: number): number {
		const dp: number[][] = new Array(m + 1).fill(0).map(() => new Array(n + 1).fill(0));

    for (let str of strs) {
        let zn = 0, on = 0;
        for (let s of str) {
            if (s === '0') {
                zn++;
            } else {
                on++;
            }
        }
        for (let i = m; i >= zn; i--) {
            for (let j = n; j >= on; j--) {
                dp[i][j] = Math.max(dp[i][j], dp[i - zn][j - on] + 1);
            }
        }
    }

    return dp[m][n];
};
```



**三维dp数组**

1. 确定dp数组以及下标的含义

有a个字符串, 最多有i个0, j个1, 组成最大子集的大小为dp[a][i][j]

2. 确定递推公式

`dp[a][i][j] = Math.max(dp[a - 1][i][j], dp[a - 1][i - zn][i - on] + 1)`

3. dp数组如何初始化

dp数组初始化为0

4. 确定遍历顺序

物品正序遍历, 背包容量正序遍历

5. 具体代码实现

```typescript
function findMaxForm(strs: string[], m: number, n: number): number {
    const len = strs.length;

    const dp: number[][][] = new Array(len).fill(0).map(() => new Array(m + 1).fill(0).map(() => new Array(n + 1).fill(0)));

    for (let i = 0; i <= m; i++) {
        for (let j = 0; j <= n; j++) {
            const { numOfZero, numOfOne } = countBinary(strs[0]);
            if (i >= numOfZero && j >= numOfOne) {
                dp[0][i][j] = 1;
            }
        }
    }

    for (let a = 1; a < len; a++) {
        const { numOfZero, numOfOne } = countBinary(strs[a]);
        for (let i = 0; i <= m; i++) {
            for (let j = 0; j <= n; j++) {
                if (i >= numOfZero && j >= numOfOne) {
                    dp[a][i][j] = Math.max(dp[a - 1][i][j], dp[a - 1][i - numOfZero][j - numOfOne] + 1);
                } else {
                    dp[a][i][j] = dp[a - 1][i][j];
                }
            }
        }
    }

    return dp[len - 1][m][n];
};
type BinaryInfo = { numOfZero: number, numOfOne: number };
function countBinary(str: string): BinaryInfo {
    let numOfZero: number = 0,
        numOfOne: number = 0;
    for (let s of str) {
        if (s === '0') {
            numOfZero++;
        } else {
            numOfOne++;
        }
    }
    return { numOfZero, numOfOne };
}
```
