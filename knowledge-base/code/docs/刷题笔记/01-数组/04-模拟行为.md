### 一、例题-[59. 螺旋矩阵 II](https://leetcode.cn/problems/spiral-matrix-ii/)
<font style="color:rgb(44, 62, 80);">给定一个正整数 n，生成一个包含 1 到 n^2 所有元素，且元素按顺时针顺序螺旋排列的正方形矩阵。</font>

<font style="color:rgb(44, 62, 80);">示例:</font>

<font style="color:rgb(44, 62, 80);">输入: 3 输出: [ [ 1, 2, 3 ], [ 8, 9, 4 ], [ 7, 6, 5 ] ]</font>  


#### 思路:
<font style="color:rgb(44, 62, 80);">这道题目可以说在</font>**<font style="color:#E8323C;">面试中出现频率较高</font>**<font style="color:rgb(44, 62, 80);">的题目，</font>**<font style="color:rgb(44, 62, 80);">本题并不涉及到什么算法，就是模拟过程，但却十分考察对代码的掌控能力。</font>**

**<font style="color:rgb(44, 62, 80);"></font>**

**<font style="color:rgb(44, 62, 80);">模拟顺时针画矩阵的过程:</font>**

+ <font style="color:rgb(44, 62, 80);">填充上行从左到右</font>
+ <font style="color:rgb(44, 62, 80);">填充右列从上到下</font>
+ <font style="color:rgb(44, 62, 80);">填充下行从右到左</font>
+ <font style="color:rgb(44, 62, 80);">填充左列从下到上</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1652095512903-e3b7c8f7-8f78-4f57-b2c2-04e7116a7809.png)  
	**其次要注意的是****<font style="color:rgb(44, 62, 80);">要坚持循环不变量原则。</font>**

<font style="color:rgb(44, 62, 80);">这里每一种颜色，代表一条边，我们遍历的长度，可以看出每一个拐角处的处理规则，拐角处让给新的一条边来继续画。</font>

<font style="color:rgb(44, 62, 80);">这也是坚持了每条边左闭右开的原则。</font>

```javascript
/**
 * @param {number} n
 * @return {number[][]}
 */
var generateMatrix = function(n) {
    let startX = startY = 0, // 起始位置
        offset = 1, // 每行不填充的数量
        count = 1, // 填充数字
        loop = Math.floor(n / 2), // 循环圈数 
        mid = Math.floor(n / 2), // 中间位置
        res = new Array(n).fill(0).map(v => new Array(n).fill(0));

    while (loop--) {
        let row = startY, col = startX;

        while (col < n + startX - offset) {
            res[row][col] = count++;
            col++;
        }

        while (row < n + startY - offset) {
            res[row][col] = count++;
            row++;
        }

        while (col > startX) {
            res[row][col] = count++;
            col--;
        }

        while (row > startY) {
            res[row][col] = count++;
            row--;
        }

        startX++;
        startY++;

        offset += 2;
    }

    if (n % 2) {
        res[mid][mid] = count;
    }
    return res;
};
```
