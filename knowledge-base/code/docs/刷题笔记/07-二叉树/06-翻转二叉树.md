<font style="color:rgb(44, 62, 80);">翻转一棵二叉树。</font>

#### 思路:
<font style="color:rgb(44, 62, 80);">可以发现想要翻转它，其实就把每一个节点的左右孩子交换一下就可以了。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">关键在于遍历顺序，前中后序应该选哪一种遍历顺序？ （一些同学这道题都过了，但是不知道自己用的是什么顺序）</font>

<font style="color:rgb(44, 62, 80);">遍历的过程中去翻转每一个节点的左右孩子就可以达到整体翻转的效果。</font>

**<font style="color:rgb(44, 62, 80);">注意只要把每一个节点的左右孩子翻转一下，就可以达到整体翻转的效果</font>**<font style="color:rgb(44, 62, 80);">。</font>

**<font style="color:rgb(44, 62, 80);">这道题目使用前序遍历和后序遍历都可以，唯独中序遍历不方便，因为中序遍历会把某些节点的左右孩子翻转了两次！建议拿纸画一画，就理解了</font>**

**<font style="color:rgb(44, 62, 80);"></font>**

<font style="color:rgb(44, 62, 80);">那么层序遍历可以不可以呢？</font>**<font style="color:rgb(44, 62, 80);">依然可以的！只要把每一个节点的左右孩子翻转一下的遍历方式都是可以的！</font>**

**<font style="color:rgb(44, 62, 80);"></font>**

#### 递归法
递归三部曲

1. <font style="color:rgb(44, 62, 80);">确定递归函数的参数和返回值</font>

<font style="color:rgb(44, 62, 80);">参数就是要传入节点的指针，不需要其他参数了，通常此时定下来主要参数，如果在写递归的逻辑中发现还需要其他参数的时候，随时补充。</font>

<font style="color:rgb(44, 62, 80);">返回值的话其实也不需要，但是题目中给出的要返回root节点的指针，可以直接使用题目定义好的函数，所以就函数的返回类型为</font>`<font style="color:rgb(71, 101, 130);">TreeNode</font>`<font style="color:rgb(44, 62, 80);">。</font>

```javascript
var invertTree = function(root) {
  // ...
  return root;
}
```

<font style="color:rgb(44, 62, 80);"></font>

2. <font style="color:rgb(44, 62, 80);">确定终止条件</font>

<font style="color:rgb(44, 62, 80);">当前节点为空的时候，就返回</font>

```javascript
if (root === null)
```

<font style="color:rgb(44, 62, 80);"></font>

3. <font style="color:rgb(44, 62, 80);">确定单层递归的逻辑</font>

<font style="color:rgb(44, 62, 80);">因为是先前序遍历，所以先进行交换左右孩子节点，然后反转左子树，反转右子树。</font>

```javascript
  inverNode(root.left, root.right);
  invertTree(root.left);
  invertTree(root.right);
```

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">基于这递归三步法，代码基本写完</font>

```javascript
var invertTree = function(root) {
  //  交换节点函数
  const inverNode = function(left, right) {
    let temp = left;
    left = right;
    right = temp;
    // 需要重新给root赋值下;
    root.left = left;
    root.right = right;
  }
  // 确定递归函数的参数和返回条件inverTree = function(root)
  // 确定终止条件
  if (root === null) {
    return root;
  }
  // 确定节点交换逻辑 交换
  inverNode(root.left, root.right);
  invertTree(root.left);
  invertTree(root.right);
  return root;
}
```





#### 迭代法-深度优先遍历-前序遍历
在[03-二叉树的迭代遍历](https://www.yuque.com/u25370234/kb/smsmle)中,<font style="color:rgb(44, 62, 80);">出了前中后序迭代方式的写法。</font>

<font style="color:rgb(44, 62, 80);">在</font>[04-二叉树的统一迭代法](https://www.yuque.com/u25370234/kb/xp51r2)中介绍了统一的写法, 所以,本题也只需要将文中的代码稍作修改即可<font style="color:rgb(44, 62, 80);">。</font>

```javascript
var invertTree = function(root) {
 const inverNode = function(root, left, right) {
    root.left = right;
    root.right = left;
  }
  const stack = [];
  
  if (root === null) return root;

  stack.push(root);
  
  while(stack.length) {
    let node = stack.pop();
    if (node === null) {
      node = stack.pop();
      inverNode(node, node.left, node.right);
      continue;
    }
    
    node.right && stack.push(node.right);
    node.left && stack.push(node.left);
    stack.push(node);
    stack.push(null);
  }
  return root;
}
```



#### 迭代法-深度优先遍历-中序遍历
<font style="color:rgb(44, 62, 80);">递归的中序遍历是不行的，因为使用递归的中序遍历，某些节点的左右孩子会翻转两次。</font>

<font style="color:rgb(44, 62, 80);">但使用迭代方式统一写法的中序是可以的。</font>

```javascript
var invertTree = function(root) {
 const inverNode = function(root, left, right) {
    [left, right] = [right,left];
    root.left = left;
    root.right = right;
  }
  const stack = [];
  
  if (root === null) return root;

  stack.push(root);
  
  while(stack.length) {
    let node = stack.pop();
    if (node === null) {
      node = stack.pop();
      inverNode(node, node.left, node.right);
      continue;
    }
    
    node.right && stack.push(node.right);
    stack.push(node);
    stack.push(null);
    node.left && stack.push(node.left);
  }
  return root;
}
```





#### 迭代法-广度优先遍历
<font style="color:rgb(44, 62, 80);">也就是层序遍历,在</font>[05-二叉树的层序遍历](https://www.yuque.com/u25370234/kb/ugyidb)<font style="color:rgb(44, 62, 80);">有介绍，层数遍历也是可以翻转这棵树的，因为层序遍历也可以把每个节点的左右孩子都翻转一遍，代码如下：</font>

```javascript
var invertTree = function(root) {
  const inverNode = function(root, left, right) {
    root.left = right;
    root.right = left;
  }
  if (root === null) return root;
  const queue = [];
  queue.push(root);
  while (queue.length) {
    let len = queue.length;
    while (len--) {
     const node = queue.shift();
      inverNode(node, node.left, node.right);
      
      node.left && queue.push(node.left);
      node.right && queue.push(node.right);
    }
  }
  return root;
}
```
