### 一、[235. 二叉搜索树的最近公共祖先](https://leetcode.cn/problems/lowest-common-ancestor-of-a-binary-search-tree/)
<font style="color:rgb(44, 62, 80);">给定一个二叉搜索树, 找到该树中两个指定节点的最近公共祖先。</font>

<font style="color:rgb(44, 62, 80);">百度百科中最近公共祖先的定义为：“对于有根树 T 的两个结点 p、q，最近公共祖先表示为一个结点 x，满足 x 是 p、q 的祖先且 x 的深度尽可能大（一个节点也可以是它自己的祖先）。”</font>

<font style="color:rgb(44, 62, 80);">例如，给定如下二叉搜索树:  root = [6,2,8,0,4,7,9,null,null,3,5]</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1655440010356-e8db8e04-e0d4-4062-a725-3df3380de8e1.png)

<font style="color:rgb(44, 62, 80);">示例 1:</font>

+ <font style="color:rgb(44, 62, 80);">输入: root = [6,2,8,0,4,7,9,null,null,3,5], p = 2, q = 8</font>
+ <font style="color:rgb(44, 62, 80);">输出: 6</font>
+ <font style="color:rgb(44, 62, 80);">解释: 节点 2 和节点 8 的最近公共祖先是 6。</font>

<font style="color:rgb(44, 62, 80);">示例 2:</font>

+ <font style="color:rgb(44, 62, 80);">输入: root = [6,2,8,0,4,7,9,null,null,3,5], p = 2, q = 4</font>
+ <font style="color:rgb(44, 62, 80);">输出: 2</font>
+ <font style="color:rgb(44, 62, 80);">解释: 节点 2 和节点 4 的最近公共祖先是 2, 因为根据定义最近公共祖先节点可以为节点本身。</font>

<font style="color:rgb(44, 62, 80);">说明:</font>

+ <font style="color:rgb(44, 62, 80);">所有节点的值都是唯一的。</font>
+ <font style="color:rgb(44, 62, 80);">p、q 为不同节点且均存在于给定的二叉搜索树中。</font>  


#### 思路
在[27-二叉树的最近公共祖先](https://www.yuque.com/u25370234/kb/dp6zok)中， 是通过<font style="color:rgb(44, 62, 80);">回溯从底向上搜索，遇到一个节点的左子树里有p，右子树里有q，那么当前节点就是最近公共祖先。</font>

<font style="color:rgb(44, 62, 80);">那么本题是二叉搜索树，二叉搜索树是有序的，那得好好利用一下这个特点。在有序树里，如果判断一个节点的左子树里有p，右子树里有q呢？</font>

<font style="color:rgb(44, 62, 80);">其实只要从上到下遍历的时候，cur节点是数值在[p, q]区间中则说明该节点cur就是最近公共祖先了。</font>

<font style="color:rgb(44, 62, 80);">理解这一点，本题就很好解了。</font>

<font style="color:rgb(44, 62, 80);">和</font>[27-二叉树的最近公共祖先](https://www.yuque.com/u25370234/kb/dp6zok)<font style="color:rgb(44, 62, 80);">不同，普通二叉树求最近公共祖先需要使用回溯，从底向上来查找，二叉搜索树就不用了，因为搜索树有序（相当于自带方向），那么只要从上向下遍历就可以了。</font>

<font style="color:rgb(44, 62, 80);">那么我们可以采用前序遍历（其实这里没有中节点的处理逻辑，遍历顺序无所谓了）。</font>

<font style="color:rgb(44, 62, 80);">如图所示：p为节点3，q为节点5</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1655440627096-d7508c4c-ecd0-45a9-88a1-8a9cba9f2f96.png)

<font style="color:rgb(44, 62, 80);">可以看出直接按照指定的方向，就可以找到节点4，为最近公共祖先，而且不需要遍历整棵树，找到结果直接返回！</font>  


#### 递归
+ <font style="color:rgb(44, 62, 80);">确定递归函数返回值以及参数</font>

<font style="color:rgb(44, 62, 80);">参数就是当前节点，以及两个结点 p、q。</font>

<font style="color:rgb(44, 62, 80);">返回值是要返回最近公共祖先，所以是TreeNode。</font>

```javascript
function traversal(node, p, q) {}
```



+ <font style="color:rgb(44, 62, 80);">确定终止条件</font>

<font style="color:rgb(44, 62, 80);">遇到空返回就可以了，代码如下：</font>

```javascript
if (node === null) return node;
```

<font style="color:rgb(44, 62, 80);">其实都不需要这个终止条件，因为题目中说了p、q 为不同节点且均存在于给定的二叉搜索树中。也就是说一定会找到公共祖先的，所以并不存在遇到空的情况。</font>

<font style="color:rgb(44, 62, 80);"></font>

+ <font style="color:rgb(44, 62, 80);">确定单层递归的逻辑</font>

<font style="color:rgb(44, 62, 80);">在遍历二叉搜索树的时候就是寻找区间[p.val, q.val]（注意这里是左闭右闭）</font>

<font style="color:rgb(44, 62, 80);">那么如果 cur.val 大于 p.val，同时 cur.val 大于q.val，那么就应该向左遍历（说明目标区间在左子树上）。</font>

**<font style="color:rgb(44, 62, 80);">需要注意的是此时不知道p和q谁大，所以两个都要判断</font>**

```javascript
if (node.val > p.val && node.val > q.val) {
    const left = traversal(node.left, p, q);
    return left !== null && left;
}

if (node.val < p.val && node.val < q.val) {
    const right = traversal(node.right, p, q);
    return right !== null && right;
}
```



**整体代码**

```javascript
var lowestCommonAncestor = function(root, p, q) {
    // 使用递归的方法
    // 1. 使用给定的递归函数lowestCommonAncestor
    // 2. 确定递归终止条件
    if(root === null) {
        return root;
    }
    if(root.val > p.val && root.val > q.val) {
        // 向左子树查询
        let left = lowestCommonAncestor(root.left, p, q);
        return left !== null && left;
    }
    if(root.val < p.val && root.val < q.val) {
        // 向右子树查询
        let right = lowestCommonAncestor(root.right, p, q);
        return right !== null && right;
    }
    return root;
};
```



#### 迭代
```javascript
var lowestCommonAncestor = function(root, p, q) {
    while(root) {
        if(root.val>p.val&&root.val>q.val) {
            root = root.left;
        }else if(root.val<p.val&&root.val<q.val) {
            root = root.right;
        }else {
            return root;
        }
        
    }
    return null;
};
```

### 总结
<font style="color:rgb(44, 62, 80);">对于二叉搜索树的最近祖先问题，其实要比</font>[27-二叉树的最近公共祖先](https://www.yuque.com/u25370234/kb/dp6zok)<font style="color:rgb(44, 62, 80);">简单的多。</font>

<font style="color:rgb(44, 62, 80);">不用使用回溯，二叉搜索树自带方向性，可以方便的从上向下查找目标区间，遇到目标区间内的节点，直接返回。</font>

<font style="color:rgb(44, 62, 80);">最后给出了对应的迭代法，二叉搜索树的迭代法甚至比递归更容易理解，也是因为其有序性（自带方向性），按照目标区间找就行了。</font>
