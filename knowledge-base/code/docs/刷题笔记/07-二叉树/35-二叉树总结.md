### <font style="color:rgb(44, 62, 80);">三、求二叉树的属性</font>
+ [08-对称二叉树](https://www.yuque.com/u25370234/kb/yqhatu)
    - <font style="color:rgb(44, 62, 80);">递归：后序，比较的是根节点的左子树与右子树是不是相互翻转</font>
    - <font style="color:rgb(44, 62, 80);">迭代：使用队列/栈将两个节点顺序放入容器中进行比较</font>
+ [09-二叉树的最大深度](https://www.yuque.com/u25370234/kb/gg332k)
    - <font style="color:rgb(44, 62, 80);">递归：后序，求根节点最大高度就是最大深度，通过递归函数的返回值做计算树的高度</font>
    - <font style="color:rgb(44, 62, 80);">迭代：层序遍历</font>
+ [10-二叉树的最小深度](https://www.yuque.com/u25370234/kb/gg332k)
    - <font style="color:rgb(44, 62, 80);">递归：后序，求根节点最小高度就是最小深度，注意最小深度的定义</font>
    - <font style="color:rgb(44, 62, 80);">迭代：层序遍历</font>
+ [11-完全二叉树的节点个数](https://www.yuque.com/u25370234/kb/wioqgh)
    - <font style="color:rgb(44, 62, 80);">递归：后序，通过递归函数的返回值计算节点数量</font>
    - <font style="color:rgb(44, 62, 80);">迭代：层序遍历</font>
+ [12-平衡二叉树](https://www.yuque.com/u25370234/kb/ngoxga)
    - <font style="color:rgb(44, 62, 80);">递归：后序，注意后序求高度和前序求深度，递归过程判断高度差</font>
    - <font style="color:rgb(44, 62, 80);">迭代：效率很低，不推荐</font>
+ [13-二叉树的所有路径](https://www.yuque.com/u25370234/kb/eu6ut1)
    - <font style="color:rgb(44, 62, 80);">递归：前序，方便让父节点指向子节点，涉及回溯处理根节点到叶子的所有路径</font>
    - <font style="color:rgb(44, 62, 80);">迭代：一个栈模拟递归，一个栈来存放对应的遍历路径</font>
+ [15-二叉树中递归带着回溯](https://www.yuque.com/u25370234/kb/pxesvw)
    - <font style="color:rgb(44, 62, 80);">详解</font>[13-二叉树的所有路径](https://www.yuque.com/u25370234/kb/eu6ut1)<font style="color:rgb(44, 62, 80);">中递归如何隐藏着回溯</font>
+ [16-左叶子之和](https://www.yuque.com/u25370234/kb/wy9tgo)
    - <font style="color:rgb(44, 62, 80);">递归：后序，必须三层约束条件，才能判断是否是左叶子。</font>
    - <font style="color:rgb(44, 62, 80);">迭代：直接模拟后序遍历</font>
+ [17-找树左下角的值](https://www.yuque.com/u25370234/kb/axldgk)
    - <font style="color:rgb(44, 62, 80);">递归：顺序无所谓，优先左孩子搜索，同时找深度最大的叶子节点。</font>
    - <font style="color:rgb(44, 62, 80);">迭代：层序遍历找最后一行最左边</font>
+ [18-路径总和](https://www.yuque.com/u25370234/kb/xgm356)
    - <font style="color:rgb(44, 62, 80);">递归：顺序无所谓，递归函数返回值为bool类型是为了搜索一条边，没有返回值是搜索整棵树。</font>
    - <font style="color:rgb(44, 62, 80);">迭代：栈里元素不仅要记录节点指针，还要记录从头结点到该节点的路径数值总和</font>
