#### 思路
#### 1、递归
<font style="color:rgb(44, 62, 80);">这道题目用递归的话就就一直向左遍历，最后一个就是答案呗？没有这么简单，一直向左遍历到最后一个，它未必是最后一行啊。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">我们来分析一下题目：在树的</font>**<font style="color:rgb(44, 62, 80);">最后一行</font>**<font style="color:rgb(44, 62, 80);">找到</font>**<font style="color:rgb(44, 62, 80);">最左边的值</font>**<font style="color:rgb(44, 62, 80);">。</font>

<font style="color:rgb(44, 62, 80);">首先要是最后一行，然后是最左边的值。</font>

<font style="color:rgb(44, 62, 80);">如果使用递归法，如何判断是最后一行呢，其实就是深度最大的叶子节点一定是最后一行。</font>

  
	<font style="color:rgb(44, 62, 80);">所以要找深度最大的叶子节点。</font>

<font style="color:rgb(44, 62, 80);">那么如果找最左边的呢？可以使用前序遍历，这样才先优先左边搜索，然后记录深度最大的叶子节点，此时就是树的最后一行最左边的值。</font>

<font style="color:rgb(44, 62, 80);"></font>

**递归三部曲**

<font style="color:rgb(44, 62, 80);">1.确定递归函数的参数和返回值</font>

<font style="color:rgb(44, 62, 80);">参数必须有要遍历的树的根节点，还需要一个变量用来记录最长深度。 这里就不需要返回值了，所以递归函数的返回类型为</font>`<font style="color:rgb(44, 62, 80);">void</font>`<font style="color:rgb(44, 62, 80);">。</font>

<font style="color:rgb(44, 62, 80);">本题还需要类里的两个全局变量，</font>`<font style="color:rgb(44, 62, 80);">maxLen</font>`<font style="color:rgb(44, 62, 80);">用来记录最大深度，</font>`<font style="color:rgb(44, 62, 80);">maxleftValue</font>`<font style="color:rgb(44, 62, 80);">记录最大深度最左节点的数值。</font>

代码如下:

```javascript
let maxLen; // 全局变量 记录最大深度
let maxLeftValue; // 全局变量 最大深度最左节点的数组
function traversal(root, leftLen) {}
```



<font style="color:rgb(44, 62, 80);">2.确定终止条件</font>

<font style="color:rgb(44, 62, 80);">当遇到叶子节点的时候，就需要统计一下最大的深度了，所以需要遇到叶子节点来更新最大深度。</font>

代码如下:

```javascript
if (root.left === null && root.right === null) {
  if (leftLen > maxLen) {
    maxLen = leftLen; // 更新最大深度
    maxLeftValue = root.val; // 更新最大深度最左节点的数值
  }
  return;
}
```



3.确定单层递归的逻辑

<font style="color:rgb(44, 62, 80);">在找最大深度的时候，递归的过程中依然要使用回溯，代码如下：</font>

```javascript
if (root.left) {   // 左
    leftLen++; // 深度加一
```

##### 完整代码
```javascript
/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {TreeNode} root
 * @return {number}
 */
var findBottomLeftValue = function(root) {
    let maxLen = 0,
        maxLeftValue;
    
    function traversal(root, leftLen) {
        if (root.left === null && root.right === null) {
            if (leftLen > maxLen) {
                maxLen = leftLen;
                maxLeftValue = root.val;
            }
            return;
        }
        root.left && traversal(root.left, leftLen + 1);
        root.right && traversal(root.right, leftLen + 1);
        return;
    }
    traversal(root, 1);

    return maxLeftValue;
};
```





#### 2、层序
<font style="color:rgb(44, 62, 80);">本题使用层序遍历再合适不过了，比递归要好理解的多！</font>

<font style="color:rgb(44, 62, 80);">只需要记录最后一行第一个节点的数值就可以了。</font>

```javascript
/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {TreeNode} root
 * @return {number}
 */
var findBottomLeftValue = function(root) {
    const queue = [root];
    let res = 0;
    
    while (queue.length) {
        res = queue[0].val;

        let len = queue.length;
        while (len--) {
            const node = queue.shift();
            node.left && queue.push(node.left);
            node.right && queue.push(node.right);
        }
    }

    return res;
};
```
