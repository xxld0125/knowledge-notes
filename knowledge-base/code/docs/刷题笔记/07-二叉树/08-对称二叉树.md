### 一、例题-[101. 对称二叉树](https://leetcode.cn/problems/symmetric-tree/)
<font style="color:rgb(44, 62, 80);">给定一个二叉树，检查它是否是镜像对称的。</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1652956503250-0d8bda54-99ef-44f8-8dc4-2edf46384645.png)



#### 思路:
对于二叉树是否对称, 要比较的是根节点的左子树与右子树是不是相互翻转的<font style="color:rgb(44, 62, 80);">, 即要比较的是两个树(这两个树是根节点的左右子树), 所以在递归遍历的过程,也是要同时遍历两棵树。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">那么如何比较呢? 我们可以通过比较两个子树的里侧和外侧的元素是否相等。</font>

<font style="color:rgb(44, 62, 80);"></font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1652956854055-8b11da99-d2a6-4244-a700-016c807a21ee.png)



遍历顺序的话, 本题只能是“后序遍历”, <font style="color:rgb(44, 62, 80);">因为我们要通过递归函数的返回值来判断两个子树的内侧节点和外侧节点是否相等。</font>

<font style="color:rgb(44, 62, 80);"></font>

**<font style="color:rgb(44, 62, 80);">正是因为要遍历两棵树而且要比较内侧和外侧节点，所以准确的来说是一个树的遍历顺序是左右中，一个树的遍历顺序是右左中。</font>**

**<font style="color:rgb(44, 62, 80);"></font>**

#### 递归法
1、确定递归函数的参数和返回值

<font style="color:rgb(44, 62, 80);">因为我们要比较的是根节点的两个子树是否是相互翻转的，进而判断这个树是不是对称树，所以要比较的是两个树，参数自然也是左子树节点和右子树节点。</font>

<font style="color:rgb(44, 62, 80);">返回值自然是boolean类型。</font>



2、确定终止条件

<font style="color:rgb(44, 62, 80);">要比较两个节点数值相不相同，首先要把两个节点为空的情况弄清楚！否则后面比较数值的时候就会操作空指针了。</font>

<font style="color:rgb(44, 62, 80);">节点为空的情况有：</font>

+ <font style="color:rgb(44, 62, 80);">左节点为空，右节点不为空，不对称，return false</font>
+ <font style="color:rgb(44, 62, 80);">左不为空，右为空，不对称 return false</font>
+ <font style="color:rgb(44, 62, 80);">左右都为空，对称，返回true</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">此时已经排除掉了节点为空的情况，那么剩下的就是左右节点不为空：</font>

+ <font style="color:rgb(44, 62, 80);">左右都不为空，比较节点数值，不相同就return false</font>

<font style="color:rgb(44, 62, 80);">此时左右节点不为空，且数值也不相同的情况我们也处理了。</font>

```javascript
if (left === null && right !== null) return false;
else if (left !== null && right === null) return false;
else if (left === null && right === null) return true;
else if (left.val !== right.val) return false;
```

<font style="color:rgb(44, 62, 80);">把以上情况都排除之后，剩下的就是 左右节点都不为空，且数值相同的情况。</font>



3、确定单层递归的逻辑

<font style="color:rgb(44, 62, 80);">此时才进入单层递归的逻辑，单层递归的逻辑就是处理 左右节点都不为空，且数值相同的情况。</font>

<font style="color:rgb(44, 62, 80);"></font>

+ <font style="color:rgb(44, 62, 80);">比较二叉树外侧是否对称：传入的是左节点的左孩子，右节点的右孩子。</font>
+ <font style="color:rgb(44, 62, 80);">比较内测是否对称，传入左节点的右孩子，右节点的左孩子。</font>
+ <font style="color:rgb(44, 62, 80);">如果左右都对称就返回true ，有一侧不对称就返回false 。</font>



```javascript
let outSide = compareNode(left.left, right.right);
let inSide = compareNode(left.right, right.left);
return outSide && inSide;
```

<font style="color:rgb(44, 62, 80);">如上代码中，我们可以看出使用的遍历方式，左子树左右中，右子树右左中，所以我把这个遍历顺序也称之为“后序遍历”（尽管不是严格的后序遍历）。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">递归法代码:</font>

```javascript
var isSymmetric = function(root) {
  // 使用递归遍历左右子树 递归三部曲
  // 1. 确认递归参数root.left root.right 和 返回值 true false
  const compareNode = function(left, right) {
    // 确定终止条件 空的情况
    if (left === null && right !== null || left !== null && right === null) {
      return false;
    } else if (left === null && right === null) {
      return true;
    } else if (left.val !== right.val) {
       return false;
    }
    // 3.确定单层递归逻辑
    let outSide = compareNode(left.left, right.right);
    let inSide = compareNode(left.right, right.left);
    return inSide && outSide;
  }
  if (root === null) return true;
  return compareNode(root.left, root.right);
}
```





#### 迭代法
<font style="color:rgb(44, 62, 80);">这道题目我们也可以使用迭代法，但要注意，这里的迭代法可不是前中后序的迭代写法，因为本题的本质是判断两个树是否是相互翻转的，其实已经不是所谓二叉树遍历的前中后序的关系了。</font>



##### 使用队列
<font style="color:rgb(44, 62, 80);">通过队列来判断根节点的左子树和右子树的内侧和外侧是否相等，如动画所示：</font>

![](https://cdn.nlark.com/yuque/0/2022/gif/25743026/1652959189450-98b665b8-27ce-480b-b32a-ab69ed35d1a7.gif)

```javascript
var isSymmetric = function(root) {
  // 迭代方法判断是否是对称树
  // 首先判断root是否为空
  if (root === null) return true;
  let queue = [];
  queue.push(root.left);
  queue.push(root.right);
  while (queue.length) {
    let leftNode = queue.shift();// 左节点
    let rightNode = queue.shift();// 右节点
    if (leftNode === null && rightNode === null) {
      continue;
    }
    if (leftNode === null || rightNode === null || leftNode.val !== rightNode.val) {
      return false;
    }
    queue.push(leftNode.left);
    queue.push(rightNode.right);
    queue.push(leftNode.right);
    queue.push(rightNode.left);
  }
  return true;
}
```



##### 使用栈
<font style="color:rgb(44, 62, 80);">细心的话，其实可以发现，这个迭代法，其实是把左右两个子树要比较的元素顺序放进一个容器，然后成对成对的取出来进行比较，那么其实使用栈也是可以的。</font>

<font style="color:rgb(44, 62, 80);">只要把队列原封不动的改成栈就可以了。</font>

```javascript
var isSymmetric = function(root) {
  if (root === null) return true;
  let stack = [];
  stack.push(root.left, root.right);
  while (stack.length) {
    let rightNode = stack.pop();
    let leftNode = stack.pop();
    if (leftNode === null && rightNode === null) {
      continue;
    }
    if (leftNode === null || rightNode === null) || leftNode.val !== rightNode.val) {
      return false;
    }
    stack.push(leftNode.left);
    stack.push(rightNode.right);
    stack.push(leftNode.right);
    stack.push(rightNode.left);
  }
  return true;
}
```

#### [100. 相同的树](https://leetcode.cn/problems/same-tree/)
##### 一、递归法


```javascript
/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {TreeNode} p
 * @param {TreeNode} q
 * @return {boolean}
 */
var isSameTree = function(p, q) {
    // 递归法
    if (p === null && q === null) return true;
    else if (p === null || q === null || p.val !== q.val) return false;
    return isSameTree(p.left, q.left) && isSameTree(p.right, q.right);
};
```

#### [572. 另一棵树的子树](https://leetcode.cn/problems/subtree-of-another-tree/)
##### 一、递归法
```javascript
/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {TreeNode} root
 * @param {TreeNode} subRoot
 * @return {boolean}
 */
var isSubtree = function(root, subRoot) {

    const compareNode = function(left, right) {
        if (left !== null && right === null || left === null && right !== null) {
            return false;
        } else if (left === null && right === null) {
            return true;
        } else if (left.val !== right.val) {
            return false;
        }

        return compareNode(left.left, right.left) && compareNode(left.right, right.right);

    }

    if (root === null) return false;

    if (compareNode(root, subRoot)) return true;

    return isSubtree(root.left, subRoot) || isSubtree(root.right, subRoot);
};
```

##### 二、迭代法
```javascript
/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {TreeNode} root
 * @param {TreeNode} subRoot
 * @return {boolean}
 */
var isSubtree = function(root, subRoot) {
    const compareNode = (left, right) => {
        const queue = [];
        queue.push(left);
        queue.push(right);
        while (queue.length) {
            const leftNode = queue.shift();
            const rightNode = queue.shift();
            if (leftNode === null && rightNode === null) {
                continue;
            } else if (leftNode === null || rightNode === null || leftNode.val !== rightNode.val) {
                return false;
            }
            queue.push(leftNode.left);
            queue.push(rightNode.left);
            queue.push(leftNode.right);
            queue.push(rightNode.right);
        }
        return true;
    }

    const queue = [];
    queue.push(root);

    while (queue.length) {
        const node = queue.shift();
        if (compareNode(node, subRoot)) return true;
        if (node !== null) {
            queue.push(node.left);
            queue.push(node.right);
        }
    }
    return false;
};
```



### 三、总结
<font style="color:rgb(44, 62, 80);">我们介绍了递归法和迭代法，递归依然通过递归三部曲来解决了这道题目，如果只看精简的代码根本看不出来递归三部曲是如果解题的。</font>

<font style="color:rgb(44, 62, 80);">在迭代法中我们使用了队列，需要注意的是这不是层序遍历，而且仅仅通过一个容器来成对的存放我们要比较的元素，知道这一本质之后就发现，用队列，用栈，甚至用数组，都是可以的。</font>
