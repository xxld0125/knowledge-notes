<font style="color:rgb(44, 62, 80);">这就重新审题了，题目中说的是：</font>**<font style="color:rgb(44, 62, 80);">最小深度是从根节点到最近叶子节点的最短路径上的节点数量。</font>**<font style="color:rgb(44, 62, 80);">，注意是</font>**<font style="color:rgb(44, 62, 80);">叶子节点</font>**<font style="color:rgb(44, 62, 80);">。</font>

#### 递归法
1、确定递归函数的参数和返回值

<font style="color:rgb(44, 62, 80);">参数为要传入的二叉树根节点，返回的是深度值。</font>

```javascript
getDepth(node){}
```

<font style="color:rgb(44, 62, 80);"></font>

2、确定终止条件

终止条件也是遇到空节点返回0, 表示当前节点的高度为0<font style="color:rgb(44, 62, 80);">。</font>

```javascript
if (node === null) return 
```

3、确定单层递归逻辑

<font style="color:rgb(44, 62, 80);">如果左子树为空，右子树不为空，说明最小深度是 1 + 右子树的深度。</font>

<font style="color:rgb(44, 62, 80);">反之，右子树为空，左子树不为空，最小深度是 1 + 左子树的深度。 最后如果左右子树都不为空，返回左右子树深度最小值 + 1 。</font>

#### 迭代法
<font style="color:rgb(44, 62, 80);">只有当左右孩子都为空的时候，才说明遍历的最低点了。如果其中一个孩子为空则不是最低点</font>**<font style="color:rgb(44, 62, 80);">。</font>**

```javascript
/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {TreeNode} root
 * @return {number}
 */
var minDepth = function(root) {
    let res = 0;
    if (root === null) return res;
    const queue = [];
    queue.push(root);

    while (queue.length) {
        let len = queue.length;
        res++;
        while (len--) {
            const node = queue.shift();
            if (node.left === null && node.right === null) return res;
            node.left && queue.push(node.left);
            node.right && queue.push(node.right);
        }
    }

    return res
};
```
