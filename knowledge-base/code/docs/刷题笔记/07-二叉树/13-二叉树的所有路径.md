### 一、例题-[257. 二叉树的所有路径](https://leetcode.cn/problems/binary-tree-paths/)
<font style="color:rgb(44, 62, 80);">给定一个二叉树，返回所有从根节点到叶子节点的路径。</font>

<font style="color:rgb(44, 62, 80);">说明: 叶子节点是指没有子节点的节点。</font>

<font style="color:rgb(44, 62, 80);">示例:</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1653401125400-53bb5d8b-a408-4373-be12-c23b785abc41.png)



#### 思路
题目要求从跟节点到叶子的路径, 所以需要使用前序遍历, 这样才方便让父节点指向孩子节点, 找到对应的路径<font style="color:rgb(44, 62, 80);">。</font>

<font style="color:rgb(44, 62, 80);">在这道题目中将第一次涉及到回溯，因为我们要把路径记录下来，需要回溯来回退一一个路径在进入另一个路径。</font>

<font style="color:rgb(44, 62, 80);">前序遍历以及回溯的过程如图：</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1653401240094-c705b46b-bada-4faa-933f-3fc993bfbb49.png)

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">我们先使用递归的方式，来做前序遍历。</font>**<font style="color:rgb(44, 62, 80);">要知道递归和回溯就是一家的，本题也需要回溯。</font>**

**<font style="color:rgb(44, 62, 80);"></font>**

#### 递归
1.确定递归函数的参数和返回值

传入根节点, 记录每一条路径的path, 和存放结果集的result, 不需要返回值**<font style="color:rgb(44, 62, 80);">。</font>**

```javascript
function getPath(node, curPath) {}
```



2.确定递归终止条件

之前在写递归的终止条件都这么写

```javascript
if (node === null){
  // 终止处理逻辑
}
```

	但是本题的终止条件这样写会很麻烦, 因为本题要找到叶子节点, 就开始结束的处理逻辑了(把路径放进result中)**<font style="color:rgb(44, 62, 80);">。</font>**

**<font style="color:rgb(44, 62, 80);"></font>**

**<font style="color:rgb(44, 62, 80);">那么什么时候算是找到叶子节点? </font>**<font style="color:rgb(44, 62, 80);">当cur不为空, 其左右孩子都为空时, 就找到了叶子节点</font>**<font style="color:rgb(44, 62, 80);">。</font>**

**<font style="color:rgb(44, 62, 80);"></font>**

<font style="color:rgb(44, 62, 80);">所以本题的终止条件是:</font>

```javascript
if (node.left === null && node.right === null) {
  // 终止处理逻辑
}
```

	

3.确定单层递归逻辑

因为是前序遍历, 需要先处理中间节点, 中间节点就是我们要记录路径上的节点, 先放进`path`中<font style="color:rgb(44, 62, 80);">。</font>

`<font style="color:rgb(44, 62, 80);">path.push(cur.val)</font>`

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">然后是递归和回溯的过程, 上面说过没有判断</font>`<font style="color:rgb(44, 62, 80);">cur</font>`<font style="color:rgb(44, 62, 80);">是否为空, 那么在这里递归的时候, 如果为空就不进行下一层的递归。</font>

<font style="color:rgb(44, 62, 80);">所以递归前需要加上判断条件, 下面要递归的节点是否为空:</font>

```javascript
  node.left && getPath(node.left, curPath);
  node.right && getPath(node.right, curPath);
```

<font style="color:rgb(44, 62, 80);"></font>



整体代码如下:

```javascript
var binaryTreePaths = function(root) {
  // 递归遍历 + 递归三部曲
  let res = [];
  // 1.确定递归函数 参数 返回值
  const getPath = function(node, curPath) {
    // 2.确定终止条件, 到叶子节点终止		
    if (node.left === null && node.right === null) {
      curPath += node.val;
      res.push(curPath);
      return ;
    }
    // 3.确定单层递归逻辑
    curPath += node.val + '->';
    node.left && getPath(node.left, curPath);
    node.right && getPath(node.right, curPath);
  }
  getPath(root, '');
  return res;
}
```





#### 迭代
```javascript
var binaryTreePaths = function(root) {
  if (!root) return [];
  const stack = [root], paths = [''], res = [];
  
  while (stack.length) {
    const node = stack.pop();
    let path = paths.pop();
    if (!node.left && !node.right) { // 到叶子节点终止, 添加路径到结果中
      res.push(path + node.val);
      continue;
    }
    
    path += node.val + '->';
    if (node.right) { // 右节点存在
      stack.push(node.right);
      paths.push(path)
    }
    
    if (node.left) { // 左节点存在
      stack.push(node.right);
      paths.push(path);
    }
  }
  return res;
}
```
