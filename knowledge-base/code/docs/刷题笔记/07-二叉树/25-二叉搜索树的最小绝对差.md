### 一、[530. 二叉搜索树的最小绝对差](https://leetcode.cn/problems/minimum-absolute-difference-in-bst/)
<font style="color:rgb(44, 62, 80);">给你一棵所有节点为非负值的二叉搜索树，请你计算树中任意两节点的差的绝对值的最小值。</font>

<font style="color:rgb(44, 62, 80);">示例：</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1655181113011-4d6aa44c-e765-4c90-aa08-986488e994f9.png)

<font style="color:rgb(44, 62, 80);">提示：树中至少有 2 个节点。</font>

<font style="color:rgb(44, 62, 80);"></font>

#### 思路
<font style="color:rgb(44, 62, 80);">题目中要求在二叉搜索树上任意两节点的差的绝对值的最小值。</font>

**<font style="color:rgb(44, 62, 80);">注意是二叉搜索树</font>**<font style="color:rgb(44, 62, 80);">，二叉搜索树可是有序的。</font>

<font style="color:rgb(44, 62, 80);">遇到在二叉搜索树上求什么最值啊，差值之类的，就把它想成在一个有序数组上求最值，求差值，这样就简单多了。</font>

<font style="color:rgb(44, 62, 80);"></font>

#### 递归
<font style="color:rgb(44, 62, 80);">那么二叉搜索树采用中序遍历，其实就是一个有序数组。</font>

**<font style="color:rgb(44, 62, 80);">在一个有序数组上求两个数最小差值，这是不是就是一道送分题了。</font>**

<font style="color:rgb(44, 62, 80);">最直观的想法，就是把二叉搜索树转换成有序数组，然后遍历一遍数组，就统计出来最小差值了。</font>

<font style="color:rgb(44, 62, 80);"></font>

```javascript
/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {TreeNode} root
 * @return {number}
 */
var getMinimumDifference = function(root) {
    const res = [];
    const dfs = node => {
        node.left && dfs(node.left);

        res.push(node.val);

        node.right && dfs(node.right);
    }

    dfs(root);

    let min = Math.pow(10, 5) + 1;

    for (let i = 1; i < res.length;i++) {
        min = Math.min(min, res[i] - res[i - 1]);
    }

    return min;
};
```



以上代码是把二叉搜索树转化为有序数组了，其实在二叉搜素树中序遍历的过程中，我们就可以直接计算了。  需要用一个`pre`节点记录一下`cur`节点的前一个节点。

```javascript
/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {TreeNode} root
 * @return {number}
 */
var getMinimumDifference = function(root) {
    let pre = null;
    let min = Infinity;

    const dfs = node => {
        node.left && dfs(node.left);

        if (pre !== null) {
            min = Math.min(min, node.val - pre.val)
        }

        pre = node;

        node.right && dfs(node.right);
    }

    dfs(root);

    return min;
};
```



#### 迭代
```javascript
/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {TreeNode} root
 * @return {number}
 */
var getMinimumDifference = function(root) {
    let min = Infinity;
    let pre = null;

    const stack = [root];

    while (stack.length) {
        let node = stack.pop();

        if (node === null) {
            node = stack.pop();
            if (pre !== null) {
                min = Math.min(min, node.val - pre.val);
            }
            pre = node;
        } else {
            node.right && stack.push(node.right);
            stack.push(node);
            stack.push(null);
            node.left && stack.push(node.left);
        }
    }

    return min;
};
```



### 总结
**<font style="color:rgb(44, 62, 80);">遇到在二叉搜索树上求什么最值，求差值之类的，都要思考一下二叉搜索树可是有序的，要利用好这一特点。</font>**

<font style="color:rgb(44, 62, 80);">同时要学会在递归遍历的过程中如何记录前后两个指针，这也是一个小技巧，学会了还是很受用的。</font>
