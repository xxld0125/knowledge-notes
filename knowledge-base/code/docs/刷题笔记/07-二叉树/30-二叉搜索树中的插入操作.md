### 一、[701. 二叉搜索树中的插入操作](https://leetcode.cn/problems/insert-into-a-binary-search-tree/)
<font style="color:rgb(44, 62, 80);">给定二叉搜索树（BST）的根节点和要插入树中的值，将值插入二叉搜索树。 返回插入后二叉搜索树的根节点。 输入数据保证，新值和原始二叉搜索树中的任意节点值都不同。</font>

<font style="color:rgb(44, 62, 80);">注意，可能存在多种有效的插入方式，只要树在插入后仍保持为二叉搜索树即可。 你可以返回任意有效的结果。</font>

<font style="color:rgb(44, 62, 80);">提示：</font>

+ <font style="color:rgb(44, 62, 80);">给定的树上的节点数介于 0 和 10^4 之间</font>

#### <font style="color:rgb(44, 62, 80);">思路</font>
<font style="color:rgb(44, 62, 80);">如下演示视频中可以看出：只要按照二叉搜索树的规则去遍历，遇到空节点就插入节点就可以了。</font>

<font style="color:rgb(44, 62, 80);">例如插入元素10 ，需要找到末尾节点插入便可，一样的道理来插入元素15，插入元素0，插入元素6，</font>**<font style="color:rgb(44, 62, 80);">需要调整二叉树的结构么？ 并不需要。</font>**<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">只要遍历二叉搜索树，找到空节点 插入元素就可以了，那么这道题其实就简单了。</font>

<font style="color:rgb(44, 62, 80);">接下来就是遍历二叉搜索树的过程了。</font>

#### <font style="color:rgb(44, 62, 80);">递归</font>
+ <font style="color:rgb(44, 62, 80);">确定递归函数参数以及返回值</font>

<font style="color:rgb(44, 62, 80);">参数就是根节点指针，以及要插入元素，这里递归函数要不要有返回值呢？</font>

<font style="color:rgb(44, 62, 80);">可以有，也可以没有，但递归函数如果没有返回值的话，实现是比较麻烦的，下面也会给出其具体实现代码。</font>

**<font style="color:rgb(44, 62, 80);">有返回值的话，可以利用返回值完成新加入的节点与其父节点的赋值操作</font>**<font style="color:rgb(44, 62, 80);">。（下面会进一步解释）</font>

<font style="color:rgb(44, 62, 80);">递归函数的返回类型为节点类型</font>`<font style="color:rgb(44, 62, 80);">TreeNode</font>`<font style="color:rgb(44, 62, 80);">  。</font>

```javascript
function insertIntoBST(root, val) {};
```



+ 确定终止条件

<font style="color:rgb(44, 62, 80);">终止条件就是找到遍历的节点为null的时候，就是要插入节点的位置了，并把插入的节点返回。</font>

```javascript
if (root === null) {
  const node = new TreeNode(val);
  return node;
}
```



+ 确定单层递归逻辑

<font style="color:rgb(44, 62, 80);">此时要明确，需要遍历整棵树么？</font>

<font style="color:rgb(44, 62, 80);">别忘了这是搜索树，遍历整棵搜索树简直是对搜索树的侮辱，哈哈。</font>

<font style="color:rgb(44, 62, 80);">搜索树是有方向了，可以根据插入元素的数值，决定递归方向。</font>

```javascript
if (root.val > val) {
  root.left = insertIntoBST(root.left, val);
}
if (root.val < val) {
  root.right = insertIntoBST(root.right, val);
}
```

  
**整体代码**

```javascript
/**
 * Definition for a binary tree root.
 * function Treeroot(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {Treeroot} root
 * @param {number} val
 * @return {Treeroot}
 */
var insertIntoBST = function(root, val) {
    if (root === null) {
        let node = new TreeNode(val);
        return node;
    }
    if (root.val > val) {
        root.left = insertIntoBST(root.left, val);
    }
    if (root.val < val) {
        root.right = insertIntoBST(root.right, val);
    }
    return root;
};
```





#### 迭代
```javascript
function insertIntoBST(root, val) {
  if (root === null) {
    const node = new TreeNode(val);
    return node;
  }
  
  let cur = root;
  let parent = root; // 这个很重要，需要记录上一个节点，否则无法赋值新节点
  
  while (cur !== null) {
    parent = cur;
    if (cur.val > val) {
      cur = cur.left;
    } else {
      cur = cur.right;
    }
  }
  
  const node = new TreeNode(val);
  if (val < parent.val) {
     parent.left = node;
  } else {
     parent.right = node;
  }
  return root;
}
```



### 总结
<font style="color:rgb(44, 62, 80);">首先在二叉搜索树中的插入操作，大家不用恐惧其重构搜索树，其实根本不用重构。</font>

<font style="color:rgb(44, 62, 80);">然后在递归中，我们重点讲了如果通过递归函数的返回值完成新加入节点和其父节点的赋值操作，并强调了搜索树的有序性。</font>

<font style="color:rgb(44, 62, 80);">最后依然给出了迭代的方法，迭代的方法就需要记录当前遍历节点的父节点了，这个和没有返回值的递归函数实现的代码逻辑是一样的。</font>
