### 题目一:[142. 环形链表 II](https://leetcode-cn.com/problems/linked-list-cycle-ii/) (中等)


#### 解法一: 快慢指针
**思路:**

**1、如何判断链表有环**

<font style="color:rgb(44, 62, 80);">可以使用快慢指针法，分别定义 </font>`<font style="color:rgb(44, 62, 80);">fast</font>`<font style="color:rgb(44, 62, 80);"> 和 </font>`<font style="color:rgb(44, 62, 80);">slow</font>`<font style="color:rgb(44, 62, 80);"> 指针，从头结点出发，</font>`<font style="color:rgb(44, 62, 80);">fast</font>`<font style="color:rgb(44, 62, 80);">指针每次移动两个节点，</font>`<font style="color:rgb(44, 62, 80);">slow</font>`<font style="color:rgb(44, 62, 80);">指针每次移动一个节点，如果 </font>`<font style="color:rgb(44, 62, 80);">fast</font>`<font style="color:rgb(44, 62, 80);"> 和 </font>`<font style="color:rgb(44, 62, 80);">slow</font>`<font style="color:rgb(44, 62, 80);">指针在途中相遇 ，说明这个链表有环。</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1652189017579-29d657f6-23c2-4b7f-8b63-c3543d491935.png)



**2、如果有环,如何找到这个环的入口**

<font style="color:rgb(44, 62, 80);">假设从头结点到环形入口节点 的节点数为x。 环形入口节点到 fast指针与slow指针相遇节点 节点数为y。 从相遇节点 再到环形入口节点节点数为 z。 如图所示：</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1652189091009-3089283b-c8b5-40c0-b237-b3ecf1282b30.png)

<font style="color:rgb(44, 62, 80);">那么相遇时： </font>`<font style="color:rgb(44, 62, 80);">slow</font>`<font style="color:rgb(44, 62, 80);">指针走过的节点数为: </font>`<font style="color:rgb(71, 101, 130);">x + y</font>`<font style="color:rgb(44, 62, 80);">， </font>`<font style="color:rgb(44, 62, 80);">fast</font>`<font style="color:rgb(44, 62, 80);">指针走过的节点数：</font>`<font style="color:rgb(71, 101, 130);">x + y + n (y + z)</font>`<font style="color:rgb(44, 62, 80);">，</font>`<font style="color:rgb(44, 62, 80);">n</font>`<font style="color:rgb(44, 62, 80);">为</font>`<font style="color:rgb(44, 62, 80);">fast</font>`<font style="color:rgb(44, 62, 80);">指针在环内走了</font>`<font style="color:rgb(44, 62, 80);">n</font>`<font style="color:rgb(44, 62, 80);">圈才遇到</font>`<font style="color:rgb(44, 62, 80);">slow</font>`<font style="color:rgb(44, 62, 80);">指针， </font>`<font style="color:rgb(44, 62, 80);">（y+z）</font>`<font style="color:rgb(44, 62, 80);">为 一圈内节点的个数</font>`<font style="color:rgb(44, 62, 80);">A</font>`<font style="color:rgb(44, 62, 80);">。</font>

<font style="color:rgb(44, 62, 80);">因为</font>`<font style="color:rgb(44, 62, 80);">fast</font>`<font style="color:rgb(44, 62, 80);">指针是一步走两个节点，</font>`<font style="color:rgb(44, 62, 80);">slow</font>`<font style="color:rgb(44, 62, 80);">指针一步走一个节点， 所以 </font>`<font style="color:rgb(44, 62, 80);">fast</font>`<font style="color:rgb(44, 62, 80);">指针走过的节点数 = </font>`<font style="color:rgb(44, 62, 80);">slow</font>`<font style="color:rgb(44, 62, 80);">指针走过的节点数 * 2：</font>

`<font style="color:rgb(71, 101, 130);">(x + y) * 2 = x + y + n (y + z)</font>`

<font style="color:rgb(44, 62, 80);">两边消掉一个</font>`<font style="color:rgb(44, 62, 80);">（x+y）</font>`<font style="color:rgb(44, 62, 80);">: </font>`<font style="color:rgb(71, 101, 130);">x + y = n (y + z)</font>`

<font style="color:rgb(44, 62, 80);">因为要找环形的入口，那么要求的是</font>`<font style="color:rgb(44, 62, 80);">x</font>`<font style="color:rgb(44, 62, 80);">，因为</font>`<font style="color:rgb(44, 62, 80);">x</font>`<font style="color:rgb(44, 62, 80);">表示 头结点到 环形入口节点的的距离。</font>

<font style="color:rgb(44, 62, 80);">所以要求x ，将x单独放在左面：</font>`<font style="color:rgb(71, 101, 130);">x = n (y + z) - y</font>`<font style="color:rgb(44, 62, 80);"> ,</font>

<font style="color:rgb(44, 62, 80);">再从</font>`<font style="color:rgb(44, 62, 80);">n(y+z)</font>`<font style="color:rgb(44, 62, 80);">中提出一个 </font>`<font style="color:rgb(44, 62, 80);">（y+z）</font>`<font style="color:rgb(44, 62, 80);">来，整理公式之后为如下公式：</font>`<font style="color:rgb(71, 101, 130);">x = (n - 1) (y + z) + z</font>`<font style="color:rgb(44, 62, 80);"> 注意这里</font>`<font style="color:rgb(44, 62, 80);">n</font>`<font style="color:rgb(44, 62, 80);">一定是大于等于1的，因为 </font>`<font style="color:rgb(44, 62, 80);">fast</font>`<font style="color:rgb(44, 62, 80);">指针至少要多走一圈才能相遇</font>`<font style="color:rgb(44, 62, 80);">slow</font>`<font style="color:rgb(44, 62, 80);">指针。</font>

<font style="color:rgb(44, 62, 80);">这个公式说明什么呢？</font>

<font style="color:rgb(44, 62, 80);">先拿</font>`<font style="color:rgb(44, 62, 80);">n</font>`<font style="color:rgb(44, 62, 80);">为1的情况来举例，意味着</font>`<font style="color:rgb(44, 62, 80);">fast</font>`<font style="color:rgb(44, 62, 80);">指针在环形里转了一圈之后，就遇到了 </font>`<font style="color:rgb(44, 62, 80);">slow</font>`<font style="color:rgb(44, 62, 80);">指针了。</font>

<font style="color:rgb(44, 62, 80);">当 </font>`<font style="color:rgb(44, 62, 80);">n</font>`<font style="color:rgb(44, 62, 80);">为1的时候，公式就化解为 </font>`<font style="color:rgb(71, 101, 130);">x = z</font>`<font style="color:rgb(44, 62, 80);">，</font>

<font style="color:rgb(44, 62, 80);">这就意味着，</font>**<font style="color:rgb(44, 62, 80);">从头结点出发一个指针，从相遇节点 也出发一个指针，这两个指针每次只走一个节点， 那么当这两个指针相遇的时候就是 环形入口的节点</font>**<font style="color:rgb(44, 62, 80);">。</font>

<font style="color:rgb(44, 62, 80);">也就是在相遇节点处，定义一个指针</font>`<font style="color:rgb(44, 62, 80);">index1</font>`<font style="color:rgb(44, 62, 80);">，在头结点处定一个指针</font>`<font style="color:rgb(44, 62, 80);">index2</font>`<font style="color:rgb(44, 62, 80);">。</font>

<font style="color:rgb(44, 62, 80);">让</font>`<font style="color:rgb(44, 62, 80);">index1</font>`<font style="color:rgb(44, 62, 80);">和</font>`<font style="color:rgb(44, 62, 80);">index2</font>`<font style="color:rgb(44, 62, 80);">同时移动，每次移动一个节点， 那么他们相遇的地方就是 环形入口的节点。</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1652189454617-1adc2bcd-1be9-4b70-b252-b25caea6d8c7.png)



<font style="color:rgb(44, 62, 80);">那么 </font>`<font style="color:rgb(44, 62, 80);">n</font>`<font style="color:rgb(44, 62, 80);">如果大于1是什么情况呢，就是</font>`<font style="color:rgb(44, 62, 80);">fast</font>`<font style="color:rgb(44, 62, 80);">指针在环形转n圈之后才遇到 </font>`<font style="color:rgb(44, 62, 80);">slow</font>`<font style="color:rgb(44, 62, 80);">指针。</font>

<font style="color:rgb(44, 62, 80);">其实这种情况和</font>`<font style="color:rgb(44, 62, 80);">n</font>`<font style="color:rgb(44, 62, 80);">为1的时候 效果是一样的，一样可以通过这个方法找到 环形的入口节点，只不过，</font>`<font style="color:rgb(44, 62, 80);">index1</font>`<font style="color:rgb(44, 62, 80);"> 指针在环里 多转了</font>`<font style="color:rgb(44, 62, 80);">(n-1)</font>`<font style="color:rgb(44, 62, 80);">圈，然后再遇到</font>`<font style="color:rgb(44, 62, 80);">index2</font>`<font style="color:rgb(44, 62, 80);">，相遇点依然是环形的入口节点。</font>



**代码**

```javascript
/**
 * Definition for singly-linked list.
 * function ListNode(val) {
 *     this.val = val;
 *     this.next = null;
 * }
 */

/**
 * @param {ListNode} head
 * @return {ListNode}
 */
var detectCycle = function(head) {
    let slow = head,
        fast = head;
    
    while (fast && fast.next) {
        slow = slow.next;
        fast = fast.next.next;
        if (slow === fast) {
            let cur = head;
            while (cur !== slow) {
                cur = cur.next;
                slow = slow.next;
            }
            return cur;
        }
    }
    return null;

};
```

#### 解法二:
**思路: **

<font style="color:rgb(38, 38, 38);">我们遍历链表中的每个节点，并将它记录下来；一旦遇到了此前遍历过的节点，就可以判定链表中存在环.</font>

<font style="color:rgb(38, 38, 38);"></font>

<font style="color:rgb(38, 38, 38);">代码</font>

```javascript
/**
 * Definition for singly-linked list.
 * function ListNode(val) {
 *     this.val = val;
 *     this.next = null;
 * }
 */

/**
 * @param {ListNode} head
 * @return {ListNode}
 */
var detectCycle = function(head) {
    const set = new Set();
    while (head) {
        if (set.has(head)) {
            return head;
        }
        set.add(head);
        head = head.next;
    }
    return null
};
```





### 题目二: [141. 环形链表](https://leetcode-cn.com/problems/linked-list-cycle/) (简单)


#### 解法一: 
思路: 遍历所有节点, 每次遍历到一个节点时, 判断该节点之前是否已经被访问过.



代码

```javascript
/**
 * Definition for singly-linked list.
 * function ListNode(val) {
 *     this.val = val;
 *     this.next = null;
 * }
 */

/**
 * @param {ListNode} head
 * @return {boolean}
 */
var hasCycle = function(head) {
  	// 也可以使用数组、对象的数据类型;
    const set = new Set();
    while (head) {
        if (set.has(head)) {
            return true;
        }
        set.add(head);
        head = head.next;
    }
    return false;
};
```

#### 解法二: 
思路: 使用快慢指针方法, 具体思路是:我们定义两个指针，一快一满。慢指针每次只移动一步，而快指针每次移动两步。初始时，慢指针在位置 head，而快指针在位置 head.next。这样一来，如果在移动的过程中，快指针反过来追上慢指针，就说明该链表为环形链表。否则快指针将到达链表尾部，该链表不为环形链表。



代码

```javascript
/**
 * Definition for singly-linked list.
 * function ListNode(val) {
 *     this.val = val;
 *     this.next = null;
 * }
 */

/**
 * @param {ListNode} head
 * @return {boolean}
 */
var hasCycle = function(head) {
    let slow = head,
        fast = head;
    while (slow && fast && fast.next) {
        slow = slow.next;
        fast = fast.next.next;
        if (slow === fast) {
            return true;
        }
    }
    return false;
};
```

### 题目三: [61. 旋转链表](https://leetcode-cn.com/problems/rotate-list/)
#### 解法一
**思路: 将链表转化成循环链表, 根据移动距离k, 计算出循环的距离add, 找到移动后的链表, 最后将循转链表断开.**

**代码**



```javascript
/**
 * Definition for singly-linked list.
 * function ListNode(val, next) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.next = (next===undefined ? null : next)
 * }
 */
/**
 * @param {ListNode} head
 * @param {number} k
 * @return {ListNode}
 */
var rotateRight = function(head, k) {

if (k === 0 || !head || !head.next) {
        return head;
    }
    let n = 1;
    let cur = head;
    while (cur.next) {
        cur = cur.next;
        n++;
    }

    let add = n - k % n;
    if (add === n) {
        return head;
    }

    cur.next = head;
    while (add) {
        cur = cur.next;
        add--;
    }

    const ret = cur.next;
    cur.next = null;
    return ret;
};
```



### 题目四:[1019. 链表中的下一个更大节点](https://leetcode-cn.com/problems/next-greater-node-in-linked-list/)
**相似题:**[**503. 下一个更大元素 II**](https://leetcode-cn.com/problems/next-greater-element-ii/)

#### 解法一:
**思路: **写指针 w 遍历整个链表，读指针 r 找到第一个比当前 w 大的节点，并返回对应的值，如果 r 走完整个链表没找到，则返回 0;

时间复杂度O(n<sup> 2 </sup>) 

**代码:**

```javascript
/**
 * Definition for singly-linked list.
 * function ListNode(val, next) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.next = (next===undefined ? null : next)
 * }
 */
/**
 * @param {ListNode} head
 * @return {number[]}
 */
var nextLargerNodes = function(head) {
    let curr = head;
    let res = [];
    while (curr) {
        let next = curr.next;
        while (next && next.val <= curr.val) {
            next = next.next;
        }
        if (!next) {
            res.push(0);
        } else {
            res.push(next.val)
        }
        curr = curr.next;
    }
    return res;
};
```

#### 解法二:
**思路:**将还没配对成功的数放在栈内，出现一个满足条件的，就逐步找栈内数据，此时显然栈已经是一个单调递减栈

解决思路： 

栈内保存还没找到更大值的数据及index，找到更大值就set结果。

没找到就继续放在栈内，最后如果栈非空，说明栈内的数据没找到合适的数据，都置零。



**代码:**

```javascript
/**
 * Definition for singly-linked list.
 * function ListNode(val) {
 *     this.val = val;
 *     this.next = null;
 * }
 */
/**
 * @param {ListNode} head
 * @return {number[]}
 */
var nextLargerNodes = function(head) {
    let arr = [];
    // 迭代ListNode，每次取值添加到数组
    while(head){
        arr.push(head.val);
        head = head.next;
    }
    let len = arr.length, res = new Array(len).fill(0), stack=[];
    for(let i=0; i<len; i++){
        while(stack.length && arr[stack[stack.length-1]] < arr[i]){
            res[stack[stack.length-1]] = arr[i];
            stack.pop();
        }
        stack.push(i);
    }
    return res;
}
```
