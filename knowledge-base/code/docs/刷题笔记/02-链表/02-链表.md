### 一、虚拟头节点
在链表中的一个非常重要的技巧就是虚拟头节点.

<font style="color:rgb(44, 62, 80);">链表的一大问题就是操作当前节点必须要找前一个节点才能操作。这就造成了，头结点的尴尬，因为头结点没有前一个节点了。</font>

**<font style="color:rgb(44, 62, 80);">每次对应头结点的情况都要单独处理，所以使用虚拟头结点的技巧，就可以解决这个问题</font>**<font style="color:rgb(44, 62, 80);">。</font>

<font style="color:rgb(44, 62, 80);"></font>

### 二、链表的基本操作
在[02-设计链表](https://www.yuque.com/u25370234/kb/iszkgh)中,<font style="color:rgb(44, 62, 80);">我们通设计链表把链表常见的五个操作练习了一遍。</font>

<font style="color:rgb(44, 62, 80);">这是练习链表基础操作的非常好的一道题目，考察了：</font>

+ <font style="color:rgb(44, 62, 80);">获取链表第index个节点的数值</font>
+ <font style="color:rgb(44, 62, 80);">在链表的最前面插入一个节点</font>
+ <font style="color:rgb(44, 62, 80);">在链表的最后面插入一个节点</font>
+ <font style="color:rgb(44, 62, 80);">在链表第index个节点前面插入一个节点</font>
+ <font style="color:rgb(44, 62, 80);">删除链表的第index个节点的数值</font>

**<font style="color:rgb(44, 62, 80);">可以说把这道题目做了，链表基本操作就OK了，再也不用担心链表增删改查整不明白了</font>**<font style="color:rgb(44, 62, 80);">。</font>
