### 例题- [15. 三数之和](https://leetcode.cn/problems/3sum/)
<font style="color:rgb(44, 62, 80);">给你一个包含 n 个整数的数组 nums，判断 nums 中是否存在三个元素 a，b，c ，使得 a + b + c = 0 ？请你找出所有满足条件且不重复的三元组。</font>

<font style="color:rgb(44, 62, 80);"></font>

**<font style="color:rgb(44, 62, 80);">注意：</font>**<font style="color:rgb(44, 62, 80);"> </font><font style="color:rgb(44, 62, 80);">答案中不可以包含重复的三元组。</font>

<font style="color:rgb(44, 62, 80);">示例：</font>

<font style="color:rgb(44, 62, 80);">给定数组 nums = [-1, 0, 1, 2, -1, -4]，</font>

<font style="color:rgb(44, 62, 80);">满足要求的三元组集合为： [ [-1, 0, 1], [-1, -1, 2] </font>

####   
思路:
<font style="color:rgb(44, 62, 80);">两层</font>`<font style="color:rgb(44, 62, 80);">for</font>`<font style="color:rgb(44, 62, 80);">循环就可以确定 </font>`<font style="color:rgb(44, 62, 80);">a</font>`<font style="color:rgb(44, 62, 80);"> 和</font>`<font style="color:rgb(44, 62, 80);">b</font>`<font style="color:rgb(44, 62, 80);"> 的数值了，可以使用哈希法来确定 </font>`<font style="color:rgb(44, 62, 80);">0-(a+b) </font>`<font style="color:rgb(44, 62, 80);">是否在 数组里出现过，其实这个思路是正确的，但是我们有一个非常棘手的问题，就是题目中说的不可以包含重复的三元组。在去重的操作中有很多细节需要注意，在面试中很难直接写出没有bug的代码。</font>

<font style="color:rgb(44, 62, 80);"></font>

**<font style="color:rgb(44, 62, 80);">所以这道题目使用哈希法并不十分合适</font>**<font style="color:rgb(44, 62, 80);">。而且使用哈希法 在使用两层</font>`<font style="color:rgb(44, 62, 80);">for</font>`<font style="color:rgb(44, 62, 80);">循环的时候，能做的剪枝操作很有限，虽然时间复杂度是</font>`<font style="color:rgb(44, 62, 80);">$O(n^2)</font>`<font style="color:rgb(44, 62, 80);">，也是可以在leetcode上通过，但是程序的执行时间依然比较长 。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">接下来我来介绍另一个解法：</font>**<font style="color:rgb(44, 62, 80);">双指针法</font>**<font style="color:rgb(44, 62, 80);">，</font>**<font style="color:rgb(44, 62, 80);">这道题目使用双指针法 要比哈希法高效一些</font>**<font style="color:rgb(44, 62, 80);">，那么来讲解一下具体实现的思路。</font>

<font style="color:rgb(44, 62, 80);"></font>

    - <font style="color:rgb(44, 62, 80);">拿这个</font>`<font style="color:rgb(44, 62, 80);">nums</font>`<font style="color:rgb(44, 62, 80);">数组来举例，首先将数组排序，然后有一层</font>`<font style="color:rgb(44, 62, 80);">for</font>`<font style="color:rgb(44, 62, 80);">循环，</font>`<font style="color:rgb(44, 62, 80);">i</font>`<font style="color:rgb(44, 62, 80);">从下标0的地方开始，同时定一个下标</font>`<font style="color:rgb(44, 62, 80);">left</font>`<font style="color:rgb(44, 62, 80);"> 定义在</font>`<font style="color:rgb(44, 62, 80);">i+1</font>`<font style="color:rgb(44, 62, 80);">的位置上，定义下标rig</font>`<font style="color:rgb(44, 62, 80);"></font>`<font style="color:rgb(44, 62, 80);">ht 在数组结尾的位置上。</font>
    - <font style="color:rgb(44, 62, 80);">依然还是在数组中找到 </font>`<font style="color:rgb(44, 62, 80);">abc</font>`<font style="color:rgb(44, 62, 80);"> 使得</font>`<font style="color:rgb(44, 62, 80);">a + b +c =0</font>`<font style="color:rgb(44, 62, 80);">，我们这里相当于 </font>`<font style="color:rgb(44, 62, 80);">a = nums[i]</font>``<font style="color:rgb(44, 62, 80);">b = nums[left]</font>`<font style="color:rgb(44, 62, 80);"> </font>`<font style="color:rgb(44, 62, 80);">c = nums[right]</font>`<font style="color:rgb(44, 62, 80);">。</font>
    - <font style="color:rgb(44, 62, 80);">接下来如何移动</font>`<font style="color:rgb(44, 62, 80);">left</font>`<font style="color:rgb(44, 62, 80);"> 和</font>`<font style="color:rgb(44, 62, 80);">right</font>`<font style="color:rgb(44, 62, 80);">呢， 如果</font>`<font style="color:rgb(44, 62, 80);">nums[i] + nums[left] + nums[right] > 0</font>`<font style="color:rgb(44, 62, 80);"> 就说明 此时三数之和大了，因为数组是排序后了，所以</font>`<font style="color:rgb(44, 62, 80);">right</font>`<font style="color:rgb(44, 62, 80);">下标就应该向左移动，这样才能让三数之和小一些。</font>
    - <font style="color:rgb(44, 62, 80);">如果 </font>`<font style="color:rgb(44, 62, 80);">nums[i] + nums[left] + nums[right] < 0</font>`<font style="color:rgb(44, 62, 80);"> 说明 此时 三数之和小了，left 就向右移动，才能让三数之和大一些，直到</font>`<font style="color:rgb(44, 62, 80);">left</font>`<font style="color:rgb(44, 62, 80);">与</font>`<font style="color:rgb(44, 62, 80);">right</font>`<font style="color:rgb(44, 62, 80);">相遇为止。</font>
    - <font style="color:rgb(44, 62, 80);">时间复杂度：$O(n^2)。</font>

#### 
**循环内不考虑去重**

```javascript
/**
 * @param {number[]} nums
 * @return {number[][]}
 */
var threeSum = function(nums) {
    const n = nums.length;
    if (n < 3) return [];
    nums.sort((a, b) => a - b);

    const set = new Set;

    for (let i = 0;i < n - 2;i++) {
        if (nums[i] > 0) break;
        let l = i + 1, r = n - 1;
        while (l < r ) {
            const sum = nums[i] + nums[l] + nums[r];
            if (sum === 0) {
                set.add(`${nums[i]},${nums[l]},${nums[r]}`);
                l++;
                r--;
            } else if (sum < 0) {
                l++;
            } else {
                r--;
            }
        }
    }
    return Array.from(set).map(v => v.split(','));
};
```



**去重优化**

```javascript
/**
 * @param {number[]} nums
 * @return {number[][]}
 */
var threeSum = function(nums) {
    const n = nums.length;
    if (n < 3) return [];
    nums.sort((a, b) => a - b);

    const res = [];

    for (let i = 0;i < n - 2;i++) {
        if (nums[i] > 0) break;
        if (i > 0 && nums[i] === nums[i - 1]) continue;

        let l = i + 1, r = n - 1;
        while (l < r ) {
            const sum = nums[i] + nums[l] + nums[r];
            if (sum === 0) {
                res.push([nums[i], nums[l], nums[r]]);
                while (l < r && nums[l] === nums[++l]) {}
                while (l < r && nums[r] === nums[--r]) {}
            } else if (sum < 0) {
                l++;
            } else {
                r--;
            }
        }
    }
    return res;
};
```
