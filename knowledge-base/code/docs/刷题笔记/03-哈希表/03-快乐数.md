### 一、例题-[202. 快乐数](https://leetcode.cn/problems/happy-number/)
<font style="color:rgb(44, 62, 80);">编写一个算法来判断一个数 n 是不是快乐数。</font>

<font style="color:rgb(44, 62, 80);">「快乐数」定义为：对于一个正整数，每一次将该数替换为它每个位置上的数字的平方和，然后重复这个过程直到这个数变为 1，也可能是 无限循环 但始终变不到 1。如果 可以变为  1，那么这个数就是快乐数。</font>

<font style="color:rgb(44, 62, 80);">如果 n 是快乐数就返回 True ；不是，则返回 False 。</font>

**<font style="color:rgb(44, 62, 80);">示例：</font>**

<font style="color:rgb(44, 62, 80);">输入：19  
</font><font style="color:rgb(44, 62, 80);">输出：true  
</font><font style="color:rgb(44, 62, 80);">解释：  
</font><font style="color:rgb(44, 62, 80);">1^2 + 9^2 = 82  
</font><font style="color:rgb(44, 62, 80);">8^2 + 2^2 = 68  
</font><font style="color:rgb(44, 62, 80);">6^2 + 8^2 = 100  
</font><font style="color:rgb(44, 62, 80);">1^2 + 0^2 + 0^2 = 1</font>

<font style="color:rgb(44, 62, 80);"></font>

#### 思路:
<font style="color:rgb(44, 62, 80);">这道题目看上去貌似一道数学问题，其实并不是！</font>

<font style="color:rgb(44, 62, 80);">题目中说了会 </font>**<font style="color:rgb(44, 62, 80);">无限循环</font>**<font style="color:rgb(44, 62, 80);">，那么也就是说</font>**<font style="color:rgb(44, 62, 80);">求和的过程中，</font>**`**<font style="color:rgb(44, 62, 80);">sum</font>**`**<font style="color:rgb(44, 62, 80);">会重复出现，这对解题很重要！</font>**

<font style="color:rgb(44, 62, 80);">正如：</font>[00-基础](https://www.yuque.com/u25370234/kb/eu84c6)<font style="color:rgb(44, 62, 80);">中所说，</font>**<font style="color:rgb(44, 62, 80);">当我们遇到了要快速判断一个元素是否出现集合里的时候，就要考虑哈希法了。</font>**

<font style="color:rgb(44, 62, 80);">所以这道题目使用哈希法，来判断这个</font>`<font style="color:rgb(44, 62, 80);">sum</font>`<font style="color:rgb(44, 62, 80);">是否重复出现，如果重复了就是</font>`<font style="color:rgb(44, 62, 80);">return false</font>`<font style="color:rgb(44, 62, 80);">， 否则一直找到</font>`<font style="color:rgb(44, 62, 80);">sum</font>`<font style="color:rgb(44, 62, 80);">为1为止。</font>

<font style="color:rgb(44, 62, 80);">判断sum是否重复出现就可以使用</font>`<font style="color:rgb(44, 62, 80);">set</font>`<font style="color:rgb(44, 62, 80);">。</font>  


```javascript
var getSum = function (n) {
    let sum = 0;
    while (n) {
        sum += (n % 10) ** 2;
        n =  Math.floor(n/10);
    }
    return sum;
}
var isHappy = function(n) {
    let set = new Set();   // Set() 里的数是惟一的
    // 如果在循环中某个值重复出现，说明此时陷入死循环，也就说明这个值不是快乐数
    while (n !== 1 && !set.has(n)) {
        set.add(n);
        n = getSum(n);
    }
    return n === 1;
};
```



#### 解法二：快慢指针
**思路：通过快慢指针判断是否进入死循环。**

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1651384760556-f54d4651-dd18-4578-86eb-9a077ce2fffd.png)

```javascript
/**
 * @param {number} n
 * @return {boolean}
 */
const  getNext = (n) => {
    return String(n).split('').reduce((pre, next) => {
        return pre + next ** 2;
    }, 0)
}
var isHappy = function(n) {
    let fast = getNext(n),
        slow = n;
    
    while (fast !== 1 && fast !== slow) {
        slow = getNext(slow);
        fast = getNext(getNext(fast));
    }

    return fast === 1;
};
```
