### 一、例题-[1. 两数之和](https://leetcode.cn/problems/two-sum/)
<font style="color:rgb(44, 62, 80);">给定一个整数数组 </font>`<font style="color:rgb(44, 62, 80);">nums</font>`<font style="color:rgb(44, 62, 80);"> 和一个目标值 </font>`<font style="color:rgb(44, 62, 80);">target</font>`<font style="color:rgb(44, 62, 80);">，请你在该数组中找出和为目标值的那 两个 整数，并返回他们的数组下标。</font>

<font style="color:rgb(44, 62, 80);">你可以假设每种输入只会对应一个答案。但是，数组中同一个元素不能使用两遍。</font>

**<font style="color:rgb(44, 62, 80);">示例:</font>**

<font style="color:rgb(44, 62, 80);">给定 nums = [2, 7, 11, 15], target = 9</font>

<font style="color:rgb(44, 62, 80);">因为 nums[0] + nums[1] = 2 + 7 = 9</font>

<font style="color:rgb(44, 62, 80);">所以返回 [0, 1]</font>



#### 思路  
	<font style="color:rgb(44, 62, 80);">很明显暴力的解法是两层for循环查找，时间复杂度是$O(n^2)$。</font>
[01-有效的字母异位词](https://www.yuque.com/u25370234/kb/arxd3p)<font style="color:rgb(44, 62, 80);">这道题目是用</font>`<font style="color:rgb(44, 62, 80);">数组</font>`<font style="color:rgb(44, 62, 80);">作为哈希表来解决哈希问题，</font>[02-两个数组的交集](https://www.yuque.com/u25370234/kb/ovr22k)<font style="color:rgb(44, 62, 80);">这道题目是通过</font>`<font style="color:rgb(44, 62, 80);">set</font>`<font style="color:rgb(44, 62, 80);">作为哈希表来解决哈希问题。</font>

<font style="color:rgb(44, 62, 80);">本题呢，则要使用</font>`<font style="color:rgb(44, 62, 80);">map</font>`<font style="color:rgb(44, 62, 80);">，那么来看一下使用数组和set来做哈希法的局限。</font>

+ <font style="color:rgb(44, 62, 80);">数组的大小是受限制的，而且如果元素很少，而哈希值太大会造成内存空间的浪费。</font>
+ <font style="color:rgb(44, 62, 80);">set是一个集合，里面放的元素只能是一个key，而两数之和这道题目，不仅要判断y是否存在而且还要记录y的下标位置，因为要返回x 和 y的下标。所以set 也不能用。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">此时就要选择另一种数据结构：</font>`<font style="color:rgb(44, 62, 80);">map</font>`<font style="color:rgb(44, 62, 80);"> ，</font>`<font style="color:rgb(44, 62, 80);">map</font>`<font style="color:rgb(44, 62, 80);">是一种</font>`<font style="color:rgb(44, 62, 80);">key value</font>`<font style="color:rgb(44, 62, 80);">的存储结构，可以用ke</font><font style="color:rgb(44, 62, 80);">y保存数值，用</font>`<font style="color:rgb(44, 62, 80);">value</font>`<font style="color:rgb(44, 62, 80);">在保存数值所在的下标。</font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">解题思路动画如下：</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1652366395826-2024078d-663e-4d6a-bb41-33af0a10da19.png)

```javascript
/**
 * @param {number[]} nums
 * @param {number} target
 * @return {number[]}
 */
var twoSum = function(nums, target) {
    const map = new Map;

    for (let i = 0;i < nums.length;i++) {
        if (map.get(target - nums[i]) !== undefined) {
            return [i, map.get(target - nums[i])];
        }
        map.set(nums[i], i);
    }
    return [];
};
```
