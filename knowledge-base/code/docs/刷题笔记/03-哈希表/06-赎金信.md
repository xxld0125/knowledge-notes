### 例题-[383. 赎金信](https://leetcode.cn/problems/ransom-note/)
<font style="color:rgb(44, 62, 80);">给定一个赎金信 (ransom) 字符串和一个杂志(magazine)字符串，判断第一个字符串 ransom 能不能由第二个字符串 magazines 里面的字符构成。如果可以构成，返回 true ；否则返回 false。</font>

<font style="color:rgb(44, 62, 80);">(题目说明：为了不暴露赎金信字迹，要从杂志上搜索各个需要的字母，组成单词来表达意思。杂志字符串中的每个字符只能在赎金信字符串中使用一次。)</font>

**<font style="color:rgb(44, 62, 80);">注意：</font>**

<font style="color:rgb(44, 62, 80);">你可以假设两个字符串均只含有小写字母。</font>

<font style="color:rgb(44, 62, 80);">canConstruct("a", "b") -> false  
</font><font style="color:rgb(44, 62, 80);">canConstruct("aa", "ab") -> false  
</font><font style="color:rgb(44, 62, 80);">canConstruct("aa", "aab") -> tru</font>

####   
思路:
<font style="color:rgb(44, 62, 80);">这道题目和</font>[02-有效的字母异位词](https://www.yuque.com/u25370234/kb/arxd3p)<font style="color:rgb(44, 62, 80);">很像，</font>[02-有效的字母异位词](https://www.yuque.com/u25370234/kb/arxd3p)<font style="color:rgb(44, 62, 80);">相当于求 字符串a 和 字符串b 是否可以相互组成 ，而这道题目是求 字符串a能否组成字符串b，而不用管字符串b 能不能组成字符串a。</font>

<font style="color:rgb(44, 62, 80);">本题判断第一个字符串ransom能不能由第二个字符串magazines里面的字符构成，但是这里需要注意两点。</font>

+ <font style="color:rgb(44, 62, 80);">第一点“为了不暴露赎金信字迹，要从杂志上搜索各个需要的字母，组成单词来表达意思”  这里</font>_<font style="color:rgb(44, 62, 80);">说明杂志里面的字母不可重复使用。</font>_
+ <font style="color:rgb(44, 62, 80);">第二点 “你可以假设两个字符串均只含有小写字母。” </font>_<font style="color:rgb(44, 62, 80);">说明只有小写字母</font>_<font style="color:rgb(44, 62, 80);">，这一点很重要 </font>

<font style="color:rgb(44, 62, 80);"></font>

<font style="color:rgb(44, 62, 80);">1、那么第一个思路其实就是暴力枚举了，两层for循环，不断去寻找.</font>

<font style="color:rgb(44, 62, 80);">2、哈希解法</font>

<font style="color:rgb(44, 62, 80);">因为题目所只有</font>`<font style="color:#E8323C;">小写字母</font>`<font style="color:rgb(44, 62, 80);">，那可以采用空间换取时间的哈希策略， 用一个长度为26的数组还记录magazine里字母出现的次数。</font>

<font style="color:rgb(44, 62, 80);">然后再用ransomNote去验证这个数组是否包含了ransomNote所需要的所有字母。</font>

<font style="color:rgb(44, 62, 80);">依然是数组在哈希法中的应用。</font>

<font style="color:rgb(44, 62, 80);">一些同学可能想，用数组干啥，都用map完事了，</font>**<font style="color:rgb(44, 62, 80);">其实在本题的情况下，使用map的空间消耗要比数组大一些的，因为map要维护红黑树或者哈希表，而且还要做哈希函数，是费时的！数据量大的话就能体现出来差别了。 所以数组更加简单直接有效！</font>**



```javascript
/**
 * @param {string} ransomNote
 * @param {string} magazine
 * @return {boolean}
 */
var canConstruct = function(ransomNote, magazine) {
    const list = new Array(26).fill(0);

    for (let s of magazine) {
        list[s.charCodeAt() - 'a'.charCodeAt()]++;
    }

    for (let s of ransomNote) {
        const i =  s.charCodeAt() - 'a'.charCodeAt();
        if (list[i] <= 0) {
            return false;
        }
        list[i]--;
    }

    return true;
};
```
