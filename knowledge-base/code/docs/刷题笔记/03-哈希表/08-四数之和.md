### 一、例题-[18. 四数之和](https://leetcode.cn/problems/4sum/)


<font style="color:rgb(44, 62, 80);">题意：给定一个包含 n 个整数的数组 nums 和一个目标值 target，判断 nums 中是否存在四个元素 a，b，c 和 d ，使得 a + b + c + d 的值与 target 相等？找出所有满足条件且不重复的四元组。</font>

**<font style="color:rgb(44, 62, 80);">注意：</font>**

<font style="color:rgb(44, 62, 80);">答案中不可以包含重复的四元组。</font>

<font style="color:rgb(44, 62, 80);">示例： 给定数组 nums = [1, 0, -1, 0, -2, 2]，和 target = 0。 满足要求的四元组集合为： [ [-1, 0, 0, 1], [-2, -1, 1, 2], [-2, 0, 0, 2] </font>

<font style="color:rgb(44, 62, 80);"></font>

#### <font style="color:rgb(44, 62, 80);">思路:</font>
<font style="color:rgb(44, 62, 80);">四数之和，和</font>[07-三数之和](https://www.yuque.com/u25370234/kb/ldtvqr)<font style="color:rgb(44, 62, 80);">是一个思路，都是使用双指针法, 基本解法就是在</font>[07-三数之和](https://www.yuque.com/u25370234/kb/ldtvqr)<font style="color:rgb(44, 62, 80);">的基础上再套一层for循环。</font>

[07-三数之和](https://www.yuque.com/u25370234/kb/ldtvqr)<font style="color:rgb(44, 62, 80);">的双指针解法是一层</font>`<font style="color:rgb(44, 62, 80);">for</font>`<font style="color:rgb(44, 62, 80);">循环</font>`<font style="color:rgb(44, 62, 80);">num[i]</font>`<font style="color:rgb(44, 62, 80);">为确定值，然后循环内有</font>`<font style="color:rgb(44, 62, 80);">left</font>`<font style="color:rgb(44, 62, 80);">和</font>`<font style="color:rgb(44, 62, 80);">right</font>`<font style="color:rgb(44, 62, 80);">下标作为双指针，找到</font>`<font style="color:rgb(44, 62, 80);">nums[i] + nums[left] + nums[right] == 0</font>`<font style="color:rgb(44, 62, 80);">。</font>

<font style="color:rgb(44, 62, 80);">四数之和的双指针解法是两层</font>`<font style="color:rgb(44, 62, 80);">for</font>`<font style="color:rgb(44, 62, 80);">循环</font>`<font style="color:rgb(44, 62, 80);">nums[k] + nums[i]</font>`<font style="color:rgb(44, 62, 80);">为确定值，依然是循环内有</font>`<font style="color:rgb(44, 62, 80);">left</font>`<font style="color:rgb(44, 62, 80);">和</font>`<font style="color:rgb(44, 62, 80);">right</font>`<font style="color:rgb(44, 62, 80);">下标作为双指针，找出</font>`<font style="color:rgb(44, 62, 80);">nums[k] + nums[i] + nums[left] + nums[right] == target</font>`<font style="color:rgb(44, 62, 80);">的情况，三数之和的时间复杂度是</font>`<font style="color:rgb(44, 62, 80);">O(n^2)</font>`<font style="color:rgb(44, 62, 80);">，四数之和的时间复杂度是</font>`<font style="color:rgb(44, 62, 80);">O(n^3)</font>`<font style="color:rgb(44, 62, 80);"> 。</font>

**<font style="color:#E8323C;">那么一样的道理，五数之和、六数之和等等都采用这种解法。</font>**

**<font style="color:#E8323C;"></font>**

```javascript
/**
 * @param {number[]} nums
 * @param {number} target
 * @return {number[][]}
 */
var fourSum = function(nums, target) {
    const len = nums.length;
    if (len < 4) return [];
    nums.sort((a, b) => a - b);

    const set = new Set;

    for (let i = 0;i < len - 3;i++) {
        for (let j = i + 1;j < len - 2;j++) {
            let l = j + 1,
                r = len - 1;
            while (l < r) {
                const sum = nums[i] + nums[j] + nums[l] + nums[r];
                if (sum === target) {
                    set.add(`${nums[i]},${nums[j]},${nums[l]},${nums[r]}`);
                    l++;
                    r--;
                } else if (sum < target) {
                    l++;
                } else {
                    r--;
                }
            }
        }
    }

    return Array.from(set).map(v => v.split(','));
};
```
