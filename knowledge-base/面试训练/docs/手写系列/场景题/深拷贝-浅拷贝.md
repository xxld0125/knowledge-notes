# 深拷贝/浅拷贝

思考

+ 如何复制不可枚举属性及symbol
+ 如何对Date、RegExp对象进行递归赋值
+ 赋值原对象的原型
+ 如何处理循环引用问题



参考：

+ <font style="color:rgb(47, 50, 56);">针对遍历对象的不可枚举属性以及symbol类型，可以使用Reflect.ownKeys()</font>
+ <font style="color:rgb(47, 50, 56);">当参数为Date、RegExp类型，直接生成一个新的实例返回</font>
+ <font style="color:rgb(47, 50, 56);">利用Object的getOwnPropertyDescriptors方法可以获得对象的所有属性，以及对应的特性，顺便结合Object的create方法创建一个新对象，并继承传入原对象的原型</font>
+ <font style="color:rgb(47, 50, 56);">使用weakMap解决循环引用问题，可以有效防止内存泄露</font>
