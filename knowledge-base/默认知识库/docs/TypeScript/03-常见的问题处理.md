#### 一、将一个对象的所有键/值变成一个联合类型
要把一个对象的所有键、值组合成一个联合类型

+ 定义一个**<font style="color:#DF2A3F;">只读</font>**对象
+ 使用`**<font style="color:#DF2A3F;">keyof typeof</font>**`得到对象的所有键类型
+ 使用键取得所有值的联合类型



```javascript
const obj = {
  name: 'Tom',
  country: 'Chile',
} as const;

// 👇️ type UValues = "Tom" | "Chile"
type UValues = typeof obj[keyof typeof obj];

// 👇️ type UKeys = "name" | "country"
type UKeys = keyof typeof obj;

```

	

<font style="color:rgb(77, 77, 77);">为什么必须是只读的，主要是</font>**<font style="color:#DF2A3F;">防止对象在中途被篡改，那么得到的联合类型就不准确</font>**<font style="color:rgb(77, 77, 77);">，因此 </font>`TypeScript`<font style="color:rgb(77, 77, 77);"> 才加上如此限制</font>
