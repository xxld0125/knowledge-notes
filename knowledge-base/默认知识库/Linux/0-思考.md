### <font style="color:rgb(44, 62, 80);">一、linux对前端的影响</font>
<font style="color:rgb(44, 62, 80);">Linux 技能对于前端，还是后端，都是十分必要的。</font>

<font style="color:rgb(44, 62, 80);"></font>

1. <font style="color:rgb(44, 62, 80);">环境变量。在前后端项目中都可能有一个文件</font><font style="color:rgb(44, 62, 80);"> </font><font style="color:rgb(71, 101, 130);">.env</font><font style="color:rgb(44, 62, 80);"> </font><font style="color:rgb(44, 62, 80);">用以设置环境变量，它是怎么工作的呢？</font>
2. <font style="color:rgb(44, 62, 80);">链接。在 pnpm 中，结合软链接与硬链接大幅降低了装包的体积，它是如何做到的？</font>
3. <font style="color:rgb(44, 62, 80);">全局命令行工具。在 nodejs/python/go 中都可以开发全局可执行命令，开发一个全局命令的原理是什么？</font>
4. <font style="color:rgb(44, 62, 80);">端口检测。在 vue/react/next 等脚手架的开发环境，如果启动时发生端口冲突，则会重开一个端口，此种原理如何？</font>
5. <font style="color:rgb(44, 62, 80);">fs。在 nginx 中我们得知 last-modified 是根据 mtime 生成，那如何获得静态资源的 mtime 呢？</font>
6. <font style="color:rgb(44, 62, 80);">vim。如果需要在测试服务器部署测试环境，那在 linux 中使用 vim 操作必不可少。</font>
7. <font style="color:rgb(44, 62, 80);">tmux。如何在个人服务器启动自己服务，过了一分钟，ssh 断开，服务挂掉，这应该如何处理？</font>
8. <font style="color:rgb(44, 62, 80);">内存监控。在写 Node.js 时，他人常说 Node.js 写服务更消耗内存，那如何监听到某个 Node.js 服务的 内存占用情况呢？</font>
9. <font style="color:rgb(44, 62, 80);">当打开一个 URL 时发生了什么。此时有一些发生在 http/dns/tcp 协议的一些事情，我们如何通过相关命令监控这一过程呢？</font>
