<font style="color:rgb(77, 77, 77);">拦截器是使用 </font>@Injectable()<font style="color:rgb(77, 77, 77);"> 装饰器注解的类。拦截器应该实现 </font>NestInterceptor<font style="color:rgb(77, 77, 77);"> 接口。</font>

![](https://cdn.nlark.com/yuque/0/2022/png/25743026/1669272855754-adf4bbd9-c879-474a-a475-11b86ce69756.png)

<font style="color:rgb(77, 77, 77);"></font>

<font style="color:rgb(77, 77, 77);">拦截器具有一系列有用的功能，这些功能受面向切面编程（</font>`<font style="color:rgb(77, 77, 77);">AOP</font>`<font style="color:rgb(77, 77, 77);">）技术的启发。它们可以：</font>

+ <font style="color:rgb(77, 77, 77);">在函数执行之前/之后绑定</font>**额外的逻辑**
+ <font style="color:rgb(77, 77, 77);">转换从函数返回的结果</font>
+ **转换**<font style="color:rgb(77, 77, 77);">从函数抛出的异常</font>
+ <font style="color:rgb(77, 77, 77);">扩展基本函数行为</font>
+ <font style="color:rgb(77, 77, 77);">根据所选条件完全重写函数 (例如, 缓存目的)</font>

#### 一、响应拦截器
可以通过响应拦截器给接口统一返回格式。

```typescript
import { Injectable, NestInterceptor, CallHandler } from '@nestjs/common';
import { map } from 'rxjs/operators';
import { Observable } from 'rxjs';

interface data<T> {
  data: T;
}

@Injectable()
export class Response<T = any> implements NestInterceptor {
  intercept(context, next: CallHandler): Observable<data<T>> {
    return next.handle().pipe(
      map((data) => {
        return {
          data,
          errcode: 0,
          message: '接口状态信息',
        };
      }),
    );
  }
}

```



在`main.ts`注册拦截器

```typescript
import { Response } from './common/response';
app.useGlobalInterceptors(new Response());
```



#### 二、异常拦截器
创建一个异常过滤器，它负责捕获作为`HttpException`类实例的异常，并为它们设置自定义响应逻辑。为此，我们需要访问底层平台 `Request`和 `Response`。我们将访问`Request`对象，以便提取原始 url并将其包含在日志信息中。我们将使用 `Response.json()`方法，使用 Response对象直接控制发送的响应。

```typescript
import {
  ExceptionFilter,
  Catch,
  ArgumentsHost,
  HttpException,
} from '@nestjs/common';

import { Request, Response } from 'express';

@Catch(HttpException)
export class HttpFilter implements ExceptionFilter {
  catch(exception, host: ArgumentsHost) {
    const ctx = host.switchToHttp();
    const request = ctx.getRequest<Request>();
    const response = ctx.getResponse<Response>();

    const status = exception.status;

    response.status(status).json({
      data: exception.message,
      time: new Date().getTime(),
      success: false,
      path: request.url,
      status,
    });
  }
}

```

	

在`main.ts`注册拦截器

```typescript
import { HttpFilter } from './common/filter';
app.useGlobalFilters(new HttpFilter());
```
